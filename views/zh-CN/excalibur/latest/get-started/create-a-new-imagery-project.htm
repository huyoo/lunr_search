您可以使用 ArcGIS Excalibur 中的指导工作流来新建影像工程。指导工作流将引导您完成每个步骤，以确保创建影像工程所需的所有资源均已提供。除了指导工作流之外，系统还提供高级选项，通过上传影像工程定义文件即可在一个步骤中创建影像工程。

指导工作流

您可以使用 ArcGIS Excalibur 中的指导工作流来新建影像工程。指导工作流将引导您完成每个步骤，以确保创建影像工程所需的所有资源均已提供。

您可以随时取消指导工作流或返回上一步骤。您还可以忽略影像工程不需要的可选步骤。

标识影像

创建影像工程中的第一步是标识要在工程中使用的影像。如果您已从影像目录搜索的已排队图像选项卡或者影像开发画布中获取影像，则可以从选择工程类型开始指导工作流。如果您不具备用于已标识工程的影像，请完成以下步骤：

  1. 在新建影像工程视图中，从影像图层选择器中选择一个影像图层。
    注：

    选择器中的影像图层是可以从您的门户中的内容和组织内容中获取的影像服务。

  2. 单击选择。

    将打开影像搜索目录，将在所选影像图层上自动运行搜索。

  3. 在搜索结果选项卡中选择图像，将其添加到已排队图像选项卡。
  4. 单击已排队图像选项卡，然后单击新建工程按钮以开始创建工程。

选择工程类型

选择工程类型。工程类型可提供在处理影像工程时可以使用的一组专用工具。

  1. 单击要使用的工程类型的影像工程类型复选框。
  2. 单击下一步按钮。

定义工程详细信息

提供用于描述影像工程的信息以及需要完成的内容。

  1. 提供以下参数的信息：

    • 标题（必填）
    • 摘要（可选）
    • 说明（必填）
    • 标签（必填）

  2. 选择与您的组织共享工程或将其设置为私有。
  3. 单击下一步。

选择 web 地图

您也可以选择要包含在工程中的 web 地图。Web 地图可包含为影像任务提供上下文的参考图层。此为可选步骤；如果您不选择 web 地图，则将使用门户中的默认 web 地图。

  1. 使用 web 地图选取器选择 web 地图。

    只能选择一个 web 地图。

  2. 单击下一步。
  3. 或者，如果您不希望包含特定 web 地图，请单击跳过。
注：

可以从每个选项下的项目选取器中获取每个 web 地图的项目详细信息。单击后，将在单独的选项卡中加载项目详细信息页面。

选择观测点图层

如果您选择了观测点影像工程类型，则必须通过选择现有观测点图层或创建新的观测点图层来至少选择一个观测点图层。如果您选择了基础影像工程类型，则此步骤不会包含在指导工作流中，您将转至工程摘要。

选择现有观测点图层

如果您选择了现有观测点图层，则将显示观测点图层选取器，其中包含可用要素图层，将使用观测点图层标签对这些图层进行正确标记。可以从该列表中选择一个或多个观测点图层。

要选择现有观测点图层：

  1. 单击现有观测点图层选项后，可以使用观测点图层选取器至少选择一个观测点图层。
    注：

    如果您选择的图层无效或者未启用时间，则将显示一个对话框。如果您选择的图层未启用时间，并且您是图层所有者，则可以选择启用时间设置。您也可以选择清除警告，但是图层在影像工程中基于时间的功能将受到限制。

  2. 单击下一步。
注：

可以从每个选项下的项目选取器中获取每个现有观测点图层的项目详细信息。单击后，将在单独的选项卡中加载项目详细信息页面。

新建观测点图层

如果您选择了新建观测点图层，则将逐步体验每个步骤，允许您选择基础观测点图层方案或者向基础方案添加附加字段以快速开始观测点采集任务。

  1. 单击新建观测点图层选项后，请提供以下详细信息：
    • 图层标题（必填）
    • 图层几何类型（必填）
    • 图层标签（必填）

    注：
    新观测点图层的空间参考将设置为工程中所选影像的空间参考。

  2. 选择与您的组织共享观测点图层或将其设置为私有，然后单击下一步。
  3. 如果基础观测点图层方案为全部所需内容，请单击下一步。
  4. 或者，您可以通过提供以下内容向基础方案添加附加字段：
    • 字段名称
    • 字段别名
    • 字段类型
  5. 如果需要，可以通过选择编码值或范围为新字段定义选择列表（编码值或范围的域）。

    注：
    无法为字符串类型的新字段定义范围属性域。

  6. 为选择列表中的每个属性域输入名称和代码，以确保将每个属性域添加到列表中。
  7. 将所有属性域添加到新字段后，请单击添加字段。
  8. 重复步骤 4-7，直到将所有附加字段都添加到基础方案为止。
  9. 完成后，请单击下一步。
  10. 选择基于单一符号或唯一值的观测点图层样式。

    默认情况下，系统将选择单一符号样式选项，并且该选项基于您在观测点图层详细信息步骤中选择的几何类型。

  11. 针对您的图层选择单一符号样式属性。
    • 点：大小、样式、颜色和轮廓
    • 线：选择颜色和宽度
    • 面：轮廓、填充和轮廓宽度
  12. 如果需要唯一值样式，请选择唯一值选项以基于编码值设置图层样式。

    注：
    如果新的观测点图层不包含任何编码值属性域，则将禁用唯一值选项。范围属性域不支持样式设置。

  13. 从可用字段选择列表中选择要设置观测点图层样式的字段，该观测点图层包含编码值属性域。

    编码值表将使用所选字段中的所有值进行填充。

  14. 单击编码值表中的每个样式图标，并更改样式属性。
  15. 完成后，请单击下一步。
  16. 查看您刚刚作出的选择的摘要，然后单击创建以创建新的观测点图层。

    创建新的观测点图层时，将显示状态指示器。

  17. 成功创建新的观测点图层后，单击下一步可查看工程摘要，也可单击添加其他观测点图层以添加更多图层。

    注：
    如果您选择添加更多观测点工程图层，您将返回到添加更多观测点工程图层步骤。

    在进行下一步之前，将显示您创建的观测点图层的摘要。

工程摘要

以下是指导工作流的最后一步。即为针对影像工程已选择内容的摘要。如果需要进行任何编辑，请单击上一步按钮以进行适当的更改。确认所有内容后，请完成下列步骤来创建影像工程：

  1. 单击创建工程按钮以创建影像工程。

    将随即创建新影像工程。

  2. 成功创建影像工程后，单击即可打开工程或查看工程列表。

上传影像工程定义文件

除了指导工作流之外，系统还提供高级选项，通过上传影像工程定义文件即可在一个步骤中创建影像工程。要成功创建影像工程，必须提供正确格式化的 JSON 文件。

要上传影像工程定义文件，请完成以下步骤：

  1. 从 ArcGIS Excalibur 主页或使用左侧导航打开新建影像工程视图。
  2. 在新建影像工程视图中，单击高级选项按钮。
  3. 单击浏览影像工程定义文件并浏览至正确格式化的 JSON 文件。
  4. 单击工程以将其打开或查看工程列表。

以下示例提供了基础和观测点影像工程类型的正确格式化示例 JSON。

基础影像工程 JSON

以下基础影像工程类型的格式化 JSON 示例包括下列内容：

  • 工程详细信息
  • 工程说明
  • 来自影像服务的影像
{
  "itemInfo": {
	"title": "Title of project",
	"summary": "Summary of project.",
	"projectType": "base"
  },
  "itemData": {
	"instructions": "Project instructions.",
	"rasterIds": [1],
	"serviceUrl": "Imagery Web Service URL"
  },
  "version": "1.0"
}

观测点影像工程 JSON

以下观测点影像工程类型的格式化 JSON 示例包括下列内容：

  • 工程详细信息
  • 工程说明
  • 来自影像服务的影像
  • Web 地图
  • 工程详细信息
  • 可编辑采集图层
{
  "itemInfo": {
	"title": "Title of project",	"summary": "Summary of project.",	"projectType": "observation"
  },  "itemData": {
	"instructions": "Project instructions",	"rasterIds": [1, 4, 7],	"observationLayers": [{
	  "itemId": "Observation layer item ID",	  "type": "Feature Layer"
	}],	"serviceUrl": "Imagery Web Service URL",	"webmapId": "Web Map item ID"
	},  "version": "1.0"
}
