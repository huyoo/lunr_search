使用显示工具

在本主题中

  1. 影像显示工具
  2. 图像顺序工具
  3. 增强工具

根据影像服务所包含的元数据类型以及该服务的准备方式，可通过多种方式来显示和渲染影像服务。根据影像的质量和您的工作流任务，您可能希望改善影像的外观。显示工具可以将常用显示设置应用于影像，并更改影像的顺序以从每个像素获得最大价值。

影像显示工具

影像显示工具允许您使用渲染器、更改波段组合，以及将常用设置（如动态范围调整 (DRA) 和 Gamma）应用于影像。

应用渲染器

可通过多种方式来显示或渲染影像服务。渲染就是显示数据的过程。一些影像服务具有与其相关联的预定义模板。如果影像服务包含预定义的渲染器，则这些渲染器将在渲染器下拉菜单中提供，其中最顶部的渲染器将默认处于选定状态。如果影像服务不包含预定义的渲染器，则渲染器下拉菜单将设置为用户定义的渲染器。

要更改影像服务渲染器，请完成以下步骤：

  1. 单击影像开发画布右上角的溢出菜单。
  2. 在显示工具列表中，单击影像显示。
  3. 在渲染器下拉列表中选择一个渲染器。
    • 预定义的渲染器 - 列表中除用户定义的渲染器之外的所有渲染器。选择预定义的渲染器时，将无法设置任何其他用户定义的设置。
    • 用户定义的渲染器 - 适用于所有影像服务。选择用户定义的渲染器时，可以应用用户定义的设置，如 DRA 和 Gamma。

    选择渲染器后，可将其自动应用于影像服务。

透明度

通过更改影像图层的透明度或不透明度，可以或多或少地查看位于其下的其他底图。可以将影像图层的透明度从 0% 调整到 100%。图层的透明度越高，其在地图上的可见性越低。

要更改“图像图层透明度”，请执行以下步骤：

  1. 单击影像开发画布右上角的溢出菜单。
  2. 在显示工具列表中，单击影像显示。
  3. 使用透明度滑块选择透明度级别。

    应用透明度后，影像将动态变化。

压缩

性能和影像质量之间面临权衡取舍，因为这二者成反比。质量较高的影像意味着较大的文件，并会对性能速度造成影响。性能还取决于您的连接速度。

要更改“图像图层压缩”，请执行以下步骤：

  1. 单击影像开发画布右上角的溢出菜单。
  2. 在显示工具列表中，单击影像显示。
  3. 使用压缩滑块选择压缩等级。
    • 性能 - 以 50% 的质量压缩为 JPEG 格式。
    • 平衡 - 当前视图中存在 NoData 像素时，压缩为 PNG32 格式；没有透明像素时，以 75% 的质量压缩为 JPEG 格式。
    • 质量 - 当前视图中存在 NoData 像素时，压缩为 PNG32 格式；没有透明像素时，以 95% 的质量创建 JPEG 格式压缩。

    应用压缩后，影像将动态变化。

应用 DRA 设置

动态范围调整通过仅基于显示中的像素值，而非服务中的所有像素动态拉伸影像来增强影像。

要应用 DRA，请完成以下步骤：

  1. 单击影像开发画布右上角的溢出菜单。
  2. 在显示工具列表中，单击影像显示。
  3. 在可用渲染器下拉列表中选择用户定义。
  4. 在“快速增强”下，单击 DRA。您将注意到当前视图中影像像素的变化。
注：

DRA 需要应用拉伸类型。切换打开 DRA 后，可将最小和最大拉伸类型自动应用于影像服务。可以使用“增强”工具来更改此拉伸类型。

应用 Gamma 设置

Gamma 通过动态拉伸影像中的中间值，但保持极高值和极低值不变的方式来增强影像。

要应用 Gamma，请完成以下步骤：

  1. 单击影像开发画布右上角的溢出菜单。
  2. 在显示工具列表中，单击影像显示。
  3. 在可用渲染器下拉列表中选择用户定义。
  4. 在“快速增强”下，单击 Gamma。您将注意到当前视图中影像像素的变化。

应用波段组合

波段组合设置用于指定哪些波段使用红、绿和蓝彩色合成方案进行显示。借助不同的颜色组合，可使用不同的颜色效果来显示影像。可以使用自然色（红、绿、蓝），其显示的图层与您正常看到的相同。其他颜色组合可突出显示植被、市区、水体和其他要素。

波段组合设置仅在已选择用户定义的渲染器且影像服务为多波段服务时可用。

要更改显示的多光谱波段组合，请完成以下步骤：

  1. 单击影像开发画布右上角的溢出菜单。
  2. 在显示工具列表中，单击影像显示。
  3. 在渲染器下拉列表中选择用户定义的渲染器。
  4. 对于设置自定义 RGB 渲染器，请单击波段选择下拉箭头，然后为红、绿、蓝选择波段号。
    注：

    如果影像服务为单波段服务，则此选项将不可见。

  5. 单击应用波段按钮以应用 RGB 合成。
  6. 或者，单击重置波段按钮以返回默认组合。

图像顺序工具

影像服务包含一个或多个动态镶嵌的图像，当您浏览服务时，这些图像的行为与单一图像相同。图像顺序工具有助于确定图像顺序以及解决重叠区域的方法。这对于同一区域上存在多个影像的情况特别有用。

要访问图像顺序工具，请完成以下步骤：

  1. 单击影像开发画布右上角的溢出菜单。
  2. 在显示工具列表中，单击图像顺序工具。
  3. 可以使用以下选项更新所选择的显示优先级。一次只能应用一个选项。
    • 按属性 - 根据数值或日期属性对影像进行排序。还可以根据此选项设置所选属性的反向顺序。
    • 中心 - 影像中心与屏幕中央距离最近的影像将显示在顶部。
    • 像底点 - 像底点与屏幕中央距离最近的影像将显示在顶部。
    • 西北 - 根据每个影像的中心距图层中所有影像的集合的西北角的距离确定影像顺序。

    顺序变化后，地图焦点面板将动态变化。

    注：

    将视图设置为显示影像图层中特定所选图像集后，将禁用此工具。

  4. 单击重置为默认顺序可恢复为原始图像顺序。

增强工具

增强工具有助于改善影像质量。拉伸增强功能通过拉伸类型更改图像的亮度、对比度和 Gamma。过滤增强功能有助于过滤影像中的像素，以使其可用于锐化、模糊或检测影像内的边。当“图像显示工具”中的渲染器设置为非用户定义的渲染器时，将禁用增强工具。

更改重采样

重采样是在变换栅格数据集时内插像素值的过程。在以下情况使用：输入和输出排列不对应、像素大小发生变化、数据被平移或以上原因的综合。要更改重采样，请单击下拉菜单以选择：

  • 双线性 - 根据四个最邻近输入像元中心的加权平均距离确定像元的新值。此选项对于连续数据非常有用。
  • 三次卷积插值法 - 通过拟合穿过 16 个最邻近输入像元中心的平滑曲线来确定像元的新值。此选项适用于连续数据，但是所生成的输出栅格可能会包含输入栅格范围以外的值。
  • 众数 - 根据过滤器窗口内的最常用值来确定像元的新值。此选项主要用于离散数据。
  • 最邻近法 - 最快的插值方法，主要用于离散数据。该方法将不会更改单元格的值。

拉伸

要对影像应用拉伸增强功能，请完成以下步骤：

  1. 单击影像开发画布右上角的溢出菜单。
  2. 在显示工具列表中，单击增强功能。
  3. 在应用函数下，单击拉伸选项卡，然后从选项列表中进行选择：
    • 无拉伸
    • 直方图均衡化
    • 最小最大
    • 裁剪百分比
    • 标准差

    应用拉伸方法后，影像将动态变化。

  4. 要清除应用的所有拉伸增强功能，请单击重置按钮。
注：

无法同时应用拉伸和过滤函数。在“增强”工具中应用的所有拉伸增强功能将直接反映在“图像显示”工具中。

过滤器

要对影像应用过滤增强功能，请完成以下步骤：

  1. 单击影像开发画布右上角的溢出菜单。
  2. 在显示工具列表中，单击增强功能。
  3. 在应用函数下，单击过滤器选项卡并选择要应用于图像的过滤器。
  4. 要清除所有过滤增强功能，请单击重置。
注：

无法同时应用拉伸和过滤函数。
