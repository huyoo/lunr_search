安装并授权 ArcGIS Excalibur

在本主题中

  1. 下载 ArcGIS Excalibur 安装程序
  2. 安装 ArcGIS Excalibur
  3. 为成员分配 ArcGIS Excalibur 许可
  4. 使用 ArcGIS Excalibur
  5. 卸载 ArcGIS Excalibur

满足系统要求并配置 ArcGIS Enterprise 基础部署后，您即可安装并授权 ArcGIS Excalibur。

下载 ArcGIS Excalibur 安装程序

要下载 ArcGIS Excalibur，请完成以下步骤：

  1. 使用与您的客户编号相关联的 Esri 帐户登录到 My Esri。
  2. 单击我的组织，然后单击下载。

    注：

    如果您同时是多个组织的成员，请确保右上角显示的是您想使用的组织的名称和客户编号。如果不是，请单击组织列表并选择其他组织。然后，单击下载。

    根据客户编号，将列出您可以使用的所有产品。

  3. 找到您要部署 ArcGIS Excalibur 的 ArcGIS Enterprise 产品，然后单击查看下载。

    例如，ArcGIS Enterprise Windows 10.7。

    ArcGIS Enterprise 10.7 及更高版本支持 ArcGIS Excalibur。

  4. 展开应用程序部分。
  5. 单击 ArcGIS Excalibur 安装程序的下载，然后将其保存到您的计算机上。

安装 ArcGIS Excalibur

您可以在 Windows 或者 Linux 计算机上安装 ArcGIS Excalibur。

在 Windows 上安装 ArcGIS Excalibur。

下载 ArcGIS Excalibur 后，请在已安装 Enterprise 门户的计算机上安装该软件。

  1. 浏览至已下载的安装文件，提取存档文件，然后双击 setup.exe 文件。

    注：
    安装程序会在您的计算机上检测 ArcGIS Enterprise 组件，并验证您是否安装了合适的组件。如果未检测到所需组件，则程序将关闭。有关详细信息，请参阅软件要求。

  2. 请按照安装向导中的说明进行操作，包括阅读和接受许可协议。如果您不同意许可协议中的条款，则无法完成安装。

    注：
    默认安装目录基于 Portal for ArcGIS 的安装目录并且无法进行更改。

在 Linux 上安装 ArcGIS Excalibur。

下载 ArcGIS Excalibur 后，请在已安装 Portal for ArcGIS 的计算机上安装该软件。

  1. 以具有管理权限的用户身份登录到计算机。
  2. 浏览至已下载的安装文件并提取存档文件。
  3. 运行安装命令。

ArcGIS Excalibur 成功安装后，必须先获得授权才能使用。可以从管理员或 Esri 客户服务中心获取授权文件。

为成员分配 ArcGIS Excalibur 许可

成功安装 ArcGIS Excalibur 并进行授权后，作为具有许可权限的管理员或成员，您必须向成员授予 ArcGIS Excalibur 许可，他们才能访问该应用程序。成员必须拥有具备相应角色的 Creator 或者 GIS Professional 用户类型，才能为其分配 ArcGIS Excalibur 许可。

  1. 以管理员或具有许可权限的成员身份登录到您的组织门户页面。
  2. 单击站点顶部的组织，然后单击许可选项卡。
  3. 确保选中加载项许可选项，然后单击应用程序部分以展开应用程序列表
  4. 单击 ArcGIS Excalibur 旁边的分配按钮。
  5. 如果需要，请使用过滤器根据用户类型或角色对成员进行排序。
  6. 单击每个成员行中的分配按钮以将 ArcGIS Excalibur 许可分配给单个成员，或者单击右上角中的在此页面上分配链接将 ArcGIS Excalibur 许可分配给列表中所有成员。
    注：

    要通过电子邮件通知成员您已向他们分配了许可，请选中向成员发送有关新分配许可的电子邮件。

使用 ArcGIS Excalibur

安装并授权 ArcGIS Excalibur 并已将许可分配给成员后，即可使用该软件。在门户应用程序启动器中，任何获得此应用程序许可的用户都可以使用 ArcGIS Excalibur 应用程序。可以直接单击 Excalibur 图标来启动应用程序。

此外，您还可以通过输入 ArcGIS Excalibur 的 URL 来直接导航至 ArcGIS Excalibur，例如，https://host.domain.com/webadaptor/apps/excalibur（其中 host、domain 和 web adaptor 将替换为您门户的相关信息）。

在 ArcGIS Excalibur 中搜索影像并了解影像的其他功能。

卸载 ArcGIS Excalibur

要卸载 ArcGIS Excalibur，请完成以下步骤：

  1. 在 Windows 控制面板中，单击程序和功能。
  2. 在程序列表中，单击 ArcGIS Excalibur。
  3. 单击卸载/更改，然后按照卸载向导中的步骤执行操作。
