可以在 ArcGIS Excalibur 中使用我的可用影像选项来确定可在应用程序中使用的您的内容和您的组织内容中的可用影像服务。我的可用影像视图将显示有关每个可用影像 web 服务的信息，允许您在影像目录搜索中将其启用或禁用以进行搜索，并且可以直接将其连接到目录或影像开发画布。

按影像图层标题进行过滤

可以按影像图层标题对影像 web 服务进行过滤。

  1. 在输入文本框中输入影像图层的名称。

    输入文本后，将对影像图层卡片进行动态过滤。

  2. 删除文本以清除文本框。

按字段排序

可以按字段（例如，创建日期、所有者或标题）以升序和降序对影像 web 服务进行排序，以帮助识别影像图层。

  1. 从排序菜单中选择字段。

    选项包括“创建日期”、“修改日期”、“所有者”和“标题”。“修改日期”为默认字段。

    可以基于所选字段对影像图层列表进行动态排序。

  2. 单击排序方向按钮可在升序和降序之间进行切换。

    降序是默认排序方向。

更改布局视图

可以在以下两种布局之间更改影像 web 服务布局：列表布局和格网布局。这两种布局都允许您查看有关影像图层的信息、直接连接到目录或画布，以及启用或禁用影像目录搜索中的服务。格网布局是“我的可用影像”的默认视图。

刷新列表

单击刷新列表按钮可重新加载我的可用影像页面。随即显示您的内容或您的组织内容中的可用服务。

影像图层详细信息

每个影像图层卡片将显示以下信息，这些信息来自影像图层的项目详细信息：

  • 图层类型
  • 名称
  • 汇总
  • 创建日期
  • 修改日期
  • 所有者
  • 访问权限

对影像图层的项目详细信息页面上的信息进行编辑后，下次刷新列表时，更改将反映在影像图层卡片上。

影像目录搜索图层

用于搜索的影像服务基于您的影像任务或需求。可以作为影像目录搜索中用于搜索的目录图层来启用或禁用每个可用影像图层。可以通过单击顶部的角框来启用和禁用影像图层。每次启用和禁用图层时，将在应用程序右下角显示通知。已启用的服务将使用绿色复选框进行标记，在目录影像图层切换器中可用。用户设置在每次启用或禁用图层时都会进行更新，并保持该状态，直到对其进行更改为止。

注：

页面顶部将显示一条消息，指示在目录搜索中针对快速访问进行标记的影像图层数量。单击消息右侧的按钮以确认所有快速访问图层均有效。如果识别到无效图层，则将移除该图层，并且快速访问图层将进行更新。

连接到目录

单击连接到目录时，影像目录搜索窗格随即打开，您可以在其中搜索、发现和排队影像以供进一步使用。影像目录中的搜索地图将随即缩放至影像服务的范围，您可以在其中定义搜索设置，例如要搜索的区域。

了解有关影像目录搜索的详细信息

连接到画布

单击连接到画布时，影像开发画布随即打开，您可以在其中使用显示和开发工具来关注和处理特定影像。

影像开发画布中的地图焦点面板将缩放至影像服务的范围并显示覆盖区，同时将使用相应影像来填充影像元数据表。

了解有关影像开发画布的详细信息

其他图层信息

有关特定影像图层的详细信息，请在格网视图或列表视图的溢出菜单中选择关于此图层的详细信息选项。

单击关于此图层的详细信息后，将打开一个新选项卡，其中显示此影像图层的图层详细信息。您可以在该页面上查看：

  • 图层摘要
  • 创建和更新日期
  • 图层所有者
  • 描述
  • 使用条款
  • 标签
  • 制作者名单
  • URL
