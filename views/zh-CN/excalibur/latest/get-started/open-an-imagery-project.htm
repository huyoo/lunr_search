打开影像工程

在本主题中

  1. 来自影像服务的影像
  2. 工程说明
  3. 包含参考图层的 Web 地图
  4. 可编辑采集图层

可以从影像工程列表中打开影像工程。默认情况下，工程会在画布视图中打开，其中可用信息基于工程类型和工程中所包含的资源。工程类型决定可用的工具。以下部分会根据所打开的工程的类型及其相应资源来描述您将遇到的各种工程项目。

来自影像服务的影像

所有影像工程都包含用于创建该影像工程的一个影像或一组影像。影像可以在受支持的影像聚焦视图中使用，具体视影像类型而定。如果工程中存在可编辑采集图层，它将使用已定义影像中的元数据来自动填充必填属性字段，以将采集的任何观测点与当前影像相关联。

工程说明

无论工程类型如何，所有影像工程都必须提供可指导您完成基于影像的任务的工程说明。该工程说明在创建工程时定义，并且可以在打开工程时随时查看。要查看工程说明，请完成以下步骤：

  1. 在影像工程中，单击影像开发画布右上角的“更多”按钮。
  2. 在工程工具下，单击工程说明。

    工具面板随工程说明动态加载。

  3. 单击关闭按钮或选择其他工具，以关闭工程说明。

包含参考图层的 Web 地图

任何工程类型在创建时可选择包括其中含有地理空间参考图层的现有 Web 地图，以便为影像工程提供上下文。

打开包含 Web 地图的工程时，Web 地图将显示在地图聚焦面板上，且工程特定影像叠加在顶部。如果 web 地图包含参考图层，则这些图层将叠加在地图焦点面板和图像焦点面板中的图像顶部。可以打开或关闭这些图层，或者与其进行交互以查看其他信息。

要在图层列表上下文图层部分的每个视图中切换打开和关闭参考图层，请完成以下步骤：

  1. 在影像工程中，单击图层列表。
  2. 在上下文图层部分中，单击视图按钮以切换打开或关闭各个地图参考图层。
  3. 也可以在地图聚焦面板中单击 Web 地图中的要素。

    将出现一个窗口，其中包含有关相应要素的详细信息。

  4. 单击右上角以展开或关闭该窗口。

可编辑采集图层

打开观测点影像工程时，时间滑块和一组观测点工具将作为影像工程的一部分包含在内。时间滑块和观测点工具将直接与创建工程时所定义的可编辑采集图层相关联。

注：

可编辑采集图层必须包含创建观测点时所需的字段。

时间滑块

打开观测点影像工程时，将默认打开图像元数据表上方的时间滑块，并且时间滑块将直接与工程中的任何可编辑采集图层相关联。每个观测点图层都具有一个时间元素，可以使用交互式时间滑块来过滤图层。可以使用工程图层标题中的按钮来切换打开和关闭时间滑块。

时间滑块具有两个控点，分别表示开始和结束时间。默认情况下，时间滑块将设置为打开工程的当前日期和时间。可以调整时间以查看工程图层内的任何先前内容。

注：

如果观测点工程中不包含启用时间的观测点图层，则将显示一个对话框，提示您可以在此图层中采集新的观测点，但无法查看之前的观测点。

要调整时间，请完成以下步骤：

  1. 单击开始时间控点，沿时间轴将其拖动到所需开始时间。
  2. 单击结束时间控点，沿时间轴将其拖动到所需结束时间。

    观测点图层将根据定义的开始和结束时间动态更新。

观测点工具

在 ArcGIS Excalibur 1.1 中，观测点影像工程中提供两个观测点工具：“创建观测点”和“创建多个观测点”。每个工具都允许您在包含已定义图标的选项板中进行选择，并与影像交互以采集和记录要素。

注：

选项板中图标的符号系统和样式将在托管要素图层中定义。

要访问观测点工具，请在影像工程中单击开发画布右上角的“更多”按钮。这些工具便会显示在工程工具菜单中的相应部分。您可以展开和折叠每个图层以使列表更加易于管理。

“创建观测点”工具

要使用“创建观测点”工具，请完成以下步骤：

  1. 在工程工具菜单中，单击创建观测点工具。

    图标选项板即会显示在工具面板中。

    注：

    如果基础可编辑采集图层由单个值进行符号化和样式化，则可能会出现单个图标。

  2. 从选项板中选择一个图标并单击影像。

    该图标即会显示在可见的聚焦视图中。工具面板将从图标选项板变为一个表单，可以在该表单中根据可编辑采集图层的已定义属性输入信息。

    注：

    表单中显示的属性反映了基础采集图层中的属性。此视图支持属性域。

  3. 单击查看自动填充字段按钮来查看已自动填充的属性字段，以将采集的观测点与当前影像相关联。
    注：
    自动填充字段是支持将要素图层作为观测点影像工程中可编辑采集图层的必填字段。
  4. 输入全部信息后，单击提交。

    右下角即会显示一条通知，用于核实是否将观测点写入基础要素图层。

  5. 单击取消，可在当前观测点写入基础要素图层之前将其删除。
  6. 要编辑或删除现有观测点，请确保创建观测点工具已打开，然后直接单击图像上的相应观测点图标。

“创建多个观测点”工具

“创建多个观测点”工具与“创建观测点”工具类似，但允许您批量采集相同类型的观测点，然后提交以将其写入基础可编辑采集图层，从而节省采集工作流所用时间。要使用“创建观测点”工具，请完成以下步骤：

  1. 在观测点工具菜单中，单击创建多个观测点工具。

    图标选项板即会显示在工具面板中。

    注：

    如果基础可编辑采集图层由单个值进行符号化和样式化，则可能会出现单个图标。

  2. 从选项板中选择一个图标并单击影像。

    该图标即会显示在可见的聚焦视图中。工具面板将从图标选项板变为一个表单，可以在该表单中根据可编辑采集图层的已定义属性输入信息。

    采集第一个观测点时，添加数量总数显示为 1。

  3. 再次单击该影像。

    采集第二个相同类型的观测点时，添加数量总数显示为 2。

  4. 单击该影像，直到相同类型的所有观测点都采集完成为止。
  5. 根据可编辑采集图层的已定义属性输入信息。
  6. 输入全部信息后，单击提交。
    注：
    所采集的每个观测点的单独条目都将写入包含相同属性信息的可编辑采集图层。唯一的区别在于 created_date、lasted_edited_date、Observation Collection Time (Local)、Observation_LatLon 和 ICS_XY（如果适用）。这些属性表示在图像上创建每个观测点时的正确信息。

    右下角即会显示一条通知，用于核实是否将观测点写入基础要素图层。

  7. 单击取消，可在当前观测点写入基础要素图层之前将其删除。
