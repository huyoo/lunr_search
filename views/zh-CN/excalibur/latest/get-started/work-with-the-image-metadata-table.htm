使用影像元数据表

在本主题中

  1. 展开或折叠影像元数据表
  2. 表格交互
  3. 显示所选影像
  4. 清除所选影像
  5. 查看元数据字段
  6. 草绘感兴趣区域
  7. 创建影像工程

影像元数据表允许您查看有关影像的信息并与每个影像进行交互。根据影像格式和类型，该表中包含有关表中每条记录的元数据。您还可以使用该表来选择影像、将特定影像设置为焦点影像，或创建影像工程。

展开或折叠影像元数据表

默认情况下，最初连接到影像开发画布时，影像元数据表将展开并位于地图焦点面板下方。单击影像元数据表按钮可折叠或展开该表。

表格交互

可以将鼠标悬停在表格中的影像记录上，以查看其在地图上以黄色高亮显示的相应覆盖区。通过单击行中的复选框可以选择影像，这会导致在地图上以青色高亮显示相应的覆盖区。选择影像后，您可以选择将该影像设置为焦点影像。

结果表也可以按字段以升序或降序排序。单击字段名称可执行此操作。

显示所选影像

默认情况下，影像图层在影像开发画布的地图焦点视图中处于选中状态，允许您查看影像元数据表中记录的全分辨率影像。影像元数据表中的显示所选内容选项允许您在地图聚焦面板中仅查看所选记录的影像。

  1. 在影像元数据表中，单击影像复选框以选择影像。
  2. 单击影像元数据表上的显示所选内容按钮。

    地图焦点面板将仅显示所选影像以及以青色标出的其覆盖区。所有其他影像覆盖区保持红色不变。

清除所选影像

清除所选影像可将焦点移回影像元数据表中的所有记录。在影像元数据表中，单击清除所选内容按钮。将发生以下情况：

  • 清除任何选定记录。
  • 清除地图焦点面板中的任何选定覆盖区。
  • 影像焦点面板关闭。
  • 打开地图焦点面板中的所有影像。
  • 地图焦点面板保持当前范围不变。

查看元数据字段

默认情况下，影像元数据表会显示所选的几个元数据字段。要查看更多或更少字段，请完成以下步骤：

  1. 在影像元数据表中，单击“添加”按钮可展开整个元数据字段列表。
  2. 在列表中，单击要在表中显示或隐藏的元数据字段的复选框。

    您的更改将动态显示在元数据表中。

  3. 完成显示和隐藏字段后，单击“添加”按钮或屏幕上的任意位置即可关闭窗口。

草绘感兴趣区域

草绘感兴趣区域选项允许您以交互方式在地图焦点视图上绘制一个区域来选择影像子集。使用影像 Web 服务选项或我的可用影像连接到影像 web 服务时，该选项非常有用，因为该选项允许您根据您所定义的感兴趣区域来关注影像子集。定义感兴趣区域后，影像元数据表将进行更新以反映与您的感兴趣区域相交的记录。地图焦点视图将更新与表中的相应记录对应的覆盖区。

  1. 单击影像元数据表上方的草绘感兴趣区域。
  2. 绘制您的感兴趣区域以缩小结果范围

    您的结果将在“影像元数据表”和“地图焦点面板”上进行过滤。

  3. 单击清除感兴趣区域按钮可将“地图焦点面板”重置为原始结果。
注：

仅当您通过“影像 Web 服务”或“我的可用影像”选项连接到画布时，才会启用草绘感兴趣区域按钮。如果通过“影像目录搜索”连接到画布，由于您已选择要查看的影像，因此该选项将不可用。

创建影像工程

创建影像工程允许您基于感兴趣的影像来组织和简化影像工作流。在影像元数据表中，您可以仅基于表中的选定影像来创建影像工程。

在影像元数据表中选择影像

要基于表中的选定影像来创建影像工程，请完成以下步骤：

  1. 在影像元数据表中，可通过选中影像的复选框来选择影像。
  2. 单击更多选项的溢出菜单。
  3. 单击从所选影像创建影像工程。

    随即打开新建影像工程指导工作流，您可以在其中选择工程类型。
