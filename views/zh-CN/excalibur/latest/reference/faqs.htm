什么是 ArcGIS Excalibur？

ArcGIS Excalibur 是一个基于 web 的影像应用程序，可通过直观的体验使基于影像的工作流更加现代化并对其进行增强。这些体验允许您在完全集成的工作流中搜索、发现和使用影像，以进行影像分析与开发。

有关详细信息，请参阅什么是 ArcGIS Excalibur。

安装 ArcGIS Excalibur 之前有哪些要求？

在安装 ArcGIS Excalibur 之前，需要 ArcGIS Enterprise 10.7 或更高版本的基础部署并使用 ArcGIS Server 对 ArcGIS Image Server 进行授权。您可以在一台或多台计算机上安装基础部署，其中任意计算机可以为物理、虚拟或云计算机。

有关硬件和软件要求的详细信息，请参阅 ArcGIS Excalibur 1.0 系统要求。

如何安装和授权 ArcGIS Excalibur？

通过使用与您的客户编号相关联的 Esri 帐户进行登录，可以从 My Esri 中下载 ArcGIS Excalibur。ArcGIS Excalibur 应该安装在已安装 Portal for ArcGIS 的计算机上。

ArcGIS Excalibur 安装完成后，必须获取授权文件并对其进行授权。

有关安装和授权的详细信息，请参阅安装和授权 ArcGIS Excalibur。

如何对 ArcGIS Excalibur 进行许可？

ArcGIS Excalibur 是一个高级应用程序，需要单独的门户许可文件。您必须拥有具备相应角色的 Creator 或 GIS Professional 用户类型，才能分配 ArcGIS Excalibur 许可。

有关详细信息，请参阅系统要求或为成员分配 ArcGIS Excalibur 许可。

ArcGIS Excalibur 是否可用于 ArcGIS Online？

否。ArcGIS Excalibur 1.0 在 ArcGIS Online 中不可用，仅适用于 ArcGIS Enterprise 10.7 及更高版本。

是否可以使用 API 对 ArcGIS Excalibur 进行自定义？

否。ArcGIS Excalibur 1.0 不支持使用 API​进行自定义。

ArcGIS Excalibur 是否专用于特定角色？

否。想要发现、分析、报告以及有效传播影像中所派生信息的所有人都可以使用 ArcGIS Excalibur。其中包括诸如国家安全、应急响应、地方和联邦政府、人道主义援助以及公共事业等行业的全源分析师、影像分析师以及影像和地理空间管理人员。

可以从何处获取影像 web 服务 URL？

可以通过以下任意方式获取影像 web 服务 URL：

  • Enterprise 门户的内容选项卡 - 对于图层类型影像图层，可以搜索我的内容或我的组织的内容。
  • ArcGIS REST 服务目录（例如， https://<host.domain.com>/<webadaptor>/rest/services）- 所有定义为 ArcGIS Image Server 服务（表示为 (ImageServer)）的服务均为受支持的服务。
  • 您的组织的 GIS 管理员。

有关使用影像 web 服务 URL 的详细信息，请参阅探索影像 Web 服务。

在影像目录中进行搜索时，为什么没有返回影像？

确保将地图范围或感兴趣区域过滤器设置为相应位置，在此位置处，针对位于目录搜索地图左下角的目录影像图层切换器中的所选影像服务图层存在影像。

如果图层切换器列表中没有可用图层，则将显示一条消息说明此情况。要填充目录影像图层切换器列表，请单击“结果”面板中的我的可用影像以启用感兴趣的影像服务图层，以便在影像目录搜索中使用。

为什么我的影像不支持在影像焦点视图中进行渲染？

影像焦点视图适用于具有传感器变换信息的影像，如有理多项式系数 (RPC) 或帧照相机。并非所有影像服务都包含具有此传感器信息的影像。

有关影像焦点视图的详细信息，请参阅使用影像视图。

为什么影像图层覆盖区中不显示影像？

如果启用了切换影像，或者由于在影像开发画布的地图焦点面板中针对影像服务设置的可见性比例范围，则影像可能未显示在影像目录中。请尝试在影像覆盖区上进一步放大以获取要显示的影像或影像集。

为什么一些影像服务图层具有更多测量选项？

测量工具会动态读取影像服务上的元数据和功能集。可用的测量选项将取决于您正在使用的影像服务图层。

为什么我的工具面板不显示工程工具？

工程工具仅当在影像开发画布中打开影像工程时才可用。创建观测点和创建多个观测点工具仅当打开观测点影像工程时才可用。

有关创建影像工程的详细信息，请参阅创建影像工程。

观测点图层是否需要特定方案？

是，观测点图层需要将在观测点影像工程中使用的特定方案。创建新的观测点影像工程时，可以选择符合所需方案的现有观测点图层，或者创建新的观测点图层，该操作可自动将所需方案添加到新图层。

有关观测点图层的详细信息，请参阅选择观测点图层。

如何更改影像工程的共享设置？

在创建新的影像工程时，可以在指导工作流过程中与您的组织共享影像工程或者保留为私有影像工程。影像工程保存在我的内容中名为 Excalibur Imagery Projects 的文件夹中，您可以在门户中更改影像工程的共享设置。

如果我从其他计算机访问应用程序，则我的自定义用户设置是否仍然有效？

会。用户设置将进行存储，并且将与您的门户身份一起保存，从而允许您在不同位置访问不同计算机上的 ArcGIS Excalibur 应用程序，同时保持您的自定义体验不变。

是否可以在其他 ArcGIS 应用程序中使用在 ArcGIS Excalibur 中创建的内容？

是。ArcGIS Excalibur 利用 ArcGIS 平台及其信息模型来使用和创建可应用于其他 ArcGIS 应用程序的内容。具体来说，在观测点类型影像工程中创建的内容将写入托管要素服务，并且可用于其他应用程序（如 Map Viewer 和 ArcGIS Pro）和 web 应用程序（如 Operations Dashboard 和 Insights for ArcGIS）。您也可以在 Esri Story Maps 模板中使用影像开发画布中的导出视图。
