ArcGIS GeoEvent Server 可以使用输入连接器接收来自几乎所有数据源的事件数据。输入连接器必须了解数据的接收方式及格式化方式。这些任务由适配器和传输进行处理，是每个连接器下的技术性实施。

要了解有关适配器和传输的详细信息，请参阅扩展 GeoEvent Server。

要查找其他的连接器？

如果可用的连接器不能提供您所需的灵活性，您可以通过 GeoEvent Manager 自定义连接器来使用 GeoEvent Server 软件开发工具包 (SDK) 开发您自己的连接器。

要了解自定义连接器或开发您自己的连接器的详细信息，请参阅扩展 GeoEvent Server。

对于新输入连接器有何构想？ 在 GeoNet 上的 Real-Time GIS 空间发表您的想法。
