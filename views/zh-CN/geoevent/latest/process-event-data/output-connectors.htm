ArcGIS GeoEvent Server 能够将接收到的事件数据发送到各种不同目标。输出连接器必须了解用来发送数据的通信信道并且能够将 GeoEvent 转换成预期的格式。这些任务由适配器和传输进行处理 - 每个连接器下的技术性实施。

要了解有关适配器和传输的详细信息，请参阅扩展 GeoEvent Server。

要查找其他的连接器？

如果可用的连接器不能提供您所需的灵活性，您可以使用 GeoEvent Manager 自定义现有连接器或创建您自己的连接器，亦或使用 GeoEvent Server 软件开发工具包 (SDK) 开发您自己的连接器。

要了解自定义连接器或开发您自己的连接器的详细信息，请参阅扩展 GeoEvent Server。

对于新输出连接器有何构想？ 在 GeoNet 上的 Real-Time GIS 空间发表您的想法。
