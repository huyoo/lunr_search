ArcGIS Enterprise 的常用客户端

ArcGIS Enterprise 用于扩展组织内部地图和 GIS 应用程序的使用。有许多客户端可供您选择实施，以便帮助组织成员与 ArcGIS Enterprise 门户中的地图及其他项目进行交互，这些客户端的示例如下。

可配置应用程序和应用程序构建器

利用可配置应用程序和应用程序构建器，可以通过组织内部或外部的现代且响应迅速的 web 应用程序来共享 web 地图。这些应用程序具有极大的灵活性，因为不要求使用编码进行创建，所以非常适用于非开发人员。您可以选择 AppStudio for ArcGIS、Web AppBuilder for ArcGIS 和可配置应用程序模板，这些选择都可以提供简单的自定义 web 应用程序构建方式，无需编写代码。

AppStudio for ArcGIS

AppStudio for ArcGIS 允许您将地图和 GIS 数据转换至原生应用程序，以便将其与企业中的其他成员共享或通过热门应用程序商店与公众共享。使用 AppStudio 构建的应用程序为跨平台应用程序，即可以在 Android、iOS、Linux、macOS 和 Windows 平台运行。当 AppStudio 包含于成员的用户类型中时，它将为创建应用程序提供低层代码解决方案。成员可以使用可配置应用程序模板构建应用程序、通过自定义代码扩展模板或从临时文件创建应用程序。管理员可为成员分配 AppStudio for ArcGIS Developer Edition 许可，以便成员访问高级应用程序模板及其他工具。使用此附加许可，成员可以创建适合公共应用程序商店（App Store 或 Google Play）和企业发布的自定义独立应用程序。具有内置许可或附加许可的成员可以通过应用程序启动器访问应用程序，也可以从 appstudio.arcgis.com 下载软件。

Web AppBuilder for ArcGIS

Web AppBuilder 是一种直观的所见即所得式应用程序，可用于构建 web 应用程序，而无需编写代码。利用 Web AppBuilder，您可以构建和自定义应用程序，这些应用程序可在桌面、平板电脑和具有即用型微件（如查询、地理处理、打印）的智能手机上运行。有关详细信息，请参阅门户用户帮助中的关于 Web AppBuilder for ArcGIS。

外业 ArcGIS 应用程序

借助 ArcGIS 基于位置的应用程序，您可以侧重于采集数据和影像、导航路径以及分配任务。即使断开连接，外业工作人员仍然可以使用这些应用程序。有关访问这些应用程序的详细说明，请访问 Esri 的适用于任何人的应用程序网站。

ArcGIS QuickCapture

ArcGIS QuickCapture 是一款外业数据捕获应用程序，允许您快速捕获数据。您可以在行驶中捕获资产或事件的位置和属性。工程作者可以决定按钮的排列、显示方式及其所采集的信息。该应用程序允许您通过设备捕获照片和传感器信息。

请参阅 ArcGIS QuickCapture 要求以获取所支持的 ArcGIS Enterprise 版本信息。

Collector for ArcGIS

Collector for ArcGIS 是一款移动数据采集应用程序，即使在断开连接的情况下，您也能轻松捕获新数据和更新外业中的现有要素，并无缝集成到 ArcGIS 中。借助 Collector，您可以通过使用 GPS 或单击地图来采集和更新数据，填写地图驱动表单，并且可以选择使用专业级 GPS 接收器。

请参阅 Collector for ArcGIS 要求以获取所支持的 ArcGIS Enterprise 版本信息。

Drone2Map for ArcGIS

Drone2Map for ArcGIS 可用于可视化和分析无人机拍摄的地理定位影像，帮助您从空中观测区域并监控变化情况。使用 Drone2Map 创建的 2D 和 3D 地图可共享至 ArcGIS Enterprise。

可通过 ArcGIS Enterprise 10.5 及更高版本配置 Drone2Map 许可。

Explorer for ArcGIS

Explorer for ArcGIS 是一款移动应用程序，允许组织中的每位成员随时随地访问其地图，即使在断开连接的情况下也能访问地图。您可对数据进行可视化、在您的地图中搜索地点和要素、查看资产信息、标记地图以高亮显示重要的要素以及使用罗盘获取到难以发现的资产的路线。

请参阅 Explorer for ArcGIS 要求以获取所支持的 ArcGIS Enterprise 版本信息。

Navigator for ArcGIS

Navigator for ArcGIS 可让移动工作人员在断开连接的情况下使用所供地图和/或自有数据到达所需位置。

可以通过 Navigator 配备ArcGIS Enterprise 许可。

请参阅 Navigator for ArcGIS 要求以获取所支持的 ArcGIS Enterprise 版本信息。

Operations Dashboard for ArcGIS

Operations Dashboard for ArcGIS 允许您实时监控人员、服务、资产和事件。使用 Operations Dashboard，您即可通过 ArcGIS Enterprise 门户创建并共享仪表盘。仪表盘包括图表、仪表、地图和其他可视元素，用于表示对实现目标最重要的活动和关键绩效指标。

Survey123 for ArcGIS

Survey123 for ArcGIS 简化了 ArcGIS 中以表单为中心的数据采集工作流。作为数据采集工作的重要部分，地理相对工作流来说是非常关键的，但并非核心的。Survey123 for ArcGIS 认识到这个事实，在 ArcGIS 中新增了行业中预期的表单复杂度。

请转至 https://survey123.esri.com/ 以创建表单并发布到 ArcGIS。您可以从 Apple、Google Play 或 Windows 应用程序商店获取 Survey123 for ArcGIS 移动应用程序。

请参阅 Survey123 for ArcGIS 要求以获取所支持的 ArcGIS Enterprise 版本信息。

Tracker for ArcGIS

Tracker for ArcGIS 在下列两个应用程序中使用位置追踪服务：Track Viewer Web 应用程序和 Tracker for ArcGIS 移动应用程序。Track Viewer Web 应用程序允许管理员创建追踪视图、定义追踪对象以及可以查看这些追踪的对象。Tracker 移动应用程序经过优化，能够在后台追踪位置，从而最大限度地减少对设备电池的影响。无论是否存在数据连接，该移动应用程序都能独立记录追踪，并使移动用户能够控制何时追踪以及何时不追踪他们。

请参阅 Tracker for ArcGIS 要求以获取所支持的 ArcGIS Enterprise 版本信息。

Workforce for ArcGIS

Workforce for ArcGIS 使用位置功能来实现更好的外业工作人员协作和团队合作。安排适当的工作人员前往适当的地点使用适当的工具进行适当的工作。调度员使用 Workforce web 应用程序来创建和分配工作、监控进度，以及实时查看工作人员的位置。移动工作人员使用 Workforce 移动应用程序来完成其个人的待完成列表和传达状态。

请参阅 Workforce for ArcGIS 要求以获取所支持的 ArcGIS Enterprise 版本信息。

办公室 ArcGIS 应用程序

ArcGIS 平台提供多个桌面和基于浏览器的应用程序，可以节省您的工作时间和金钱。有关访问这些应用程序等的详细说明，请访问 Esri 的适用于任何人的应用程序网站。

ArcGIS Pro

ArcGIS Desktop 还包括 ArcGIS Pro，该应用程序允许您在桌面上创建并使用空间数据。它提供用于显示、分析、编译和共享 2D 和 3D 环境中数据的工具。您可从 ArcGIS Pro 连接到 ArcGIS Enterprise 门户，然后在门户中使用地图和图层。此外还可将托管要素和切片图层发布到您的门户。可将从 ArcGIS Pro 生成的地图文件 (.mapx)、图层文件 (.lyrx) 以及布局文件 (.pagx) 添加到您的门户内容，以便与其他人共享您的地图和数据。

作为门户管理员，您可在门户内配置 ArcGIS Pro 许可，并且通过门户向您的组织成员分发 ArcGIS Pro。

为了获得最佳性能，请使用最新版本的 ArcGIS Pro。

ArcGIS Enterprise 兼容同时发布的 ArcGIS Pro 版本或仍提供支持的 ArcGIS Pro 的较早版本。ArcGIS Pro 还需要使用受支持版本的 ArcGIS License Manager。

ArcGIS Desktop

ArcGIS Desktop 是一组桌面应用程序，以 ArcMap 为中心，提供创作、编辑和分析空间数据的高级功能。您可从 ArcGIS Desktop 登录到门户，然后在您的门户中使用地图和图层。此外，您还可以将 GIS 服务从 ArcGIS Desktop 发布到您的门户。这些 GIS 服务（如地图、地理处理、定位器以及影像）是在 ArcGIS Enterprise 中执行的大多数操作的基础。

要将 ArcMap 与 ArcGIS Enterprise 配合使用，您必须使用相同版本或较早版本。例如，如果您有 ArcGIS Enterprise 10.7.1，请使用 ArcMap 10.7.1 或更早版本。

ArcGIS Business Analyst Enterprise

ArcGIS Business Analyst Enterprise 可用于在组织中共享人口分析和空间分析。此解决方案中包括一个开箱即用型 web 应用程序和 ArcGIS GeoEnrichment Service。商业房地产经纪人 、零售商、经济发展机构以及其他组织可通过 ArcGIS Business Analyst Enterprise 来评估市场以决定是否要进行扩大、整合或投资。

ArcGIS Earth

ArcGIS Earth 是一款轻量级桌面应用程序，允许您组织中的成员探索 2D 和 3D 数据，其中包括 KML 和 ArcGIS 服务。ArcGIS Earth 使用针对门户配置的底图、高程服务和地理编码设置。

请参阅 ArcGIS Earth 向导式浏览以获取所支持的 ArcGIS Enterprise 版本信息。

ArcGIS Excalibur

ArcGIS Excalibur 是一个 web 应用程序，可通过直观体验对基于影像的工作流进行现代化设计和增强。这些体验使您能够在完全集成的工作流中搜索、发现和使用影像，以进行影像分析与开发。有关详细信息，请参阅 ArcGIS Excalibur 帮助。

ArcGIS Maps for Office

ArcGIS Maps for Office（之前为 Esri Maps for Office）通过 Microsoft 加载项在 Microsoft Office 中提供制图功能。可将此加载项用于 Microsoft Excel，以在门户中创建可显示电子表格数据的地图。此外，还可以通过 PowerPoint 加载项将这些地图嵌入到演示文稿中。

可从 Esri 网站下载 ArcGIS Maps for Office。

有关 ArcGIS Maps for Office 版本的信息，请参阅 ArcGIS Enterprise 受支持环境。

ArcGIS Business Analyst web 应用程序

商业房地产经纪人、零售商、经济发展机构以及其他组织可通过 ArcGIS Business Analyst web 应用程序来评估市场以决定是否要进行扩大、整合或投资。

可以通过 ArcGIS Business Analyst 配备 ArcGIS Enterprise Web 应用程序许可。

Esri Maps for IBM Cognos

Esri Maps for IBM Cognos 可允许您从门户向 IBM Cognos Business Intelligence 报表添加动态地图和数据。

有关所支持 Esri Maps for IBM Cognos 版本的信息，请参阅 ArcGIS Enterprise 受支持环境。

ArcGIS Maps for SharePoint

ArcGIS Maps for SharePoint 通过 SharePoint 页面的 Map Web Part 在 Microsoft SharePoint 中提供制图功能。您可借此在 SharePoint 页面中显示门户中的内容。该应用程序还包含支持通过业务连通性服务 (BCS) 在空间上使用并在地理层面上丰富 SharePoint 列表或外部数据的工作流。

可从 Esri 网站下载 ArcGIS Maps for SharePoint。

有关所支持 ArcGIS Maps for SharePoint 版本的信息，请参阅 ArcGIS Enterprise 受支持环境。

注：

离线环境下不支持 ArcGIS Maps for SharePoint 3.0 和 3.0.1。如果您的组织无法访问 Internet，则无法通过 ArcGIS Enterprise 使用以下应用程序版本。

GeoPlanner for ArcGIS

GeoPlanner for ArcGIS 允许您在基于土地的规划和设计方面与门户组织的其他成员进行协作。

从 ArcGIS Enterprise 10.5 版本起，GeoPlanner for ArcGIS 随 Portal for ArcGIS 提供。要了解如何开始使用 GeoPlanner for ArcGIS，请参阅什么是 GeoPlanner 工程。

要使门户成员能够使用 GeoPlanner，您必须通过 ArcGIS Enterprise 配置 GeoPlanner 许可。

ArcGIS Insights

ArcGIS Insights 是一款通过门户访问的应用程序，可用于对 ArcGIS web 服务、Excel 电子表格和数据库中所存储数据执行迭代、探索性数据分析。

要使门户成员能够使用 Insights，您必须通过 Insights 配置 ArcGIS Enterprise 许可。

请参阅 ArcGIS Insights 要求以获取所支持的 ArcGIS Enterprise 版本信息。

Roadway Reporter for Esri Roads and Highways

Esri Roadway Reporter for Roads and Highways 是一款由 Esri Roads and Highways for Server 提供支持的应用程序，可基于以地图为中心的用户体验来创建里程、路段和道路记录报告。Roadway Reporter 可用于配置、保存和共享报告，并支持将报告生成为 Adobe PDF 或可在 Microsoft Excel 中进一步操作的 CSV 格式。

要使门户成员能够使用 Roadway Reporter，您必须具有 Esri Roads and Highways Server。

ArcGIS Enterprise 支持同等版本的 Roadway Reporter。例如，如果您有 ArcGIS Enterprise 10.8，请使用 Roadway Reporter 10.8。

适用于 Event Editor 和 Esri Roads and Highways for Server 的 ArcGIS Pipeline Referencing for Server

Event Editor 是一个以地图为中心的 Web 应用程序，支持 Esri Roads and Highways for Server 和 ArcGIS Pipeline Referencing for Server 的线性参考事件数据编辑。Event Editor 可用于搜索路径、定义事件图层组、查询和编辑数据、运行质量控制检查和创建红线路径。

要使门户成员能够使用 Event Editor，您必须具有可用的 Esri Roads and Highways for Server 和 ArcGIS Pipeline Referencing for Server。

ArcGIS Enterprise 支持同等版本的 Esri Roads and Highways for Server 和 ArcGIS Pipeline Referencing for Server。例如，如果您有 ArcGIS Enterprise 10.8，请使用 Esri Roads and Highways for Server 和 ArcGIS Pipeline Referencing for Server 10.8。

ArcGIS Enterprise 门户网站

组织中的多个成员将仅通过门户网站来访问门户内容。您可以根据自己的喜好来配置此站点，可以在首页展示地图和应用程序，并将地图和应用程序组织到库和组中。门户网站还包含 Map Viewer，可供经过身份验证的用户使用，以创建和使用地图。通过托管服务器配置门户后，用户还可根据自己的 shapefile、.csv 文件、文件地理数据库以及 GeoJSON 文件来创建要素图层。

ArcGIS API 和 SDK

无论您是否拥有开发经验，都可以创建和管理地理空间应用程序。构建整合了制图、可视化、分析等功能的 Web 应用程序、移动应用程序和桌面应用程序。有关详细信息，请参阅 ArcGIS for Developers。
