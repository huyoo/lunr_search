添加任务成员和团队

在本主题中

  1. 成员
  2. 团队

ArcGIS Mission 成员是任务中的参与者，可以直接互相协作，也可以按群组（又称团队）的方式进行协作。可以将任何门户用户添加为任务成员，但是它们必须具有应用程序的相应许可。

以下内容将介绍门户用户、任务成员与任务团队之间的主要区别：

  • 门户用户 - Portal for ArcGIS 中尚未分配至特定任务的授权用户。可将门户用户分配至任意数量的任务，但门户用户只能查看或参与以任务成员身份分配给他们的任务。
  • 任务成员 - 分配至任务的门户用户。默认情况下，任务所有者将是任务中的第一个成员。任务成员可以使用 ArcGIS Mission Manager 和 Responder 参与任务。未分配至任务团队的任务成员仍可以参与任务。
  • 团队成员：分配至任务中特定群组的任务成员。

成员

成员选项卡上有以下两个列表：门户用户和任务成员。任务所有者或管理员可以向任务中添加或从任务中移除门户用户。如果将门户用户添加到任务，则他们将成为任务成员。如果将其从任务中移除，则他们将恢复为门户用户。

可以使用搜索栏搜索和过滤这两个列表。非任务所有者或管理员的成员对成员选项卡具有仅查看访问权限。

将门户用户添加到任务

要将门户用户添加为任务的成员，请完成以下步骤：

  1. 单击查看任务。
  2. 单击任务的任务详细信息。
  3. 单击成员选项卡以查找门户用户列表。
  4. 单击用户名称或使用搜索栏查找要添加的用户，从而将其选中。
  5. 单击添加将其传输到任务成员列表中。

    添加或移除门户用户时，“总任务成员”字段将更新。

从任务中移除门户用户

要从任务中移除作为成员的门户用户，请完成以下步骤：

  1. 单击查看任务。
  2. 单击任务的任务详细信息。
  3. 单击成员选项卡以查找任务成员列表。
  4. 单击用户名称或使用搜索栏查找要移除的用户，从而将其选中。
  5. 单击移除将其传输到门户用户列表中。

    添加或移除门户用户时，“总任务成员”字段将更新。

团队

任务团队被设计为一种组织工具，允许将信息发送给特定的成员或团队而非整个任务。任务团队可根据组织需求进行创建。为任务分配任务成员后，可以将这些成员添加到团队中。一个任务可以包含任意数量的任务团队，且任务成员可以属于多个团队。非任务所有者或管理员的成员对团队选项卡具有仅查看访问权限。

创建任务团队

要创建任务团队，请完成以下步骤：

  1. 单击查看任务。
  2. 单击任务的任务详细信息。
  3. 单击团队选项卡。
  4. 在对话框中输入团队名称，然后单击添加团队。

    您的团队将显示在对话框下方的面板中，其中包含两列：“任务成员”和“团队成员”。

编辑任务团队

要编辑任务团队，请完成以下步骤：

  1. 单击查看任务。
  2. 单击任务的任务详细信息。
  3. 单击团队选项卡。
  4. 单击要编辑的团队的名称。
  5. 单击编辑。
  6. 输入新的名称，然后单击确定。

    您的团队已重命名。应用程序的底部会显示一条通知，用于确认此更改。

删除任务团队

要删除任务团队，请完成以下步骤：

  1. 单击查看任务。
  2. 单击任务的任务详细信息。
  3. 单击团队选项卡。
  4. 单击要删除的团队的名称。
  5. 单击编辑。
  6. 单击删除团队。
  7. 单击删除确认删除。

    您的团队已删除。应用程序的底部会显示一条通知，用于确认此更改。

向任务团队添加任务成员

要向任务团队添加任务成员，请完成以下步骤：

  1. 单击查看任务。
  2. 单击任务的任务详细信息。
  3. 单击团队选项卡。
  4. 手动或使用搜索栏选择要添加到团队中的任务成员。
  5. 单击添加。

    所选任务成员将从“任务成员”列表传输到“团队成员”列表。针对每个团队重复上述步骤。

从任务团队中移除任务成员

要从任务团队中移除任务成员，请完成以下步骤：

  1. 单击查看任务。
  2. 单击任务的任务详细信息。
  3. 单击团队选项卡。
  4. 手动或使用搜索栏选择您要从团队中移除的任务成员。
  5. 单击移除。

    所选任务成员将从“团队成员”列表传输到“任务成员”列表。

    注：

    可以随时将先前从任务团队中移除的任务成员再次添加到任务团队。
