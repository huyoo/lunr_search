为感兴趣点添加地理消息

  1. 创建一个 POI
  2. 为 POI 添加标题
  3. 拍摄 POI 的照片
  4. 将文件附加到 POI
  5. 为 POI 添加备注
  6. 将 POI 发送给其他人
  7. 更改 POI 的颜色

感兴趣点 (POI) 是放置在地图上的一个点，用于在地图上标记单个位置、事件或其他静态要素。POI 将直接添加到任务地图，且不使用用于创建线或面的绘图工具。

创建一个 POI

  1. 在地图上定位要创建 POI 的位置。
    放大以获得更高精度。
  2. 在所需位置上长按屏幕，至少持续一秒钟。
    将出现一个带有标线的放大镜，可让您准确地放置此点。
  3. 释放长按 操作。

    将有一个图钉放置在地图上的指定位置处，且屏幕底部会出现一个通栏。添加到地图按钮将变为可见。

通栏将显示点和地址信息的标题。默认标题为“放置的图钉”。点击通栏可将其展开，并会显示细节和这是什么选项卡。

  • 细节选项卡包含图钉的地址和坐标，将以十进制度坐标格式显示。
  • 这是什么？ 选项卡显示先前在 ArcGIS Explorer 中创建的所有弹出窗口。

如果确定不添加点，点击地图，则不会存储该点。

添加到地图仅在将 POI 放置到任务地图上后才显示。除非使用此按钮将 POI 添加到地图，否则不会保存任何 POI。点击添加到地图时，屏幕底部的通栏会展开，并显示以下选项：

  • 标题
  • 拍照
  • 附加
  • 备注
  • 发送至
  • POI 颜色

要取消该过程并移除您的点以及您对其所做的全部更改，请点击取消。要撤消更改而不移除点，请点击返回。要保存并发送 POI，请点击复选标记，而无论是否进行了以下任何自定义选择。

为 POI 添加标题

任何 POI 的默认标题均为 Point of Interest，但是您可以通过完成以下步骤来更改 POI 的标题：

  1. 创建一个 POI。
  2. 点击添加到地图。
  3. 点击标题字段。
  4. 输入新标题。

您可以继续自定义 POI，也可以点击复选标记最终确定 POI 并将其添加到地图中。

拍摄 POI 的照片

要为 POI 拍照，请完成以下步骤：

  1. 创建一个 POI。
  2. 点击添加到地图。
  3. 点击拍照。
  4. 使用设备的图像功能拍摄照片。
  5. 当您对照片感到满意后，点击确定。

如果您的照片已成功添加到 POI，则会显示该照片。可将多个照片添加到一个 POI 中。您可以继续自定义 POI，也可以点击复选标记最终确定 POI 并将其添加到地图中。

将文件附加到 POI

要将文件附加到 POI，请完成以下步骤：

  1. 创建一个 POI。
  2. 点击添加到地图。
  3. 点击附加。
  4. 设备的文件管理器将打开，允许您搜索设备上的文件。
  5. 点击要添加的文件。

如果文件已成功添加到 POI，则会显示该文件。可将多个文件添加到一个 POI 中。您可以继续自定义 POI，也可以点击复选标记最终确定 POI 并将其添加到地图中。

为 POI 添加备注

例如摘要、描述或事件报告等备注可以作为 POI 的一部分编写。要在 POI 中包括备注，请完成以下步骤：

  1. 创建一个 POI。
  2. 点击添加到地图。
  3. 点击备注。
  4. 填写备注字段。

备注完成后，您可以继续自定义 POI，也可以点击复选标记最终确定 POI 并将其添加到地图中。

将 POI 发送给其他人

POI 可以直接放置在地图上，也可以发送给特定的任务成员或团队。要发送 POI，请完成以下步骤：

  1. 创建一个 POI。
  2. 点击添加到地图。
  3. 点击溢出按钮。
  4. 点击发送至。
  5. 选择收件人。可以选择多个任务成员或团队。顶部的搜索栏可用于查找或过滤团队和成员列表。
  6. 在页面底部点击选择。

您可以继续自定义 POI，也可以点击复选标记最终确定 POI 并将其添加到地图中。

更改 POI 的颜色

POI 的默认颜色为橙色，但是您可以通过完成以下步骤对其进行更改：

  1. 创建一个 POI。
  2. 点击添加到地图。
  3. 点击标题旁边的颜色按钮。
  4. 点击所需的颜色。

您可以继续自定义 POI，也可以点击复选标记最终确定 POI 并将其添加到地图中。
