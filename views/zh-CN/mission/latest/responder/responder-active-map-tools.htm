地图工具

在本主题中

  1. 团队范围工具
  2. 测量工具
  3. 底图工具
  4. 书签工具
  5. 图层/图例工具

地图工具可让您更改与地图互动的方式，但与地理消息无关。可通过点击任务地图上的溢出按钮访问这些工具。

以下工具可用：

  • 团队范围 - 显示您的团队
  • 测量 - 在任务地图上测量距离或面积
  • 底图 - 允许您更改正在使用的底图
  • 书签 - 缩放到预定位置和范围
  • 图层/图例 - 图形或地图要素所显示的更改

团队范围和测量工具可用于快速查看地图要素并与之交互。底图、书签和图层工具都与任务地图交互，从而使您可以突出显示、排除特定地图数据或与之交互。

团队范围工具

利用“团队范围”工具，您可以在任务地图上以尽可能近的地图范围查看整个团队。该工具可用于测量团队覆盖的大致区域并确定任务团队的相对位置。要使用此工具，请完成以下步骤：

  1. 点击溢出按钮。
  2. 点击团队范围。

您的团队现在将显示在地图上。

测量工具

测量是通过在任务地图上绘制测量图形来进行的，该工具根据此图形提供数据。支持两种类型的测量：距离和面积。此工具的使用用途如下：

  • 用户与地图要素之间的距离
  • 用户在执行任务期间的出行总距离
  • 任务区域或地图要素的大小

测量包含用于创建图形的不同选项，包括选择要使用的度量单位。对于任何测量，请确保在创建测量图形之前选择正确的测量单位。

更改测量类型

警告：
使用此工具进行的测量，精度仅扩展到两位小数。尝试使用较大单位（例如，英里或平方英里）测量较短距离或较小面积时，会显示零值。因此，要使用数据，可能有必要更改测量类型。

距离测量是通过测量两点或一系列点之间的总距离来进行的。距离将以英尺、码、英里、米、千米或海里为单位。

将通过创建面并测量该面的内部区域来测量面积。面积将以平方英尺、平方码、平方英里、平方米、平方千米、英亩或公顷为单位。

可以在创建测量图形之前或之后更改测量单位，但是，更改测量类型会移除地图上已经存在的所有测量图形。

要更改测量类型，请完成以下步骤：

  1. 点击溢出按钮。
  2. 点击测量。
  3. 点击当前所选测量类型旁边的下拉菜单。
  4. 点击一种测量类型。
  5. 点击测量值旁边的下拉菜单。
  6. 点击一个测量单位。

创建测量图形

将在任务地图上手动绘制测量图形。这些图形不是永久性的，仅在测量工具处于打开状态且正在使用时才持续存在。测量图形将通过在地图上放置一系列与要测量的要素相对应的点来进行绘制。

要创建图形，请完成以下步骤：

  1. 点击溢出按钮。
  2. 点击测量。
  3. 将测量更改为适当的类型。
  4. 点击地图以在图形起始位置上放置一个点，无论是距离还是面积测量。
  5. 在测量图形中点击尽可能多的后续点。
  6. 在屏幕右下方查看测量的总距离或面积。

要在此过程中将您的位置用作测量点，请点击当前位置按钮。此方法可快速测量您自己与地图要素或其他用户之间的距离。

删除或撤消部分测量图形

使用“撤消”工具可以移除测量图形的最新绘制点，而无需删除整个图形。可以多次使用“撤消”工具，每次可移除图形中最近创建的点。

要使用此工具，请完成以下步骤：

  1. 创建测量图形。
  2. 识别图形中的错误。
  3. 点击撤消，直到错误被移除。
  4. 重新绘制测量图形中在移除错误时所移除的各个部分。

垃圾桶工具可让您删除整个测量图形并重新绘制，而无需关闭测量工具。如果识别出的错误是在绘制过程较早进行的，则可能要完全删除测量图形并重新绘制，而不要使用“撤消”工具。

底图工具

底图是任务地图的底层，以及其上放置的任何其他图层或图形。更改底图将更改作为地图一部分显示的要素，但不会更改图形、地理消息或其上所放置的其他图层。更改底图不会更改用户与任务之间的交互方式。

要更改底图，请完成以下步骤：

  1. 点击溢出按钮。
  2. 点击底图。
  3. 点击要使用的底图。选项包括但不限于以下各项：
    • 影像
    • 影像混合图
    • 导航
    • 街道图
    • 带标注的地形图
    • 地形图
  4. 对所选底图感到满意后，可关闭选择菜单。

    现在您的选择就成为了任务的底图。所有图形、图层或其他地图组成部分都将显示在其上方。

书签工具

书签会创建指向地图上重要位置的快捷方式，这对于地图导航很有用。

要使用书签，请完成以下步骤：

  1. 点击溢出按钮。
  2. 点击书签。
  3. 点击要使用的书签。

    任务地图会缩放到创建书签时选择的地图范围和缩放级别。

图层/图例工具

地图图层显示地理位置定位的地图图形。图层的显示可以在可见与不可见之间切换，这可让您显示或隔离在任何给定时间看到的图形。每种图形类型都有其自己的图层，可以独立打开或关闭。

要更改图层，请完成以下步骤：

  1. 点击溢出按钮。
  2. 点击图层/图例。
  3. 选中或取消选中要更改的图层。
  4. 关闭工具。

    图层会根据您的更改显示在地图上。

图例将显示有关各个图层的信息。
