Adding DGN and DWG data: Deployment guide

在本主题中

  1. Limiting the upload file size
  2. Deployment for Windows Server

This deployment guide shows you how to add DGN and DWG data visualization capability to the Event Editor.

注：

You will need one DGN or DWG file and one PRJ file with the same name to accomplish this task. The deployment is shown with a DGN file as an example.

  1. 启动 ArcCatalog。
  2. Double-click the Convert CAD to Feature Dataset tool under EventEditor > Tools > Location Referencing Server Tools.
  3. Select an input DGN file.

    This DGN file should already exist in your data directory.

  4. Select an input .prj file.

    This DGN file should already exist in your data directory.

  5. Click OK again.

    The Convert CAD to Feature Dataset tool runs.

  6. Click Close after the tool runs successfully.
  7. 启动 ArcMap。
  8. On the Geoprocessing menu, click Results.

    The Results window appears in ArcMap.

  9. In the Results window, right-click the Current Session node, then click Share As > Geoprocessing Service.

    The Share as Service dialog box appears.

  10. Click the Publish a service option.
  11. Click Next.
  12. In the Service name text box, type ConvertCADToFeatureDataset.
  13. Click Next.
  14. Either use an existing folder or create a new folder for housing the published service.
  15. Click Continue.

    The Service Editor window appears.

  16. Click the Geoprocessing tab and make sure that the Uploads check box is checked.
  17. Click the Parameters tab and check the View results with a map service check box.
  18. Click Publish.

    Two services are created as a result. You can access them in ArcCatalog. The DGN visualization capability is now enabled in the Event Editor.

    • ConvertCADToFeatureDataset (GPServer)
    • ConvertCADToFeatureDataset (MapServer)

Limiting the upload file size

Use the following steps to limit the size of the DGN file.

  1. Browse to your \\<EventEditor>\ folder and open the Config.json file in a text editor such as Notepad.
  2. Search for the text maxUploadBytes and provide the value max file size in bytes (10485760 in this case).

    The file size limit will be shown as megabytes (MB) to users in the Add DGN Data dialog box.

Deployment for Windows Server

Use the following steps when deploying Event Editor on Windows Server 2012 (IIS 8).

  1. Go to the Internet Information Services (IIS) Manager by clicking Control Panel > System and Security > Administrative Tools > Internet Information Services (IIS) Manager.
  2. Expand the Server node and click Sites > Default Website.
  3. Right-click the EventEditor folder and click Convert to Application.
  4. In the Add Application dialog box, click Select.
  5. Choose the application pool that uses .NET Framework 2.0 with integrated pipeline mode.
