配置 ArcGIS Pro 许可

在本主题中

  1. 为组织配置 ArcGIS Pro 许可
  2. 在 ArcGIS Enterprise 门户中配置 ArcGIS Pro 许可
  3. 在 ArcGIS Pro 中连接到许可门户
  4. 配置备份 ArcGIS License Server Administrator
  5. 通过 ArcGIS Enterprise 门户分发 ArcGIS Pro

ArcGIS Pro 应用程序是 ArcGIS Desktop 的一部分，允许您在桌面上创建并使用空间数据。它提供用于在 2D 和 3D 环境中显示、分析、编译和共享数据的工具。

许可 ArcGIS Pro 的方式有以下几种：

  • 为 ArcGIS Enterprise 门户成员配置指定用户许可。
  • 为 ArcGIS Online 组织成员配置指定用户许可。
  • 使用 License Manager Server 实施浮动版许可。
  • 使用单机版许可。
了解有关授权 ArcGIS Pro 许可的详细信息

注：

使用上述任一方法为 ArcGIS Pro 安装配置许可后，即可从 ArcGIS Pro 连接到其他门户。

本主题涵盖了第一种许可方法：为 Portal for ArcGIS 成员配置指定用户许可。

作为为门户成员分配 ArcGIS Pro 许可的门户组织管理员，您需要执行以下操作：

  • Purchase and download a license file through My Esri.
  • 通过组织的 ArcGIS License Server Administrator 配置 ArcGIS Pro 许可。
  • 通过门户网站为您的门户组织成员配置 ArcGIS Pro 许可。

只有 ArcGIS Pro 可通过门户组织进行授权。其他 ArcGIS Desktop 应用程序（ArcMap 和 ArcCatalog）均使用本地 ArcGIS Administrator 应用程序来设置级别和扩展模块。

为组织配置 ArcGIS Pro 许可

使用以下步骤配置 ArcGIS Pro 许可。您指定用于管理许可的门户被称作许可门户。该门户十分重要，因为 ArcGIS Pro 会连接至此门户来获取组织成员的许可信息。如果您的组织具有多个门户，请确保您指定的门户是您想要用来管理许可的那个门户。只能有一个门户使用 ArcGIS License Server Administrator；您无法配置多个许可门户。但您可以选择使用一个备份 ArcGIS License Server Administrator 来配置该门户。

  1. 如果您尚未执行此操作，请查看 ArcGIS License Manager 系统要求并安装软件。可从 My Esri 下载安装程序。

    此表显示了支持的 ArcGIS License Manager 版本，从而通过 ArcGIS Enterprise 为 ArcGIS Pro 启用指定用户许可。

  2. 打开 ArcGIS License Server Administrator 并启动许可服务（如果未运行）。单击授权选项卡。
  3. 选择版本下拉对话框中的指定用户并单击立即授权。
  4. 选择我已收到来自 Esri 的授权文件，现在准备完成授权过程，然后浏览至用于许可 ArcGIS Enterprise 门户的许可 (.json) 文件。单击打开。
  5. 单击下一步。随即显示一个对话框，指示已应用了指定用户许可。单击确定。对话框可能会显示几秒钟，指示正在重新读取许可。
  6. 单击可用性选项卡并从下拉对话框中选择指定用户。确认显示了预期授权。列出的总数将是指定用户实际数量的三倍。这是因为每个指定用户最多允许打开三个同步 ArcGIS Pro 会话。
  7. 单击确定并关闭 ArcGIS License Server Administrator。
  8. 如果需要更新 ArcGIS License Manager，请打开 web 浏览器并以组织中默认管理员角色成员的身份登录到 ArcGIS Portal Directory。URL 地址格式为 https://portal.domain.com:7443/arcgis/portaladmin。
  9. 浏览至主页 > 许可 > 更新许可管理器。
  10. 在许可管理器信息字段中，指定 ArcGIS License Server Administrator 计算机的主机名。

    {
"hostname":"portal.domain.com"
"port":27000
}

  11. 单击 Update。

您的组织现在已配置了 ArcGIS Pro 许可。

在 ArcGIS Enterprise 门户中配置 ArcGIS Pro 许可

为您的组织配置完 ArcGIS Pro 许可后，请将 ArcGIS Pro 许可分配给您自己及您的成员。有关完整说明，请参阅管理许可。相应地配置完许可后，请将 ArcGIS Pro 按照以下描述连接至许可门户。

许可：

仅可以将 ArcGIS Pro 许可配置给具有兼容用户类型的门户成员。

需要进一步控制离线使用 ArcGIS Pro 的组织可以阻止离线使用组织内的所有 ArcGIS Pro 许可。要针对您的组织执行此操作，管理员或具有许可权限的成员必须选中 ArcGIS Pro 选项卡顶部的禁用离线使用 ArcGIS Pro 复选框。选中此复选框后，将允许已离线使用 ArcGIS Pro 的用户检回许可，但是将来任何尝试离线使用 ArcGIS Pro 许可都将会产生错误。

在 ArcGIS Pro 中连接到许可门户

将 ArcGIS Pro 连接到您的许可门户。许可门户控制门户内所有成员帐户的软件授权。如果您的组织具有多个门户，请确保门户是您想要配置用于管理许可的那个门户。

  1. 启动 ArcGIS Pro。
  2. 在 ArcGIS 登录页面的底部，单击配置许可选项。
    注：

    如果绕过 ArcGIS 登录页面，请单击 ArcGIS Pro 开始页面左下角的设置。在“设置”页面上，单击左侧的许可，然后单击配置许可选项。

  3. 选择 ArcGIS Enterprise，然后输入许可门户的 URL。URL 的格式为 https://webadaptorhost.domain.com/webadaptorname。ArcGIS Pro 必须有访问许可门户和已获得许可授权的 ArcGIS License Manager 计算机的权限。
  4. 单击确定。
  5. 输入凭据并单击登录。

ArcGIS Pro 现已连接至许可门户。ArcGIS Pro 将继续在每次启动时连接至许可门户。通过执行上述步骤可更改许可门户。

如果您正在使用 ArcGIS Pro 但应用程序角落中显示未登录指示，则可能需要登录至包含您的内容的门户才能浏览和共享文件。通过单击工程选项卡并选择门户可查看可用门户。要连接至其他门户，请右键单击门户的 URL 并选择设为活动门户。还可以通过单击添加门户来添加其他门户。

配置备份 ArcGIS License Server Administrator

您可以选择为许可门户配置备份 ArcGIS License Server Administrator。例如，如果主 ArcGIS License Server Administrator 离线，则门户将故障转移到您指定的备份许可服务器。这样，您便可以将组织的停机时间降至最低。

  1. 按照上述“为您的组织配置 ArcGIS Pro 许可”部分完成步骤 1 到 7。确保您指定了相同的 ArcGIS Pro 配置信息。
  2. 打开 Web 浏览器并以组织中默认管理员角色成员的身份登录到 ArcGIS Portal Directory。URL 地址格式为 https://portal.domain.com:7443/arcgis/portaladmin。
  3. 单击主页 > 许可 > 更新许可管理器。
  4. 在许可管理器信息字段中，指定备份 ArcGIS License Server Administrator 的主机名称（例如，backuplm.domain.com）：
    {
  "hostname": "primarylm.domain.com,backuplm.domain.com",
  "port": 27000
}
  5. 单击 Update。

    Portal for ArcGIS 自动重启。重启完成后，即可使用您指定的备份许可服务器配置门户。

    6. 旧版本：

    在之前的 Portal for ArcGIS 版本中，您需要手动重启门户才能将更改应用至 ArcGIS License Server Administrator 配置。从 10.3.1 版本开始，无需再手动重启门户。

通过 ArcGIS Enterprise 门户分发 ArcGIS Pro

可为 ArcGIS Pro 安装文件创建门户应用程序项目，这样得到 ArcGIS Pro 使用许可的门户成员便可轻松进行安装。

使 ArcGIS Pro 可通过 URL 进行安装

将 ArcGIS Pro 安装文件部署至 web 服务器。

  1. 从 My Esri 下载 ArcGIS Pro 安装文件。
  2. 在 web 服务器上托管 ArcGIS Pro 自解压 .exe 安装文件。

    这将为您提供添加应用程序项目所需的 URL。

创建群组

创建一个组，并将组织成员邀请或添加至该组。

  1. 登录门户。
  2. 创建群组。
  3. 邀请或添加您想要访问的所有成员，并安装 ArcGIS Pro。

    该成员可以是获得 ArcGIS Pro 使用许可的任意成员。

添加和共享 ArcGIS Pro 应用程序项目

将应用程序项目添加至指向 ArcGIS Pro URL 的门户。与创建的组共享应用程序项目。

  1. 单击内容页面的我的内容选项卡。
  2. 单击添加项目。
  3. 在添加项目对话框中，从下拉列表中选择应用程序。
  4. 选择应用程序。
  5. 输入项目标题；例如 ArcGIS Pro。
  6. 输入将帮助门户成员查找项目的标签。
  7. 单击添加项目。

    将打开项目的项目详细信息页面。

  8. 单击共享，然后选择为共享安装文件而创建的组。
  9. 单击确定关闭共享对话框。
  10. 单击编辑并更新项目。
  11. 在摘要字段中输入安装项目的简短描述。
  12. 输入更加详细的描述，并在描述字段中创建安装文件链接。
    1. 单击描述字段中的创建链接按钮。
    2. 在链接属性对话框中，输入或粘贴 ArcGIS Pro 可执行文件的 URL。例如，输入 http://webserver.domain.com/ArcGISPro/ArcGISPro.exe。
    3. 输入链接的描述性文本，例如下载 ArcGIS Pro 安装ArcGIS Pro。
    4. 单击设置以创建链接。
  13. 输入有关哪些人员可在访问和使用限制字段中使用该应用程序的信息。
  14. 单击应用程序注册部分中的注销应用程序。
  15. 在属性部分的 URL 字段中输入安装文件的 URL。
  16. 单击保存。

ArcGIS Pro 安装文件现在可用于其共享至的组的所有成员。

门户成员可登录并搜索应用程序项目，或从组中访问项目。然后，他们便可将安装可执行文件下载至其本地计算机，并安装 ArcGIS Pro。
