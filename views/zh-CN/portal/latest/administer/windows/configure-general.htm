配置常规设置

默认管理员以及具有正确权限的人员可以为您的站点配置常规设置。您可以为站点输入名称、缩略图和描述。也可选择默认语言、创建自定义联系人链接以及为由组织成员创建的 web 应用程序定义共享主题。门户网站的默认语言是英语。要更改语言设置，请以管理员身份登录并按照以下步骤进行操作。

  1. 验证您是否以默认管理员或具有正确管理权限的自定义角色身份进行登录。
  2. 单击站点顶部的组织，然后单击设置选项卡。
  3. 单击页面左侧的常规。
  4. 配置以下任意常规设置：
    • 徽标和名称
      注：

      要配置此设置，您必须具有管理组织网站的权限。

    • 描述
      注：

      要配置此设置，您必须具有管理组织网站的权限。

    • 组织摘要
      注：

      要配置此设置，您必须具有管理组织网站的权限。

    • 语言
      注：

      要配置此设置，您必须具有管理组织网站的权限。

    • 管理联系人
      注：

      要配置此设置，您必须具有管理安全设置的权限。

    • 链接
      注：

      要配置此设置，您必须具有管理组织网站的权限。

    • 共享主题
      注：

      要配置此设置，您必须具有管理组织网站的权限。

    • 应用程序启动器
      注：

      要配置此设置，您必须具有管理组织网站的权限。

  5. 单击保存保存您所做出的更改。

徽标和名称

在名称字段中，输入希望代表网站的名称。名称最多可以包含 55 个字符。要添加徽标，请单击单击更改缩略图并浏览到计算机上的图像文件。可接受的图像格式包括 PNG、GIF 和 JPEG。要移除现有徽标，可单击缩略图上角的 X。

名称将显示在图库标题及组织页面中。徽标显示在主页和组织页面中。要获得最佳效果，徽标大小应为 65 像素（宽）x 65 像素（高）。如果徽标是其它尺寸，则主页之外所有页面上的徽标将会被调整为 65 x 65，而且可能会出现扭曲。主页上的徽标大小将保持不变，并显示在通栏图像中。

如果使用 HTML 配置主页通栏，则不会出现徽标和门户名称，除非将其包括在 HTML 中。

描述

输入组织描述。要粘贴或输入自己的受支持的 HTML 代码，请切换至 HTML 源代码视图。

如果希望该描述显示在主页上，请选中在主页底部显示描述。该描述将显示在通栏下。

默认文本插入到主页底部的描述中。要移除或更改此文本，请输入您自己的描述。

汇总

输入您组织的简短摘要，摘要内容将显示在与自定义应用程序和网站相关联的登录页面中。摘要最多可包含 310 个字符。

语言

选择网站的默认显示语言。您指定的语言将确定显示在用户界面中的语言，以及时间、日期和数值的显示方式。默认情况下，组织语言设置为英语。要为网站指定其他语言，请从下拉列表中选择所需语言，或者选择浏览器默认选项以使用浏览器的语言设置显示网站。

各成员均可在其用户个人资料页面设置其自己的首选语言。这使得用户在登录至网站后可覆盖组织语言。不具有 ArcGIS 组织帐户的用户无法指定其自己的语言。如果有不具有组织帐户的用户访问您的网站，请考虑使用浏览器默认选项设置。

注：

更改组织区域设置将仅更新门户的用户界面，以显示您的首选语言。要查看某非英语语言的门户帮助，需要从 My Esri 中下载并安装语言包。

数字和日期格式

如果将语言设置为英语或西班牙语，还可以指定 ArcGIS Enterprise 门户中数字和日期的显示方式， 方法是从数字和日期格式下拉菜单中选择一个区域设置选项。例如，如果语言设置为英语，则您可以选择使用由澳大利亚、英国、加拿大或美国的通用区域设置数据存储库 (CLDR) 定义的格式来显示日期和数字。 各个组织成员可通过个人资料页面更改数字和日期格式。

注：

目前，只有当语言设置为英语或西班牙语时，您才能够设置数字和日期格式。数字和日期格式设置不会反映在属性表、地图标注或动画时间切片中显示的日期值中，也不会反映于查看的弹出窗口中。但是，您可以使用 Text() 函数创建简单的 Arcade 属性表达式，以更改弹出窗口中日期的格式。

管理联系人

选择管理员或列为 ArcGIS Enterprise 门户联系方的管理员。如果提供了这些联系人，则指定管理员的名称将在组织页面上对所有非管理员组织成员可见。如果管理联系人希望提供联系人详细信息（例如电子邮件地址），则应将其添加到用户配置文件的个人简历部分。

链接

选中联系我们复选框，然后在站点页脚处输入网站 URL 或自定义联系人链接 mailto:。请注意，如果您的组织允许匿名访问，任何找到您的 URL 的用户都能够查看您的联系人信息。取消选中此复选框可隐藏联系人链接。

共享主题

请指定您要应用到支持的可配置 web 应用程序、Story Maps 和 Web AppBuilder 应用程序的标识颜色和徽标（如有需要）。如果要指定徽标，可以在徽标图像框中输入徽标的 URL 引用，或者单击上传徽标以上传图像文件，此文件将存储为内容中的公共项目。徽标 URL 引用必须使用 HTTPS 协议，否则将不会被识别为有效。也可以通过在徽标链接框中输入目标网站 URL 来设置可单击徽标。保存更改后，这些设置将自动应用到任何新的受支持应用程序，其中，这些应用程序由您的组织成员创建。目前，大多数 Story Maps 和其他可配置应用程序支持共享主题。

应用程序启动器

向应用程序启动器窗口添加 web 应用程序，以向组织成员提供这些应用程序。对于具有相应许可的成员，获得许可的应用程序将自动显示在应用程序启动器中。要使其他 web 应用程序可供组织成员使用，请参阅在应用程序启动器中管理应用程序。
