配置地图

  4. 可配置应用程序
  5. Web 样式
  6. 单位
  7. 分析图层

默认管理员以及具有正确权限的人员可以选择您的组织将用于创建地图、场景和应用程序的底图和可配置应用程序。您还可以设置默认底图、范围和单位。如果您的组织需要使用 Bing 地图，也可添加 Bing 地图密钥。

  1. 验证您是否以默认管理员或具有管理权限的自定义角色身份进行登录以管理组织网站。
  2. 单击站点顶部的组织，然后单击设置选项卡。
  3. 单击页面左侧的地图。
  4. 配置以下任意地图设置：
    • 底图库
    • 默认底图和范围
    • Bing 地图
    • 可配置应用程序
    • Web 样式
    • 单位
    • 分析图层
  5. 单击保存保存您所做出的更改。

底图库

您有多种方式为 Map Viewer、Scene Viewer 以及您组织中的应用程序配置底图库。

注：

底图必须满足 Scene Viewer 切片方案要求才能显示在 Scene Viewer 底图库中。

请选择以下选项之一：

  • Esri 默认矢量底图 - 从下拉菜单中选择默认并选中在支持的 ArcGIS 应用程序中使用 Esri 矢量底图框。
  • Esri 默认栅格底图 - 从下拉菜单中选择默认并清除在支持的 ArcGIS 应用程序中使用 Esri 矢量底图框。
  • 自定义底图库群组中的底图 - 要仅使用您创建的底图库群组中的底图，请从下拉菜单中选择该群组，然后清除单击“保存”时将 Esri 默认底图共享到该群组框。要更改底图在图库中的排序方式，请在排序依据下拉菜单中选择一个字段。要更改底图的排列方式，请选中或清除升序复选框。如果在此处更改了排序字段或顺序，群组页面以及嵌入图库或图库应用程序中的项目也将被更改。
  • 除 Esri 默认栅格或矢量底图外来自自定义底图库中的底图 - 如果选择预定义的底图库群组，您可以选中单击“保存”时将 Esri 默认底图共享到该群组框，然后选择包含栅格底图或矢量底图。在您的群组中包含 Esri 底图时，您也可以对群组进行排序和重新排序。如果在此处更改了排序字段或顺序，群组页面、嵌入图库以及图库应用程序中的项目也将被更改。

    要添加更多 Esri 默认底图（例如，在站点更新后进行添加），请再次选中该复选框并保存设置。要移除 Esri 默认底图，请打开底图库群组并移除不再需要的项目。

单击保存应用底图库设置。

默认底图和范围

如果为底图图库设置选择自定义底图群组，则请从想要在用户使用 Map Viewer 创建新地图或使用 Scene Viewer 创建新场景时打开的群组中选择一张底图。如果选择 Esri 默认作为底图库，则将对新地图和场景显示默认 Esri 底图。

注：

Scene Viewer 将仅使用特定的默认底图（如果其受支持）。

默认底图将在您设置的默认范围内打开。要将默认范围设置为当前所选默认底图的范围，请单击使用地图范围作为默认范围。要设置其他范围，请单击默认范围部分中的选择范围，然后输入数值或绘制范围。提供了定位器以协助您查找具体的地点或地址。

Bing 地图

如果组织希望在地图中使用 Bing 地图，请输入 Microsoft 提供给您的 Bing 地图密钥。

如果启用该选项，组织中的成员将会在从 web 添加图层中看到 Bing 选项。

选中复选框后，将允许在组织成员公开共享的地图中使用 Bing 密钥（您的 Bing 密钥将显示给公众）。此选项仅在组织允许匿名访问站点时才显示（在安全性部分中）。 如果禁用匿名访问，则公众将不再能够通过 Bing 地图查看您的地图。要重新启用通过 Bing 地图进行地图公共访问，则需要重新选中匿名访问并重新选中允许在公开共享的地图中使用 Bing 密钥的选项。

如果存在关于在 ArcGIS Enterprise 中使用 Bing 地图的问题，请联系客户经理或当地的 Esri 办事处。

可配置应用程序

选择您想在可配置应用程序库中使用的应用程序模板所在的群组。

选中以将适用于您语言的当前 Esri 默认的可配置应用程序添加到您的群组。要添加其他 Esri 默认的可配置应用程序（例如，在站点更新后进行添加），请再次选中该复选框并保存设置。还可以通过与群组共享项目来添加默认设置中不包括的 Esri 公共可配置应用程序（例如，已停用的应用程序）。要移除添加到群组中的可配置应用程序，请打开该群组并移除不再需要的项目。

要更改应用程序在图库中的排序方式，请选择排序依据旁下拉菜单中的字段。要更改应用程序排序的方式，请选中或取消选中升序旁的复选框。如果在此处更改了排序字段或顺序，则群组页面以及应用程序中的项目也将发生更改。

提示：

每个应用程序模板都会提供一个包含应用程序模板源代码的可供下载的 ZIP 文件以及一个可在 web 浏览器中显示的预览版本。下载 ZIP 文件并自定义模板以将其部署到自己的 web 服务器上。您还可以将自定义模板作为可配置的 Web 制图应用程序项目添加回门户中，将源代码附加到该项目，以便其他用户可以找到并使用该模板。如果您随后与指定为可配置应用程序群组的群组共享自定义模板，则其他门户成员将在可配置应用程序库中看到自定义模板选项。

Web 样式

选择一个包含用于 Scene Viewer 的样式的群组。创建 3D web 样式项目的 ArcGIS Pro 用户可以将这些样式与该群组共享。用户在 Scene Viewer 符号库中设置点的样式时可以访问这些样式。

单位

选择美国标准或公制作为地图比例尺、测量工具、方向和分析的默认单位。美制将单位设置为英里、英尺和英寸；公制将单位设置为千米、米和厘米。成员可通过其个人资料页面对显示的单位进行设置。

分析图层

默认情况下，不会为门户配置一组特定的分析图层。作为门户管理员，您已决定一组适用于 Map Viewer 中标准要素、栅格和 GeoAnalytics 工具 中的特定图层，您可以将门户成员引导至这组特定图层。为此，可选择一个分析图层群组。有关详细信息，请参阅配置分析图层。

即使未指定分析图层群组，门户成员仍能通过使用浏览图层选项在门户中有权访问的任意图层上执行分析。请注意，门户成员必须使用浏览图层选项才能访问大数据分析工具中的大数据文件共享。
