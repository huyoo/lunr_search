要素分析工具区别

在本主题中

  1. 标准分析工具和 GeoAnalytics 工具 之间的主要区别
  2. 单个工具区别

如果标准空间分析和 GeoAnalytics 工具 工具集中均包含某个工具，则在该工具的执行方面可能存在区别。本主题将介绍两个工具集中均包含的工具之间的区别。其中一些区别是工具集中的所有工具共有的，一些仅适用于特定工具。有关各个工具集中包含的工具的概述，请参阅标准和 GeoAnalytics 工具 要素分析比较。

标准分析工具和 GeoAnalytics 工具 之间的主要区别

在标准分析工具和 GeoAnalytics 工具 中提供的 GeoAnalytics Server 之间始终存在一些功能上的区别。

  • GeoAnalytics 工具 可以分析大数据文件共享中的数据（门户中的“大数据文件共享”项目），而标准分析工具不可以。有关大数据文件共享的详细信息，请参阅大数据文件共享入门。
  • 在许多 GeoAnalytics 工具 中，GeoAnalytics Server 可以向分析应用时间步长，而标准分析工具不可以。
  • GeoAnalytics Server 可以选择计算在标准分析工具中不可用的字段上的以下统计数据：数值字段上的 COUNT、VARIANCE 和 RANGE以及字符串字段上的 COUNT 和 ANY。

单个工具区别

除了多个工具之间共有的区别之外，单个工具还具有一些较小的区别。下面列出了标准和 GeoAnalytics 工具 工具集中包含的工具以及特定区别的概述。

聚合点

标准要素分析和 GeoAnalytics Server 中的聚合点工具之间的区别如下：

  • 可以使用 GeoAnalytics 工具 版本的工具应用时间步长。

计算密度

标准要素分析和 GeoAnalytics Server 中的“计算密度”工具之间的区别如下：

  • GeoAnalytics 工具 版本的工具允许同时在多个字段上计算密度。
  • GeoAnalytics 工具 版本可以将密度转换成正方形或六边形图格。
  • GeoAnalytics 工具 版本可以应用时间步长。
  • GeoAnalytics 工具 工具可以指定是否应使用核或加权权重计算密度。
  • GeoAnalytics 工具 版本不允许折线输入数据。

创建缓冲区

标准要素分析和 GeoAnalytics Server 中的“创建缓冲区”工具之间的区别如下：

  • 可以在标准分析工具中指定多个缓冲区距离，但是在 GeoAnalytics 工具 中只能指定一个缓冲区距离。
  • 可以在 GeoAnalytics 工具 中使用字段或字段表达式创建缓冲区距离。标准分析中不含这些选项。
  • 可以在 GeoAnalytics 工具 中选择平面和测地线缓冲区。在标准分析中，不存在用于执行该操作的选项，将默认应用测地线缓冲区。
  • 可以在标准分析中选择环类型、侧类型和末端类型。GeoAnalytics 工具 中不含这些选项。
  • 可以通过以下方式在标准分析和 GeoAnalytics 工具 中进行不同的融合：
    • 在标准分析工具集中，可以选择是否融合重叠面。
    • 在 GeoAnalytics 工具 中，可以选择融合所有生成的要素、不融合或基于输出字段进行融合。还可以决定生成的要素为多部分要素还是单部分要素。如果想要重新创建已在标准分析中完成的融合重叠，您需要融合全部并允许生成多部分要素。
  • 在 GeoAnalytics 工具 中进行融合时，可以针对全部或字段计算统计数据。

查找热点

标准要素分析和 GeoAnalytics Server 中的“查找热点”工具之间的区别如下：

  • GeoAnalytics 工具 版本的工具需要您指定图格大小和邻域大小。
  • GeoAnalytics 工具 版本的工具可以应用时间步长。
  • GeoAnalytics 工具 版本不允许面输入数据。
  • GeoAnalytics 工具 版本不会应用按字段划分或边界面图层。

查找相似位置

标准要素分析和 GeoAnalytics Server 中的“查找相似位置”工具之间的区别如下：

  • GeoAnalytics 工具 版本允许将字段选择附加到结果。
  • GeoAnalytics 工具 版本将应用属性剖面匹配方法。

连接要素

标准要素分析和 GeoAnalytics Server 中的“连接要素”工具之间的区别如下：

  • GeoAnalytics Server 可应用时态连接。
  • GeoAnalytics 工具 和标准工具之间存在可用空间关系的差异。
  • 可应用一个 Arcade 表达式，指定想要加入到 GeoAnalytics 工具 版本工具中的要素。

合并图层

标准要素分析和 GeoAnalytics Server 中的“合并图层”工具之间的区别如下：

  • GeoAnalytics Server 不支持将数字字段与字符串字段相匹配，而标准工具支持将数字字段与字符串字段相匹配。

叠加图层

标准要素分析和 GeoAnalytics Server 中的“叠加图层”工具之间的区别如下：

  • 该 GeoAnalytics 工具 版本支持相交和擦除叠加类型，而标准工具支持相交、擦除和联合叠加类型。
  • 该 GeoAnalytics 工具 版本支持面与面、线与线以及点与点的叠加。标准分析工具还支持点与线、点与面，以及线与面。

对表格中的位置进行地理编码

GeoAnalytics 工具 无法使用 ArcGIS Online World Geocoder。

融合边界

标准要素分析和 GeoAnalytics Server 中的“融合边界”工具之间的区别如下：

  • GeoAnalytics 工具 可以在要素不重叠的情况下融合具有相同字段值的区域。而标准工具要求在要素重叠并且具有相同的字段值时才能对其进行融合。
