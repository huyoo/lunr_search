您可以为您的 ArcGIS Enterprise 组织购买应用程序，作为应用程序包和用户类型的一部分，也可以单独作为附加应用程序。这些应用程序将包含在您的门户许可文件中。

以下部分介绍了可以与 ArcGIS Enterprise 门户一起使用的应用程序及其许可方式。

AppStudio for ArcGIS

AppStudio for ArcGIS 允许您将地图和 GIS 数据转换至原生应用程序，以便将其与企业中的其他成员共享或通过热门应用程序商店与公众共享。使用 AppStudio 构建的应用程序为跨平台应用程序，即可以在 Android、iOS、Linux、macOS 和 Windows 平台运行。当 AppStudio 包含于成员的用户类型中时，它将为创建应用程序提供低层代码解决方案。成员可以使用可配置应用程序模板构建应用程序、通过自定义代码扩展模板或从临时文件创建应用程序。管理员也可为成员分配 AppStudio for ArcGIS Developer Edition 附加许可，以便成员访问高级应用程序模板及其他工具。使用此附加许可，成员可以创建适合公共应用程序商店（App Store 或 Google Play）和企业发布的自定义独立应用程序。

AppStudio for ArcGIS 随基本应用程序包一起包含在核心 Esri 用户类型中，例如 GIS Professional、Creator、Editor、Field Worker 和 Viewer。AppStudio for ArcGIS Developer Edition 可作为附加许可使用。具有内置许可或附加许可的成员可以通过应用程序启动器访问应用程序，也可以从 appstudio.arcgis.com 下载软件。

按照管理许可内的说明进行操作以查看许可详细信息和管理成员许可。

ArcGIS Business Analyst web 应用程序

商业房地产经纪人、零售商、经济发展机构以及其他组织可通过 ArcGIS Business Analyst web 应用程序来评估市场以决定是否要进行扩大、整合或投资。

要将 ArcGIS Business Analyst 和 ArcGIS Enterprise 一起使用，请参阅安装概述了解相关要求和说明。ArcGIS Business Analyst web 应用程序需要附加许可，每个门户成员都需要拥有一个许可，然后才能使用 web 应用程序。按照管理许可内的说明进行操作以查看许可详细信息和管理成员许可。

ArcGIS Configurable Apps

ArcGIS Configurable Apps 允许您通过 web 地图、场景和群组构建 web 应用程序，无需编写代码。ArcGIS Configurable Apps 随基本应用程序包一起包含在核心 Esri 用户类型中，例如 GIS Professional、Creator、Editor、Field Worker 和 Viewer。ArcGIS Configurable Apps 包含在门户中，无需额外安装。

ArcGIS Excalibur

ArcGIS Excalibur 是一个 web 应用程序，可通过直观体验对基于影像的工作流进行现代化设计和增强。这些体验使您能够在完全集成的工作流中搜索、发现和使用影像，以进行影像分析与开发。ArcGIS Excalibur 需要附加许可和额外安装。

ArcGIS Enterprise 站点

ArcGIS Enterprise 站点允许您为您的用户提供量身定制的 web 页面体验，可以帮助您将门户的权威内容共享到其他部门。成员可以直接转到您创建的自定义网页来导航与其相关的内容，而无需学习如何导航到门户。站点随基本应用程序包一起包含在核心 Esri 用户类型中，例如 GIS Professional、Creator、Editor、Field Worker 和 Viewer。无需额外安装。

ArcGIS Insights

ArcGIS Insights 是一款通过门户访问的应用程序，可用于对 ArcGIS web 服务、Excel 电子表格和数据库中所存储数据执行迭代、探索性数据分析。

ArcGIS Insights 需要附加许可或 Insights Analyst 用户类型，并且需要额外安装。每个门户成员都需要拥有一个许可，然后才能使用 Insights。当成员获得使用 Insights 的许可时，他们将可以访问 Insights web 应用程序和 ArcGIS Insights Desktop。按照管理许可内的说明进行操作以查看许可详细信息和管理成员许可。

ArcGIS Maps for Office

ArcGIS Maps for Office（之前为 Esri Maps for Office）通过 Microsoft 加载项在 Microsoft Office 中提供制图功能。可将此加载项用于 Microsoft Excel，以在门户中创建可显示电子表格数据的地图。此外，还可以通过 PowerPoint 加载项将这些地图嵌入到演示文稿中。

ArcGIS Maps for Office 随办公室应用程序包一起包含在部分核心 Esri 用户类型中，例如 GIS Professional、Creator，或作为附加许可。可从 Esri 网站下载 ArcGIS Maps for Office。

有关 ArcGIS Maps for Office 版本的信息，请参阅 ArcGIS Enterprise 受支持环境。

ArcGIS Maps for SharePoint

ArcGIS Maps for SharePoint 通过 SharePoint 页面的 Map Web Part 在 Microsoft SharePoint 中提供制图功能。您可借此在 SharePoint 页面中显示门户中的内容。该应用程序还包含支持通过业务连通性服务 (BCS) 在空间上使用并在地理层面上丰富 SharePoint 列表或外部数据的工作流。

ArcGIS Maps for SharePoint 随办公室应用程序包一起包含在部分核心 Esri 用户类型中，例如 GIS Professional 和 Creator，或作为附加许可。可从 Esri 网站下载 ArcGIS Maps for SharePoint。

有关所支持 ArcGIS Maps for SharePoint 版本的信息，请参阅 ArcGIS Enterprise 受支持环境。

注：

离线环境下不支持 ArcGIS Maps for SharePoint 3.0 和 3.0.1。如果您的组织无法访问 Internet，则无法通过 ArcGIS Enterprise 使用以下应用程序版本。

ArcGIS Mission Manager

ArcGIS Mission Manager 是一款 web 应用程序，允许用户使用 ArcGIS Mission（一款实时地理空间通信和态势感知工具）来创建任务并与任务进行交互。

ArcGIS Mission Manager 需要附加许可，并作为 ArcGIS Mission 的一部分单独安装。

ArcGIS Pro

应用程序 ArcGIS Pro 是 ArcGIS Desktop 的一部分，它具有三个级别：Desktop Basic、Desktop Standard 和 Desktop Advanced，每个级别都会提供附加功能。只有 ArcGIS Pro 通过组织使用许可。其他 ArcGIS Desktop 应用程序 - ArcMap、ArcCatalog、ArcGlobe 和 ArcScene - 均使用本地 ArcGIS Administrator 应用程序来设置级别和扩展模块。

ArcGIS Pro 包含在 GIS Professional 用户类型中，或作为附加许可。您可以为成员分配一个许可级别，也可以启用对可提供专业工具的可选扩展模块的访问权限。如果成员想使用扩展模块，该成员需同时具备特定扩展模块产品的级别和许可。

当分配有许可的成员启动 ArcGIS Pro 并进行登录时，该应用程序将在您指定的级别和扩展模块下运行。但是，如果您没有为特定成员指定许可，则 ArcGIS Pro 不会在该成员尝试登录时运行。您可以根据需要随时重新分配许可，所产生的更改会立即在 ArcGIS Pro 中生效。如果当您完全撤消许可时有成员正在使用 ArcGIS Pro，ArcGIS Pro 会显示一条通知提示软件没有授权，然后自动开启关闭程序。在宽限期结束后，系统会提示成员保存所有工作成果，并告知成员 ArcGIS Pro 将不再可用。

在 ArcGIS Pro 中，成员可检出许可来授权 ArcGIS Pro 用于离线环境。检出许可后将在成员姓名旁显示红框。检出许可后，您将无法进行更改或撤消许可。如果您选择多个成员，并且至少有一名成员正在离线模式下运行 ArcGIS Pro，那么您将无法为任何所选成员配置许可。

对于需要进一步控制离线使用 ArcGIS Pro 的组织，管理员或具有许可授权的成员可以阻止离线使用组织内的所有 ArcGIS Pro 许可。

如果使用检出许可的设备丢失或发生损坏，成员将无法在在线环境中检出新许可或使用 ArcGIS Pro。这种情况下，需要使用 ArcGIS Portal Directory 来释放检出许可。这将释放未完成的许可，并且允许成员在在线环境中检出新许可或使用 ArcGIS Pro。

处于在线环境中时，您可以远程终止 ArcGIS Pro 会话。例如，一名已登录的成员同时在三台计算机上有效运行了 ArcGIS Pro（即达到最大同时运行数），但他离开了办公室并且现在需要在外业使用 ArcGIS Pro。通过终止会话，成员可进行登录并启动 ArcGIS Pro。此外，如果意外关闭 ArcGIS Pro，您可以终止会话以快速释放许可。

有关通过 Portal for ArcGIS 设置许可的说明，请参阅配置 ArcGIS Pro 许可。

ArcGIS Pro 许可配置完成后，请按照管理许可中的说明进行操作以查看许可详细信息和管理成员许可。

ArcGIS QuickCapture

ArcGIS QuickCapture 是一款外业数据捕获应用程序，允许您快速捕获数据。您可以在行驶中捕获资产或事件的位置和属性。工程作者可以决定按钮的排列、显示方式及其所采集的信息。该应用程序允许您通过设备捕获照片和传感器信息。

ArcGIS QuickCapture 随外业应用程序包一起包含在部分核心 Esri 用户类型中，例如 GIS Professional、Creator 和 Field Worker，或作为附加许可。QuickCapture 设计器包含在门户中，您可以从 Apple、Google Play 或 Windows 应用程序商店中下载移动应用程序。

Collector for ArcGIS

Collector for ArcGIS 是一款移动数据采集应用程序，即使在断开连接的情况下，您也能轻松捕获新数据和更新外业中的现有要素，并无缝集成到 ArcGIS 中。借助 Collector，您可以通过使用 GPS 或单击地图来采集和更新数据，填写地图驱动表单，并且可以选择使用专业级 GPS 接收器。

Collector for ArcGIS 随外业应用程序包一起包含在部分核心 Esri 用户类型中，例如 GIS Professional、Creator 和 Field Worker，或作为附加许可。您可以从 Apple、Google Play 或 Windows 应用程序商店下载 Collector for ArcGIS 移动应用程序。

Drone2Map for ArcGIS

Drone2Map for ArcGIS 可用于可视化和分析无人机拍摄的地理定位影像，帮助您从空中观测区域并监控变化情况。使用 Drone2Map 创建的 2D 和 3D 地图可共享至您的 ArcGIS Enterprise 门户。

Drone2Map 需要附加许可和额外安装。每个门户成员都需要拥有一个许可，然后才能运行 Drone2Map。有关通过门户设置许可的说明，请参阅配置 Drone2Map for ArcGIS 许可。

Drone2Map 许可配置完成后，请按照管理许可中的说明进行操作以查看许可详细信息和管理成员许可。

Explorer for ArcGIS

Explorer for ArcGIS 是一款移动应用程序，允许组织中的每位成员随时随地访问其地图，即使在断开连接的情况下也能访问地图。您可对数据进行可视化、在您的地图中搜索地点和要素、查看资产信息、标记地图以高亮显示重要的要素以及使用罗盘获取到难以发现的资产的路线。

Explorer for ArcGIS 包含在核心 Esri 用户类型中，例如 GIS Professional、Creator、Editor、Field Worker 和 Viewer。您可以从 Apple、Google Play 或 Windows 应用程序商店获取移动应用程序。

GeoPlanner for ArcGIS

GeoPlanner for ArcGIS 允许您在基于土地的规划和设计方面与门户组织的其他成员进行协作。

GeoPlanner for ArcGIS 需要附加许可，每个门户成员都需要拥有一个许可，然后才能使用此应用程序。GeoPlanner 包含在门户中，无需额外安装。按照管理许可内的说明进行操作以查看许可详细信息和管理成员许可。

Navigator for ArcGIS

Navigator for ArcGIS 是一款专业的移动导航应用程序，使用 GIS 软件为您的外业工作人员安排路线。它附带数百个全球地图，并支持添加自定义地图。定义出行模式，在您自定义的道路上行驶，并搜索您的资产。可支持 Android 或 iOS 在线或离线使用，以及与其他 ArcGIS 外业应用程序配合使用。

Navigator for ArcGIS 需要附加许可，每个门户成员都需要拥有一个许可，然后才能使用此应用程序。Navigator 包含在门户中，无需额外安装。按照管理许可内的说明进行操作以查看许可详细信息和管理成员许可。

Operations Dashboard for ArcGIS

Operations Dashboard for ArcGIS 允许您实时监控人员、服务、资产和事件。使用 Operations Dashboard，您即可通过 ArcGIS Enterprise 门户创建并共享仪表盘。仪表盘包括图表、仪表、地图和其他可视元素，用于表示对实现目标最重要的活动和关键绩效指标。

Operations Dashboard 随基本应用程序包一起包含在核心 Esri 用户类型中，例如 GIS Professional、Creator、Editor、Field Worker 和 Viewer。此应用程序包含在门户中，无需额外安装。

Survey123 for ArcGIS

Survey123 for ArcGIS 简化了 ArcGIS 中以表单为中心的数据采集工作流。作为数据采集工作的重要部分，地理相对工作流来说是非常关键的，但并非核心的。Survey123 for ArcGIS 认识到这个事实，在 ArcGIS 中新增了行业中预期的表单复杂度。

Survey123 for ArcGIS 随外业应用程序包一起包含在部分核心 Esri 用户类型中，例如 GIS Professional、Creator 和 Field Worker，或作为附加许可。请转至 https://survey123.esri.com/ 以创建表单并发布到 ArcGIS Enterprise。您可以从 Apple、Google Play 或 Windows 应用程序商店获取 Survey123 for ArcGIS 移动应用程序。

Tracker for ArcGIS

Tracker for ArcGIS 使组织能够捕获外业工作人员的轨迹、监控他们的位置并分析他们曾经的位置。追踪模式可为决策者提供实时信息，以支持关键的外业活动并在 ArcGIS 中执行分析。它是由两个应用程序（Tracker for ArcGIS 移动应用程序和 Track Viewer web 应用程序）组成的移动解决方案。

Track Viewer web 应用程序包含在门户中，可供管理员和具有查看位置追踪权限的门户成员使用。此应用程序不需要 Tracker for ArcGIS 附加许可。

每个门户成员都需要拥有一个许可，然后才能使用 Tracker 移动应用程序。此许可可作为核心 Esri 用户类型的附加许可使用，例如 GIS Professional、Creator、Editor、Field Worker 和 Viewer。您可以从 Apple、Google Play 或 Windows 应用程序商店下载移动应用程序。按照管理许可内的说明进行操作以查看许可详细信息和管理成员许可。

Web AppBuilder for ArcGIS

Web AppBuilder 是一种直观的所见即所得式应用程序，可用于构建 web 应用程序，而无需编写代码。利用 Web AppBuilder，您可以构建和自定义应用程序，这些应用程序可在桌面、平板电脑和具有即用型微件（如查询、地理处理、打印）的智能手机上运行。

Web AppBuilder 随基本应用程序包一起包含在核心 Esri 用户类型中，例如 GIS Professional、Creator、Editor、Field Worker 和 Viewer。此应用程序包含在门户中，无需额外安装。

Workforce for ArcGIS

Workforce for ArcGIS 使用位置功能来实现更好的外业工作人员协作和团队合作。安排适当的工作人员前往适当的地点使用适当的工具进行适当的工作。调度员使用 Workforce web 应用程序来创建和分配工作、监控进度，以及实时查看工作人员的位置。移动工作人员使用 Workforce 移动应用程序来完成其个人的待完成列表和传达状态。

Workforce for ArcGIS 随外业应用程序包一起包含在部分核心 Esri 用户类型中，例如 GIS Professional、Creator 和 Field Worker，或作为附加许可。您可以从 Apple、Google Play 或 Windows 应用程序商店下载 Workforce 移动应用程序。
