作为访客参与者管理协作

访客参与者可以查看协作工作空间和其他访客组织；加入、编辑或离开工作空间；以及离开协作。如有必要，也可以选择手动同步内容。

查看协作参与者

每个协作参与者可以通过完成以下步骤来查看其它访客组织列表：

  1. 以管理员身份登录到组织。
  2. 浏览至组织 > 设置 > 协作。
  3. 单击要查看其参与者的协作名称。
  4. 单击查看访客。

在协作访客页面上，访客参与者可以查看其他协作参与者的列表以及其协作状态。

查看协作工作空间

一个协作可具有多个工作空间。可以通过以下步骤查看协作中的工作空间：

  1. 以管理员身份登录到您的组织。
  2. 浏览至组织 > 设置 > 协作。
  3. 单击要查看其工作空间的协作名称。

    在工作空间页面上，可以查看协作中的工作空间列表以及与工作空间、访问模式和状态关联的组。

    注：

    计划的时间将以客户的时区显示，并转换为运行门户的服务器上的协调世界时间 (UTC)。

  4. 要查看特定协作工作空间的详细信息，请单击工作空间的名称。

    可以查看具有工作空间访问权限的访客组织列表以及其访问模式和状态。

编辑协作工作空间

可以通过以下步骤来编辑与协作工作空间相关联的组以及工作空间的同步设置，也可以编辑工作空间的名称和描述选项。

  1. 以管理员身份登录到您的组织。
  2. 浏览至组织 > 设置 > 协作。
  3. 单击要查看其工作空间的协作名称。
  4. 在表中找到要编辑的协作工作空间名称，单击操作按钮下拉箭头，然后选择编辑工作空间。
  5. 或者，也可以创建新组或选择要链接到工作空间的其他现有组。也可以编辑工作空间的同步设置。选择立即同步或按照计划间隔进行同步选项。
    6. 如果选择按照计划间隔进行同步，请定义计划同步的起始时间和重复同步的时间。
    注：
    计划的时间将以客户的本地时区显示，并自动转换为客户门户托管服务器的协调世界时间 (UTC)。有关详细信息，请参阅使用计划的间隔同步工作空间。
  6. 请选择将托管要素图层发送到协作工作空间的方式。
    • 作为参考：协作参与者将在指定的工作空间中接收到要素图层的实时访问。协作参与者必须具有从其原点查看要素图层的权限。
    • 作为副本：协作参与者将按照计划间隔接收要素图层更新。选择此选项时：
    1. 必须在协作工作空间内设置同步间隔。
    2. 在协作中的每个参与项目上必须启用同步。

    将首先提取参与的要素图层的副本，并将其作为项目发布到每个具有接收权限的参与者。创建这些项目后，要素将根据建立的同步间隔进行更新。

  7. 单击保存。

编辑协作

对协作进行编辑，以在移除工作空间或协作时更改内容删除策略。

  1. 以管理员身份登录到您的组织。
  2. 浏览至组织 > 设置 > 协作。
  3. 在表中找到要编辑的协作工作空间名称，单击操作按钮下拉箭头，然后选择编辑协作。
  4. 在编辑协作对话框中，指定离开协作或工作空间时如何处理协作内容。从您的组织发送至协作的项目可以是：
    • 根据删除的工作空间或协作从所有参与者中移除。
    • 为所有参与者保留。选择此选项后，您可以在发送参与者允许的情况下保留协作项目。这是默认配置。
  5. 指定删除协作或任意工作空间时对组织接收的协作项目执行的操作。项目可以：
    • 根据删除的工作空间或协作，从您的组织中移除。
    • 为所有参与者保留。选择此选项后，您可以在发送参与者允许的情况下保留协作项目。这是默认配置。

    注意：内容删除策略是在协作级别下配置的，并应用于所有关联的协作工作空间中。

  6. 单击保存。

将群组加入到协作工作空间

如果尚未将群组加入到协作工作空间，可以通过以下步骤来实现：

  1. 以管理员身份登录到您的组织。
  2. 浏览至组织 > 设置 > 协作。
  3. 单击要加入工作空间的协作名称。
  4. 在表中找到要将加入组的协作工作空间名称，单击操作按钮下拉箭头，然后选择加入工作空间。
  5. 选择将与工作空间相关联的群组。可以是现有群组，也可以是专门为此创建的群组。一个群组仅能与一个协作工作空间相关联。
    • 如果您为协作工作空间创建了一个群组，则该群组将为私有群组，所有群组成员都可以向该群组贡献内容。如有必要，在群组创建完成之后，可以更改状态及贡献设置。有关这些设置的详细信息，请参阅创建群组。为协作工作空间创建群组时，必须指定群组的标签才能继续进行。
    • 如果您选择连接现有组，则下拉菜单会显示一个包含有门户中所有组（包括私有组）的列表，已启用了更新访问功能的组除外。
  6. 请从以下选项中选择将托管要素图层发送到协作工作空间的方式：
    • 作为参考：协作参与者将在指定的工作空间中接收到要素图层的实时访问。协作参与者必须具有从原始门户查看要素图层的权限。
    • 作为副本：协作参与者将按照计划间隔接收要素图层更新。
      注：

      将协作工作空间配置为将要素图层共享为副本后，此设置将无法更改。

    选择发送托管功能图层数据的副本时：

    • 必须在协作工作空间内设置同步间隔
    • 在协作中的每个参与项目上必须启用同步。
    注：

    将首先提取参与的要素图层的副本，并将其作为项目发布到每个具有接收权限的参与门户。创建这些项目后，要素将根据建立的同步间隔进行更新。

  7. 单击加入工作空间。
    8. 默认情况下，如果主办方门户或访客门户将群组加入到协作工作空间，则同步配置将设置为立即同步。可以通过编辑工作空间对此进行更改。协作参与者将收到访客门户已将群组加入到协作工作空间的通知。

编辑访客组织联系人

可以通过以下步骤来编辑协作联系人的详细信息。

  1. 以管理员身份登录到您的组织。
  2. 浏览至组织 > 设置 > 协作。
  3. 单击要编辑联系人的协作名称。
  4. 单击查看访客。
  5. 在访客门户列表中单击门户的名称，然后单击编辑联系人。
  6. 编辑联系人的全名和联系人的电子邮件地址选项，然后单击保存。

离开协作工作空间

可以通过以下步骤离开协作中的工作空间：

  1. 以管理员身份登录到您的组织。
  2. 浏览至组织 > 设置 > 协作。
  3. 单击要离开工作空间的协作名称。
  4. 在表中找到要离开的协作工作空间名称，单击操作按钮下拉箭头，然后选择离开工作空间。
  5. 出现离开工作空间对话框后，单击离开工作空间。

    离开工作空间对话框将显示内容删除策略的汇总，并将指示如何保留协作中发送和接收的内容。要更改此策略，请单击取消，然后使用编辑协作操作。

离开协作

可以通过以下步骤离开协作：

  1. 以管理员身份登录到您的组织。
  2. 浏览至组织 > 设置 > 协作。
  3. 单击要离开的协作名称，然后单击操作按钮下拉箭头并选择离开协作。
  4. 单击离开协作。
  5. 出现离开协作对话框后，单击离开协作。

    离开协作对话框将显示内容删除策略的汇总，并将指示如何保留协作中发送和接收的内容。要更改此策略，请单击取消，然后使用编辑协作操作。

    协作主办方管理员将收到您已离开协作的通知。

将在下次同步时从参与的协作工作空间中移除组织成员与协作共享的内容。协作主办方管理员将收到您已离开协作的通知。

按需同步协作工作空间

要按需同步协作工作空间，请完成以下步骤：

  1. 以管理员身份登录到门户。
  2. 浏览至组织 > 设置 > 协作。
  3. 单击要同步的协作名称。
  4. 找到您要同步的工作空间。单击操作按钮下拉箭头并选择同步工作空间。

根据组织对此工作空间的访问权限，将发送和接收任何新内容和更新的内容。可以在“计划同步”下的协作摘要页面上查看同步结果。

查看同步状态

可以通过完成以下步骤来查看您所在组织的同步状态：

  1. 以管理员身份登录到门户。
  2. 浏览至组织 > 设置 > 协作。
  3. 单击要查看的协作名称。
  4. 单击要查看其同步状态的协作工作空间名称。
  5. 在协作工作空间的汇总页面上，可以看到计划同步和立即同步最近的同步状态。以下是可能的状态：

    计划同步：

    • 上次同步 - 显示最近一次计划同步的日期和时间。
    • 状态 - 显示上一次计划同步的状态。以下是可能的状态：
      • 已成功 - 表明所有项目的同步均已成功完成。
      • 已成功但存在错误 - 在同步已完成但部分项目共享失败的情况下显示。
      • 失败 - 在同步失败且未共享任何项目时显示。单击失败链接了解其他详细信息。

    • 上次完成同步 - 显示成功共享所有项目的上次同步的日期和时间。

    • 下次同步 - 显示下一次计划同步的日期和时间。如果已暂停计划同步，则此值将显示为已暂停。

    立即同步：

    • 上次同步 - 显示上一次立即同步的日期和时间。
    • 状态 - 显示上一次立即同步的状态。可能的状态为：
      • 已成功 - 表明所有项目的同步均已成功完成。
      • 已成功但存在错误 - 在同步已完成但部分项目共享失败的情况下显示。
      • 失败 - 在同步失败且未共享任何项目时显示。单击失败链接了解其他详细信息。

注：
计划的时间将以客户的本地时区显示，并自动转换为门户托管服务器的协调世界时间 (UTC)。有关详细信息，请参阅使用计划的间隔同步工作空间。

暂停并恢复计划同步

要暂停并重新启动计划同步，请完成以下步骤：

  1. 以管理员身份登录到门户。
  2. 浏览至组织 > 设置 > 协作。
  3. 单击要更新的协作的名称。
  4. 找到要更新的协作工作空间。单击操作按钮下拉箭头并选择暂停同步。
  5. 在暂停同步对话框中，单击暂停同步。选择此选项将导致暂停协作工作空间的计划同步。暂停协作工作空间的计划同步时，“同步状态”报告会将下次同步显示为已暂停。
  6. 要重新启动计划同步，请单击协作工作空间上的操作按钮下拉箭头，然后选择恢复同步。现在，“同步状态”报告将显示下次计划同步的时间，并将重新启动计划同步。
