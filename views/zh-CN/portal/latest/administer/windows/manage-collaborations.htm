作为协作主办方，您可以管理工作空间、访客参与者和协作细节。协作主办方可以编辑协作的属性或完全删除协作。主办方还可以为现有协作添加其他工作空间。这些任务的说明如下所示。也可以更改协作工作空间的共享计划。有关详细信息，请参阅配置同步设置。

编辑协作

编辑协作以更改以下任一属性：

  • 协作名称
  • 协作描述
  • 协作主办方联系人全名
  • 联系人的电子邮件地址
  • 移除工作空间或协作时的协作内容删除策略

要编辑协作，请完成以下步骤：

  1. 以管理员身份登录到主办方组织。
  2. 浏览至组织 > 设置 > 协作。
  3. 在表中找到要编辑的协作名称，单击操作按钮下拉箭头，然后选择编辑协作。
  4. 在编辑协作对话框中，执行以下操作：
    • 输入协作名称。
    • 输入协作描述。
    • 输入协作主办方联系人全名。
    • 输入联系人的电子邮件地址。
  5. 单击下一步。
  6. 使用已发送参数指定删除协作或其任意工作空间时对发送自组织的协作项目执行的操作。有以下两个选项可供选择：
    • 根据删除的工作空间或协作从所有参与者中移除。
    • 为所有参与者保留。选择此选项时，参与者可以决定是否在删除工作空间或协作时保留协作项目。这是默认配置。
  7. 使用已接收参数指定删除协作或其任意工作空间时对组织中接收的协作项目执行的操作。有以下两个选项可供选择：
    • 根据删除的工作空间或协作从所有参与者中移除。
    • 为所有参与者保留。选择此选项后，您可以在发送参与者允许的情况下保留协作项目。这是默认配置。
    8. 注：

    内容删除策略是在协作级别下配置的，并应用于所有关联的协作工作空间中。

  8. 单击保存。

编辑主办方联系人

要编辑协作的联系人，请完成以下步骤：

  1. 以管理员身份登录到主办方组织。
  2. 浏览至组织 > 设置 > 协作。
  3. 在表中单击要编辑的协作名称，然后在协作摘要中单击联系人旁边的编辑按钮。
  4. 编辑联系人的全名和联系人的电子邮件地址选项，然后单击保存。

删除协作

要删除协作，请完成以下步骤：

  1. 以管理员身份登录到主办方组织。
  2. 浏览至组织 > 设置 > 协作。
  3. 在表中找到要删除的协作名称，单击操作按钮下拉箭头，然后选择删除协作。
  4. 出现删除协作对话框后，选择删除协作即可删除协作。
注：

删除协作对话框将显示内容删除策略的摘要，并将指示如何保留协作中已发送和已接收的内容。要更改此策略，请单击取消，然后使用编辑协作操作。

访客参与者管理员将收到协作已删除的通知。

编辑协作工作空间

可以通过以下步骤来编辑与协作工作空间相关联的组，也可以编辑工作空间的名称和描述选项。

  1. 以管理员身份登录到您的组织。
  2. 浏览至组织 > 设置 > 协作。
  3. 单击要查看其工作空间的协作名称。
  4. 在表中找到要编辑的工作空间名称，单击操作按钮下拉箭头，然后选择编辑工作空间。
  5. 显示编辑工作空间对话框后，编辑工作空间名称和工作空间描述选项。单击下一步。
  6. 或者，也可以创建新组或选择要链接到工作空间的其他现有组，然后单击下一步。
  7. 如果您是协作主办方管理员，可以查看用于更改访客参与者如何访问协作工作空间的选项。或者，也可以从下拉菜单中选择新的访问模式，然后单击保存。
    8. 还可以通过单击协作工作空间列表中的工作空间名称来编辑访客参与者访问。您将看到访客参与者访问模式，但是可以从下拉菜单中选择新模式。

添加协作工作空间

可以通过完成以下步骤将其他工作空间添加到协作：

  1. 以管理员身份登录到主办方组织。
  2. 浏览至组织 > 设置 > 协作。
  3. 单击要添加工作空间的协作名称。
  4. 单击创建工作空间。
  5. 提供协作工作空间的名称和描述，然后单击下一步。
  6. 选择将与工作空间相关联的群组。可以是现有群组，也可以是专门为此创建的群组。一个群组仅能与一个协作工作空间相关联。
    • 如果创建群组，则该群组将为私有群组，所有群组成员都可以向该群组贡献内容。如有必要，在群组创建完成之后，可以更改状态及贡献设置。有关这些设置的详细信息，请参阅创建群组。针对协作创建群组时，必须指定群组的标签才能继续。
    • 如您选择连接至现有群组，则下拉菜单将显示一个包含门户中所有群组的列表，其中包括私有群组。
  7. 在下拉菜单中针对每个协作参与者选择访问模式，然后单击保存。

访客参与者管理员将收到新协作工作空间已添加的通知。

删除协作工作空间

要删除协作工作空间，请完成以下步骤：

  1. 以管理员身份登录到主办方组织。
  2. 浏览至组织 > 设置 > 协作。
  3. 单击包含要删除的工作空间的协作名称。
  4. 在表中找到要删除的协作工作空间名称，单击操作按钮下拉箭头，然后选择删除工作空间。
  5. 在删除工作空间对话框中，单击删除工作空间。
注：

删除工作空间对话框将显示内容删除策略的摘要，并将指示如何保留协作中已发送和已接收的内容。要更改此策略，请单击取消，然后使用编辑协作操作。

查看协作参与者

可以通过完成以下步骤来查看其他参与者列表：

  1. 以管理员身份登录组织。
  2. 浏览至组织 > 设置 > 协作。
  3. 单击包含您要查看的参与者的协作名称。

    在工作空间页面上，可以查看协作中的工作空间列表以及与工作空间、访问模式和状态关联的组。

  4. 单击查看访客。

在协作访客页面上，协作主办方管理员可以选择邀请访客来邀请新访客门户加入协作。

接受访客参与者的响应

可以通过完成以下步骤来接受访客参与者对协作邀请的响应：

  1. 以管理员身份登录到主办方组织。
  2. 浏览至组织 > 设置 > 协作。
  3. 单击查看访客。
  4. 单击包含您要查看的参与者的协作名称。
  5. 在协作访客页面上，如果尚未将访客门户响应导入至协作，则其状态将被列为待处理邀请。要接受访客参与者，请单击表中访客条目的操作按钮下拉箭头，然后选择接受访客组织。
  6. 在接受访客组织对话框中，浏览至从访客参与者接收的邀请响应文件，然后单击接受访客组织。

协作中的管理员将收到一条通知，指示新访客参与者已加入。

编辑访客参与者的信息或访问模式

可以通过完成以下步骤来更改访客参与者的联系信息和访问模式：

  1. 以管理员身份登录到主办方组织。
  2. 浏览至组织 > 设置 > 协作。
  3. 单击查看访客。
  4. 单击包含您要编辑的访客组织的协作名称。
  5. 在协作访客页面上，如果访客门户的状态列为活动，则可以编辑其详细信息和访问模式。要编辑访客门户，请单击表中访客条目的操作按钮下拉箭头，然后选择编辑访客组织。
  6. 在编辑访客组织对话框中，也可以编辑联系信息并从下拉菜单中选择新的访问模式，然后单击保存。

如果其访问模式已更改，则访客参与者管理员将会收到通知。

重新发出邀请

如果协作邀请已丢失或已到期，则可以通过完成以下步骤来重新发出邀请：

  1. 以管理员身份登录到主办方组织。
  2. 浏览至组织 > 设置 > 协作。
  3. 单击查看访客。
  4. 单击要重新发出邀请的协作名称。
  5. 在协作访客页面上，如果尚未将访客参与者响应导入至协作，则其状态将被列为待处理邀请。要重新向访客发出邀请，请单击表中访客条目的操作按钮下拉箭头，然后选择重新发出邀请。
  6. 在重新发出邀请对话框中，单击重新发出邀请。

移除协作邀请

如果不再希望与组织建立信任，可以通过完成以下步骤来移除协作邀请：

  1. 以管理员身份登录到主办方组织。
  2. 浏览至组织 > 设置 > 协作。
  3. 单击要移除邀请的协作名称。
  4. 单击查看访客。
  5. 在协作访客页面上，如果尚未将访客参与者响应导入至协作，则其状态将被列为待处理邀请。要移除访客参与者的邀请，请单击表中访客条目的操作按钮下拉箭头，然后选择移除邀请。
  6. 在移除邀请对话框中，单击移除邀请。

访客参与者的所有现有邀请文件都将失效。如果访客参与者接受邀请并发送响应文件，则响应文件将无效。

从协作中移除访客参与者

如果协作主办方已接受访客组织加入到协作，则协作主办方可以通过完成以下步骤从协作中移除访客参与者：

  1. 以管理员身份登录到主办方组织。
  2. 浏览至组织 > 设置 > 协作。
  3. 单击要移除访客门户的协作名称。
  4. 单击查看访客。
  5. 在协作访客页面上，如果访客参与者的状态列为活动，则可以将其从协作中移除。要移除访客参与者，请单击表中访客条目的操作按钮下拉箭头，然后选择移除访客组织。
  6. 在移除访客组织对话框中，单击移除访客组织。
注：

移除访客组织对话框将显示内容删除策略的摘要，并将指示如何保留协作中已发送和已接收的内容。要更改此策略，请单击取消，然后使用编辑协作操作。

协作参与者管理员将收到一条通知，指示访客参与者已从协作中移除。

查看同步状态

注：
计划的时间将以客户的本地时区显示，并自动转换为运行门户的服务器对应的协调世界时间 (UTC) 值。有关详细信息，请参阅使用计划的间隔同步工作空间。

可以通过完成以下步骤来查看每个协作参与者的同步状态：

  1. 以管理员身份登录到主办方组织。
  2. 浏览至组织 > 设置 > 协作。
  3. 单击要查看的协作名称。
  4. 单击要查看其同步状态的协作工作空间名称。
  5. 在协作工作空间汇总页面上，每个访客组织的同步状态都会以图标标出。将鼠标悬停在图标上即可查看最新的同步状态。可能的状态如下：
    • 已成功 - 所有项目的同步均已成功完成。
    • 已成功但存在错误 - 同步已完成，但部分项目共享失败。
    • 失败 - 同步失败，未共享任何项目。单击已失败以查看详细信息。
    注：

    计划的时间将以客户的时区显示，并转换为运行门户的服务器对应的协调世界时间 (UTC)。

  6. 要查看访客组织的同步详细信息，请单击同步按钮。随即显示一个弹出窗口，其中提供了计划同步和立即同步的详细信息：

    计划同步：

    • 上次同步 - 最近计划同步的日期和时间。
    • 状态 - 最近一次计划同步的状态。可能的状态如下：
      • 已成功 - 所有项目的同步均已成功完成。
      • 已成功但存在错误 - 同步已完成，但部分项目共享失败。
      • 失败 - 同步失败，未共享任何项目。单击已失败了解其他详细信息。

    立即同步：

    • 上次同步 - 最近一次立即同步的日期和时间。
    • 状态 - 最近一次立即同步的状态。可能的状态如下：
      • 已成功 - 所有项目的同步均已成功完成。
      • 已成功但存在错误 - 同步已完成，但部分项目共享失败。
      • 失败 - 同步失败，未共享任何项目。单击已失败了解其他详细信息。
注：

使用 ArcGIS Enterprise 10.5 或 10.5.1 的访客组织无法使用同步状态。这些组织的状态将为不可用。
