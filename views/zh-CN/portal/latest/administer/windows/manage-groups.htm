管理群组

在本主题中

  1. 编辑群组属性和设置
  2. 管理特定成员的群组
  3. 管理群组成员资格和角色

如果您是您所在组织的管理员，或具备相应的权限，则可以管理成员所拥有的群组。例如，您可以编辑群组详细信息、更改可查看和加入群组的成员、更改共享属性、邀请成员、管理组员资格申请、更改所有者、更新群组中的成员角色、从群组中移除成员以及删除群组。也可以邀请其他成员加入群组。

提示：

建议创建一些供成员加入的群组，从而展示组织的专题内容、底图和模板。这将有助于用户理解门户中内容的组织方式以及如何与他人协作。有关详细信息，请参阅站点配置群组。

编辑群组属性和设置

要编辑成员所拥有的或所管理的群组的属性和设置，请打开群组页面。根据群组的创建方式，某些操作可能不可用。

  1. 登录门户网站并验证您是否有权限管理成员群组。 
  2. 单击站点顶部的群组。
  3. 使用选项卡、过滤器、排序选项，并根据需要搜索以查找要编辑的群组。
  4. 单击群组名称以打开群组页面，然后执行以下任一操作：
    • 要编辑群组信息，请单击要编辑的属性旁的编辑选项，进行编辑，然后单击保存以保存更改。您可以编辑群组的名称、摘要、描述、缩略图等。
    • 要更改群组设置，如可以查找群组、加入群组以及向群组贡献内容的用户，或者指定在群组的内容和概述选项卡上默认显示的项目类型（例如，地图或图层），请单击设置选项卡并更新要更改的属性。您必须具有相应的权限才能更新私人群组的状态或定义企业群组以管理成员资格。
    • 要更改群组的所有者，请在概述选项卡上单击更改所有者，选择一个新的群组所有者，然后单击更改所有者。
      注：

      对于允许群组成员更新群组中的所有项目的群组（即，启用了项目更新功能的群组），只有同时满足以下两个条件时才能转让群组所有权：

      • 正在转让群组所有权的成员或者是默认管理员，或者具有群组：重新分配所有权管理权限。
      • 正在接受群组所有权转让的成员或者是默认管理员，或者具有群组：使用更新功能进行创建管理权限。

      提示：

      如果需要批量更改群组和其他项目的所有权，则可使用 TransferOwnership 命令行工具。有关详细信息，请参阅管理项目。

    • 要防止群组被意外删除，请单击设置选项卡，然后选中删除保护下的复选框。
    • 要删除群组，请单击设置选项卡，然后单击删除群组。在删除窗口中，单击删除。如果该群组已启用删除保护，则此选项不可用。
      注：

      如果群组链接至来自企业标识存储的企业组，则仅会从门户中删除群组项目。企业群组由标识存储进行管理，因此不会受到影响。

      如果直接从标识存储中删除群组，则群组项目会保留在门户中。所有群组成员都要从项目中移除，群组的所有者除外。

管理特定成员的群组

作为管理员，您可以在成员的群组页面对组织中特定成员的群组成员资格进行管理。

  1. 登录门户网站并验证您是否有权限管理成员群组。 
  2. 单击站点顶部的组织，然后单击成员选项卡。
  3. 按姓名或用户名搜索成员，并根据需要应用过滤器。
  4. 对于要管理其所在群组的成员，单击更多选项按钮 ，然后单击查看群组。

    随即显示成员的群组页面。

  5. 执行以下任一操作：
    • 如果看到邀请链接，则表示该成员已受邀加入链接旁的编号所指定的编号群组。单击邀请，然后针对每个群组单击加入该群组或拒绝。完成后单击关闭。
    • 要添加成员到您所在组织中的一个或多个群组，单击添加到群组，查找并选择要添加成员的群组，然后单击添加到群组。
    • 要移除群组中的成员，请单击该群组的从组中移除成员链接并单击移除。
    • 要删除成员所拥有的群组，请单击群组的删除群组链接，然后单击删除。
    • 要更新群组中的成员角色，请单击群组的名称以打开群组页面。有关详细信息，请参阅管理群组成员资格和角色。

管理群组成员资格和角色

您可以移除群组中的成员，并在群组页面的成员选项卡更改其在群组中的角色。您可以邀请成员加入群组，并在概述选项卡或成员选项卡中管理加入群组的申请。根据群组的创建方式，某些操作可能不可用。例如，如果群组链接至来自企业标识存储的企业群组，则用于邀请用户和管理群组员资格申请的选项将不可用。

提示：

默认管理员也可以通过群组设置将成员批量分配到群组。

  1. 登录门户网站并验证您是否有权限管理成员群组。 
  2. 单击站点顶部的群组。
  3. 使用选项卡、过滤器、排序选项，并根据需要搜索以查找要编辑属性和设置的群组。
  4. 单击群组名称以打开群组页面，然后执行以下任一操作：
    • 要移除群组中的一个或多个成员，请单击成员选项卡，选中要移除的成员旁边的复选框，然后单击从群组中移除成员。
    • 要更改群组中的成员角色，请单击成员选项卡，选中要更改其群组角色的成员旁的复选框，然后单击更新成员的群组角色。从下拉菜单中选择其他角色，然后单击更新。群组角色包括所有者、群组管理员和成员，以及用户可用选项（具体取决于成员的权限）。例如，要升级为群组管理员，则成员必须为与群组所有者所在的组织相同的成员，并且具备创建和拥有群组的权限。
    • 要管理加入群组的申请，请在概述或成员选项卡上单击成员资格申请，来接受或拒绝申请。
    • 要邀请其他成员加入群组，请在概述或成员选项卡上单击邀请用户，然后选择要邀请加入到该群组的成员。您可以直接将用户添加到群组中即可让他们成为群组中的一员，而无需向他们发送邀请并让他们接受该邀请。  为此，请选中无需确认，立即添加组织成员复选框。单击 发送邀请邀请用户加入您的群组，或单击添加至群组直接添加成员。
      注：

      邀请将作为通知发送至所选成员；不采用电子邮件形式发送。成员在登录门户时会看到群组邀请的通知。通知在门户网站中的成员用户名旁边列出。
