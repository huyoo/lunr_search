什么是 ArcGIS Enterprise 门户？

在本主题中

  1. ArcGIS Enterprise 门户可提供哪些内容？
  2. 入门

ArcGIS Enterprise 门户是一种允许您在组织内与其他人共享地图、场景、应用程序和其他地理信息的 ArcGIS Enterprise 组件。前端门户由后端 Portal for ArcGIS 基础架构提供支持。您可对门户进行自定义，以使其符合您所在组织的外观。

ArcGIS Enterprise 门户可将 ArcGIS 平台中的所有地理信息汇集到一起，并在整个组织中共享。拥有该门户后，您可以：

  • 创建、保存和共享 web 地图和场景。
  • 创建和托管 web 制图应用程序。
  • 在组织内搜索 GIS 内容。
  • 创建群组以与同事共享 GIS 信息。
  • 共享 GIS 应用程序的链接。
  • 共享地图和图层包以用于 ArcGIS Pro 或 ArcGIS Desktop

ArcGIS Enterprise 门户让 GIS 可供所有体验级别的用户访问。地理查看器是专为 GIS 初学者设计的，而有经验的 GIS 用户则可以通过 ArcGIS Pro、ArcGIS Desktop、开发人员 API 以及其他应用程序连接至门户。

ArcGIS Enterprise 门户可提供哪些内容？

ArcGIS Enterprise 门户包含以下内容：

  • ArcGIS Enterprise 门户网站 - 在门户的前端，用户可以登录到该网站来搜索、使用和共享内容。

  • Map Viewer - 门户网站中的此应用程序可用于查看、设计和保存 web 地图。可将其视为这样一个“画布”：您可在其中合并 GIS 信息和服务来制作自己的地图。可以保存地图、将其与其他人共享以及将其嵌入使用 ArcGIS Configurable Apps 模板或 Web AppBuilder for ArcGIS 创建的应用程序。

  • Web AppBuilder for ArcGIS - 门户网站包括一个嵌入的交互式应用程序，可根据 Web 地图设计和构建 Web 应用程序。通过 Web AppBuilder 创建的应用程序不需要任何编程知识且是基于 HTML 的，这表示它们可在桌面浏览器、平板电脑和智能手机上使用，不需要插件。随附 Web AppBuilder 提供了可自定义的各种主题以及用于提供高级功能（如高质量打印、地理处理、编辑和搜索）的微件。要了解详细信息，请参阅关于 Web AppBuilder for ArcGIS。

  • Activity Dashboard for ArcGIS - 网站包括可用于报告项目、用户和群组的各种使用统计数据的嵌入式应用程序。门户管理员可使用此信息监控组织中用于详细描述活动的报告、信息和指标。有关详细信息，请参阅关于使用情况报告。

  • Scene Viewer - 在 ArcGIS Enterprise 门户中，Scene Viewer 应用程序用于查看 3D 地理空间内容。Scene Viewer使用支持 WebGL 的桌面 Web 浏览器，WebGL 是用于渲染 3D 图形的 Web 技术标准。

  • ArcGIS 应用程序 - ArcGIS Enterprise 门户支持易于操作的应用程序，允许您通过门户与 Web 地图和场景进行交互。有关门户就绪型应用程序的完整列表，请参阅 ArcGIS Enterprise 的常用客户端。

  • ArcGIS Living Atlas of the World 内容 - 默认情况下，门户配置为访问 Esri 提供的 ArcGIS Living Atlas of the World 内容。Living Atlas 无论是在在线还是离线环境中，都可以将内容提交到门户；有关详细信息，请参阅配置 Living Atlas 内容。

  • 管理员工具 - ArcGIS Enterprise 门户网站包括管理员工具，可用于对用户、群组和内容进行管理。只有具有管理员权限的门户成员在登录网站时会看到这些选项。

  • ArcGIS Portal Directory - ArcGIS Portal Directory 是使用门户的 ArcGIS REST API 资源的交互式视图。例如，部分管理任务（如注销 ArcGIS Web Adaptor (IIS)、更新门户内容目录的位置和设置标识存储）在 ArcGIS 门户目录中可用。有关详细信息，请参阅关于 ArcGIS Portal 目录。
