关于 CityEngine Web 查看器

功能

通过 CityEngine Web 查看器，您可以在浏览器中与 3D 城市场景和其他 3D 场景进行交互，从而执行以下操作：

  • 通过平移、缩放和改变视角来导航场景。
  • 选择要查看的特定图层。
  • 切换场景以显示其他方案和情景。
  • 搜索要素、属性和元数据等场景内容。

有关如何与 3D 场景进行交互的详细信息，请参阅使用 CityEngine Web 查看器。另外，请参阅 Scene Viewer 和 CityEngine Web 查看器之间的差异以了解这两种查看器的功能和用途。

浏览器要求

CityEngine Web Viewer 需要支持 WebGL 的桌面 web 浏览器，WebGL 是用于渲染 3D 图形的 web 技术标准。最新版本的最常见桌面浏览器都内置有 WebGL。要测试浏览器是否启用了 WebGL，请转至 get.webgl.org。要解决与 WebGL 相关的问题，请访问 WebGL 疑难解答。由于 WebGL 技术在不断发展，每一浏览器版本都会融入改进的成果，而 CityEngine Web 查看器正是依赖这些改进才能实现最佳性能，因此，请确保您的浏览器为最新版本。CityEngine Web Viewer 目前不支持在移动设备上使用。

CityEngine Web 查看器支持以下 web 浏览器：

  • Chrome
  • Firefox
  • Internet Explorer 11*
  • Edge*
  • Safari 9 及更高版本*

*此浏览器在 WebGL 实施方面未针对存储密集型应用程序进行优化，因此，打开某些场景时可能会出现不稳定的情况。

Internet Explorer 11

使用 Internet Explorer 11 (IE 11) 时，您应该将浏览器更新为最新版本。CityEngine Web 查看器仅可在 WebGL 版本为 0.94 的 IE 11 中获得支持；较早的 WebGL 版本已过期，可能无法正常运行。要检查 WebGL 版本，可访问 www.Webglreport.com。

注：

IE 11 的 CityEngine Web 查看器中有一个已知的内存问题。如果您遇到性能问题，请关闭浏览器，然后重新打开。重新加载该页面或打开新选项卡将不会清除内存。您必须关闭并重新打开浏览器。

硬件要求

计算机硬件必须至少具有 2 GB 的系统内存以及支持 WebGL 的显卡。建议系统内存至少为 4 GB。

硬件应配有至少 512 MB 显存的非移动显卡。为了获得最佳性能，尤其在处理较大场景或占用大量内存的场景时，建议配置至少 1 GB 显存的显卡。与移动显卡相比（以 m 或 mobile 表示），非移动版本的显卡通常具有更好的性能。有关 WebGL 要求的更多详细信息，请访问 WebGL 疑难解答。

显卡

  • 显卡需要支持 OpenGL 3.0 (Shader Model 4.0)
  • AMD/ATI：Radeon 3xxx 或更高版本
  • NVIDIA：GeForce 8xxx 或更高版本，或 GeForce 2xx 或更高版本
  • NVIDIA：支持 OpenGL 3.0 的 Quadro 或更高版本
  • Intel：HD 3000 或更高版本
注：

一些系统可以自动在集成显卡和独立显卡之间进行切换，以渲染 3D 图形。CityEngine Web 查看器使用独立显卡时性能最佳；因此，需在显卡驱动程序设置中确保 Web 浏览器的每个应用程序设置均设置为独立显卡。如果未进行设置，则无法正确显示一些点符号和标注。

为 CityEngine Web 查看器创建 Web 场景

可使用 Esri CityEngine 或导出为 3D Web 场景地理处理工具创建 Web 场景。

  • Esri CityEngine 是一个软件包，用于使用现有 GIS 数据高效创建 3D 城市景观并进行 3D 形式的概念地理设计。制作景观后，可以将 3D 场景直接上传到 ArcGIS Enterprise。然后，用户可以使用 ArcGIS 系统共享这些可视化内容，使其他人可以通过 Web 浏览器使用 CityEngine Web 查看器查看这些内容，而无需安装任何软件。
  • 导出为 3D Web 场景可以将 ArcScene 文档导出为 CityEngine Web 场景格式（自 ArcGIS 10.2 版本起可用）。

了解有关 Esri CityEngine 的详细信息

Esri CityEngine 网站是查找有关 Esri CityEngine 及其功能详细信息的主要资源。您也可以访问 CityEngine 社区查找有关 CityEngine 的其他信息。社区中具有与 CityEngine 和 CityEngine Web 查看器相关的实例、博客帖子、论坛主题帖、视频以及文档。
