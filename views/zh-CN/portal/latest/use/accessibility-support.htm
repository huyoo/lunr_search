辅助功能支持

在本主题中

  1. 屏幕阅读器
  2. 主题
  3. 微件

已添加了辅助功能支持，以使应用程序可供所有用户（包括残障人士）访问。Web AppBuilder for ArcGIS 现在包含对 2D 应用程序的某些微件和主题中的辅助功能支持。对于增量版本，更多微件和主题将支持辅助功能。目前，其仅在运行时适用于 2D 应用程序。其在应用程序配置过程中不受支持。

下表包含 Web AppBuilder 受支持的屏幕阅读器、微件和主题。

提示：

有关 Web AppBuilder 应用程序辅助功能的详细信息，请参阅此博客文章。

注：

表格内容会随增量版本的不同而变化。
