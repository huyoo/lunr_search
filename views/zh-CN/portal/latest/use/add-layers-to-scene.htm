将图层添加到新建或现有场景中，以便于更好地讲述您的故事。您可以浏览 Scene Viewer 资源，如我的内容、我的组织或 Living Atlas 以添加内容并增强场景。还可以通过输入 URL 添加图层。

注：

有关图层要求，请参阅添加图层：全球与局部。

您可以通过该设计器窗格将设计和创作更改应用于可保存并可以共享的场景，如添加图层、捕获幻灯片、样式和过滤图层。

  1. 验证您是否已登录到门户，并打开 Scene Viewer。
  2. 单击修改场景按钮 以打开设计器。

Scene Viewer 支持哪些图层？

Scene Viewer 支持以下图层：

  • 高程图层
  • 要素图层
  • 影像图层*
  • 地图图像图层
  • OGC Web 地图服务 (WMS)
  • OGC Web 地图切片服务 (WMTS)
  • 场景图层
    • 3D 对象
    • 建筑物
    • 集成式网格
    • 点
    • 点云
  • 切片图层**
  • 矢量切片图层

*Scene Viewer 仅支持 Map Viewer 中设置的部分图像渲染属性。

**Scene Viewer 支持发布到 ArcGIS Server 10.2.2 及更高版本的该图层。

有关 ArcGIS Enterprise 中 ArcGIS Online 和 Scene Viewer 图层可用性的信息，请参阅场景查看器图层支持。

搜索图层

要向您的场景添加图层，请执行以下操作：

  1. 单击添加图层。
  2. 通过从以下内容中搜索现有图层，为场景添加图层：

    • 我的内容 - 包括您拥有的图层。
    • 我的收藏夹 - 包括您已标记为收藏项目的图层。
    • 我的群组 - 包括您所属群组的图层。
    • 我的组织 - 包括组织中的图层。

    • Living Atlas - 此选项会返回 ArcGIS Living Atlas of the World 中的所选图层，其中包括交通、人口统计和土地覆被图层。仅当您登录到配置为使用 ArcGIS Living Atlas of the World 内容的门户时，此选项才可用。

    在搜索要添加至场景的图层时，您可以更改结果的显示方式，对结果进行排序和过滤，以方便您查找所需图层并将结果添加到收藏夹列表中。您也可以查看各个图层的详细信息，以了解更多相关信息。

    要缩小添加图层中的结果范围，在搜索窗口中输入要添加到场景的图层的名称。

  3. 执行以下任意操作以便查找所需图层：
    • 在搜索框中输入搜索术语。您也可以使用搜索字段帮助缩小搜索结果范围。
    • 单击过滤器按钮 以缩小搜索结果范围。例如，如果您正在搜索组织中的图层，且已设置内容类别，则可以使用类别过滤器来缩小结果范围。如果您希望您的搜索仅返回已标记为权威的图层，请使用状态过滤器。您所见的过滤器选项取决于您搜索图层的位置、您是否已登录以及您的登录方法。
      注：

      要根据当前视图限制搜索结果，请选择仅显示地图区域中的内容。例如，如果将场景缩放至美国内华达州，则搜索结果将进行排序，并根据当前视图显示。如果将视图更改为宾夕法尼亚州，通常会返回不同的结果（取决于关键字）。位于当前视图内（并且与关键字匹配）的所有图层都将返回。

    • 单击排序按钮 对图层结果进行排序并根据需要更改排序方向。
    • 要查看有关图层的重要详细信息，请单击其标题。在图层详细信息窗格中，单击查看完整详细信息按钮 打开项目页面以查看关于图层的详细信息。
  4. 单击添加按钮 可将图层添加到场景。
  5. 对要添加的每个图层重复之前的步骤。
    提示：

    可通过单击图层详细信息窗格中的添加到收藏夹将任何图层结果添加到您的收藏夹。您可以通过在 Scene Viewer 中搜索图层时选择我的收藏夹或从内容页面的我的收藏夹选项卡中查看完整的收藏夹列表。

  6. 单击移除按钮 从场景中移除图层。
  7. 您可以向您的场景添加其他图层，或单击完成。

使用 URL 添加图层

您还可以通过输入图层的 URL 将图层添加到场景。

  1. 单击添加图层。
  2. 单击输入图层 URL 或打开按钮 来打开输入图层 URL 窗格。
  3. 输入想要添加的图层的 URL 并为其指定一个名称。
  4. 单击添加将图层添加到场景中。
