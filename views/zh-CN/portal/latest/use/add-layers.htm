将图层添加到地图

在本主题中

  1. 可添加哪些图层？
  2. 搜索图层
  3. 浏览 Living Atlas 图层
  4. 从 web 添加图层
  5. 从文件添加图层
  6. 添加地图注释
  7. 添加路径图层

图层是地图的内容。其包含的信息范围十分广泛。这些主题涉及人类、地球、生活等，由影像、切片、要素等构成。要开始添加图层，请登录到站点，打开 Map Viewer，然后单击添加按钮。您可以搜索图层、浏览 Living Atlas 图层、从 Web 添加图层、从文件添加图层以及添加地图标注。 您也可以获取方向并将路径添加为图层。

注：

即使您没有创建内容的权限，您仍可以通过单击修改地图添加图层来浏览数据。

可添加哪些图层？

下表显示了您可以添加至地图的图层以及可使用的添加方法。保存地图时，添加的任何项目都会随地图一起保存，并且地图会出现在内容页面的我的内容选项卡上。

图层类型添加方法

ArcGIS Server 服务

服务类型包括要素、地图、影像和切片服务。

从 web 添加图层。

Bing 底图

您的组织必须配置有 Bing 地图密钥才能使用此图层类型。

从 web 添加图层。

逗号分隔值 (CSV) 文件 (.csv)*

从文件添加图层或将文件拖放到地图上。

来自第三方数据供应商的 CSV 文件

从 web 添加图层。

来自 ArcGIS Living Atlas of the World 的 Esri 图层

浏览 Living Atlas 图层或从 Living Atlas 中搜索图层。

GeoJSON 文件（.json 或 .geojson）

从文件添加图层。

GPS 交换格式文件 (.gpx)

从文件添加图层或将文件拖放到地图上。

Keyhole 标记语言 (KML) 文档

从 web 添加图层。

地图注释

在 Map Viewer 中创建。

开放地理空间联盟 (OGC) Web 要素服务 (WFS) 图层

从 web 添加图层或从 ArcGIS Server 站点搜索图层。

OGC Web 地图服务 (WMS) 图层

从 web 添加图层或从 ArcGIS Server 站点搜索图层。

OGC Web 地图切片服务 (WMTS) 图层

从 web 添加图层或从 ArcGIS Server 站点搜索图层。

ArcGIS Online 上的公共图层

从 ArcGIS Online 中搜索图层。

路径

在 Map Viewer 中创建或在门户上搜索您可用的图层。

Shapefile (.zip)*

从文件添加图层。

文本文件 (.txt)*

从文件添加图层或将文件拖放到地图上。

来自第三方数据供应商的切片图层

从 web 添加图层或从 ArcGIS Server 站点搜索图层。

组织中托管的 Web 图层

图层类型包括要素图层、地图图像图层、影像图层和切片图层。

在门户中搜索您可用的图层。

*门户必须具备托管服务器才能将压缩 shapefile、CSV 文件或文本文件添加至 Map Viewer。此外，门户必须配置定位器服务，才能添加包含地址的 CSV 或文本文件。

搜索图层

默认情况下，您可以从我的内容中搜索图层，但是您还可以选择从以下内容中搜索图层：

  • 我的收藏夹 - 包括您已标记为收藏项目的图层。
  • 我的群组 - 包括您所属群组的图层。
  • 我的组织 - 包括组织中的图层。

  • Living Atlas - 此选项会返回 ArcGIS Living Atlas of the World 中的所选图层，其中包括交通、人口统计和土地覆被图层。仅当您登录到配置为使用 ArcGIS Living Atlas of the World 内容的门户时，此选项才可用。

  • ArcGIS Online - ArcGIS Online 中的公共内容。

要搜索图层，请完成以下步骤：

  1. 单击添加并选择搜索图层。
  2. 选择要在何处搜索图层：我的内容、我的收藏夹、我的群组、我的组织、Living Atlas 或 ArcGIS Online。
  3. 执行以下任意操作以便查找所需图层：
    • 在搜索框中输入搜索术语。您也可以使用高级搜索帮助缩小搜索结果范围。
    • 使用不同的视图按钮以不同的方式显示图层结果（表 或列表 ）。
    • 单击排序按钮 对图层结果进行排序并根据需要更改排序方向。
    • 单击过滤器按钮 以缩小搜索结果范围。 例如，如果您正在搜索组织中的图层，且已设置内容类别，则可以使用类别过滤器来缩小结果范围。 您也可以使用 ArcGIS 标准类别集或区域过滤器对 Living Atlas 图层结果进行过滤。如果您希望您的搜索仅返回已标记为权威的图层，请使用状态过滤器。您所见的过滤器选项取决于您搜索图层的位置、您是否已登录以及您的登录方法。
      注：

      要根据当前地图范围限制搜索结果，请选择仅显示地图区域中的内容。例如，如果将地图缩放至美国内华达州，则搜索结果将进行排序，并根据地图范围显示。如果将地图范围更改为宾夕法尼亚州，通常会返回不同的结果（取决于关键字）。位于当前地图范围内（并且与关键字匹配）的所有图层都将返回。

    • 要查看有关图层的重要详细信息，请单击其标题。在图层详细信息窗格中，单击查看完整详细信息按钮 打开项目页面以查看关于图层的详细信息。
  4. 要向地图中添加图层，请单击图层结果上的添加按钮 。
    注：

    当您通过搜索向 Map Viewer 添加托管 WFS 图层时，系统将提示您添加单个图层并定义自定义参数。

  5. 对要添加的每个图层重复之前的步骤。
    提示：

    可通过单击图层详细信息窗格中的添加到收藏夹将任何图层结果添加到您的收藏夹。您可以通过在 Map Viewer 中搜索图层时选择我的收藏夹或从内容页面的我的收藏夹选项卡中查看完整的收藏夹列表。

    将图层添加至 Map Viewer 后，您可以配置各个图层。例如，您可以为添加至地图的每个图层更改样式、创建标注以及配置弹出窗口。

    如果您已添加托管 WFS 图层，则可以保存含有托管 WFS 图层中图层的地图，并且这些配置将保存在地图中。将配置保存到每个单独图层时，系统将提示您创建新 WFS 图层。这一新 WFS 图层是对托管 WFS 图层的引用。所有的配置均保存到新图层而不是托管图层。

  6. 要从地图中移除图层，请单击搜索结果列表中图层上的移除按钮 。
  7. 要在完成时关闭搜索窗格，请单击返回箭头。

浏览 Living Atlas 图层

如果您的门户已配置为可以访问 ArcGIS Living Atlas of the World 内容，则您可以浏览 Living Atlas 图层并直接将其添加至地图。您门户组织的成员可以查看这些图层。如果您想在组织外共享的地图包含 ArcGIS Online 订阅者内容或 ArcGIS Online 高级内容，您可以创建基于地图的应用程序并对其进行配置，以允许通过您的组织订阅访问内容。（如果使用高级内容，您的 ArcGIS Online 订阅将产生若干配额成本。）

  1. 单击添加，然后选择浏览 Living Atlas 图层。
  2. 执行以下任意操作以便查找所需图层：
    • 在搜索框中输入搜索术语。您也可以使用高级搜索帮助缩小搜索结果范围。
    • 使用不同的视图按钮以不同的方式显示图层结果（表 或列表 ）。
    • 单击排序按钮 对图层结果进行排序并根据需要更改排序方向。
    • 单击过滤器按钮 以缩小搜索结果范围。例如，您可以使用类别过滤器过滤图层结果，其中包括 ArcGIS 标准类别集中的类别，或者使用区域或项目类型过滤器。
      注：

      要根据当前地图范围限制搜索结果，请选择仅显示地图区域中的内容。例如，如果将地图缩放至美国内华达州，则搜索结果将进行排序，并根据地图范围显示。如果将地图范围更改为宾夕法尼亚州，通常会返回不同的结果（取决于关键字）。位于当前地图范围内（并且与关键字匹配）的所有图层都将返回。

    • 要查看有关图层的重要详细信息，请单击其标题。在图层详细信息窗格中，单击查看完整详细信息按钮 打开项目页面以查看关于图层的详细信息。
  3. 要向地图中添加图层，请单击图层结果上的添加按钮 。
  4. 对要添加的每个图层重复之前的步骤。
    提示：

    可通过单击图层详细信息窗格中的添加到收藏夹将任何图层结果添加到您的收藏夹。您可以通过在 Map Viewer 中搜索图层时选择我的收藏夹或从内容页面的我的收藏夹选项卡中查看完整的收藏夹列表。

  5. 要从地图中移除图层，请单击图层结果上的移除按钮 。
  6. 要在完成时关闭搜索窗格，请单击返回箭头。

从 web 添加图层

您可通过 URL 添加以下类型的图层：ArcGIS Server web 服务、OGC WFS web 服务、OGC WMS web 服务、OGC WMTS web 服务、切片图层、KML 文件、GeoRSS 文件和 CSV 文件。如果您的组织配置了 Bing Maps，您也可以添加 Bing Maps 底图。

选择从 Web 添加图层选项，然后选择要添加的 web 图层的类型。

ArcGIS Server web 服务

ArcGIS Server web 服务是位于 ArcGIS Server 站点上的地图、影像或要素资源。如果已知用于访问 ArcGIS Server 站点服务的用户名和密码，则可添加使用 ArcGIS Server 10 SP1 及更高版本创建的安全服务。将安全服务添加到地图时，系统会提示您输入用户名和密码。任何人查看您的地图时，系统都会提示其登录 ArcGIS Server 以访问受保护服务。如果 ArcGIS Server 登录信息无效，显示的地图将不包含该图层。

使用此选项，您还可以指定 ArcGIS Server 站点的 URL 并从该服务器上的图层列表中进行选择。

  1. 选择 ArcGIS Server Web 服务，然后在 URL 字段中输入图层的 web 地址。

    如果您已配置了 ArcGIS Web Adaptor，则 ArcGIS Server web 服务 URL 的格式为 https://webadaptorhost.domain.com/webadaptorname/rest/services/folder/service/servicetype。如果服务位于根文件夹中，则无需在 URL 中包含文件夹名称。在这种情况下，URL 的格式为 https://webadaptorhost.domain.com/webadaptorname/rest/services/service/servicetype。

    例如，要连接到 ArcGIS Server 地图服务，需输入如下所示的 URL：https://webadaptorhost/webadaptorname/rest/services/service/MapServer。有关详细信息和示例，请参阅 Portal for ArcGIS 管理员指南中的 ArcGIS URL 组件。

    如果需要通过 HTTPS 访问服务，请使用 https 添加图层。

  2. 如果要将图层作为底图使用，请选中用作底图。
  3. 如果您输入的 ArcGIS Server 站点 URL 包含多个图层，并且您要将特定图层添加至地图，请单击浏览更多图层并选择所需图层。
  4. 单击添加图层将图层添加到地图中。

OGC WFS web 服务

OGC WFS 是遵循 OGC Web 要素服务规范的要素服务。 OGC WFS 图层将作为地图图像图层添加到 Map Viewer。添加 OGC WFS 图层后，您可以查看其属性表、进行简单的样式更改、配置弹出窗口、更改透明度及执行分析。

注：

如果您从 ArcGIS Server 站点添加 WFS 服务，Map Viewer 将改为使用 ArcGIS Server 服务。

  1. 选择 WFS OGC Web 服务，然后在 URL 字段中输入图层的 web 地址。

    以下为 WFS 服务 URL 的示例：https://mrdata.usgs.gov/wfs/asbestos?SERVICE=WFS&REQUEST=GetCapabilities。

  2. 选择要添加的图层。无法选择多个图层。
  3. 如果所选图层包含复杂要素，请选择要显示的要素类型：点、线或面。
  4. 单击显示高级选项并根据需要更改以下内容：
    • 最大要素计数 - 输入一个新值以更改从 WFS 请求的、和在 web 地图中绘制的最大要素计数。
    • 交换 X/Y 坐标 - 选中此框以交换 x，y 坐标。
  5. 单击添加图层。
  6. 如果要在 WFS 中添加更多的图层，请重复上述步骤。
提示：

添加图层后还可以设置或更改高级选项，方法是在内容窗格中单击图层的更多选项按钮 ，然后单击 WFS OGC 选项。

OGC WMS web 服务

OGC WMS 是遵循 OGC Web 地图服务规范的地图。 添加 OGC WMS 图层后，您可以启用弹出窗口、配置哪些图层可见、更改透明度、设置可见性范围和更改刷新间隔等。

注：
  • 如果启用弹出窗口的选项对于 WMS 图层不可用，您可能需要再次将图层添加到 Map Viewer 或重新创建 WMS 项目。某些 WMS 图层不支持弹出窗口。
  • 仅当 WMS 服务支持标准 HTTP 缓存规则时，才能对 WMS 图层进行刷新。
  • 如果您从 ArcGIS Server 站点添加 WMS 服务，Map Viewer 将改为使用 ArcGIS Server 服务。

  1. 选择 WMS OGC Web 服务，然后在 URL 字段中输入图层的 web 地址。

    以下为 WMS 服务 URL 的示例：http://ows.terrestris.de/osm/service?SERVICE=WMS&VERSION=1.1.1&REQUEST=GetCapabilities。

    注：

    Esri 并不拥有或维护此服务，因此无法保证此服务可随时访问。

  2. 如果要将图层作为底图使用，请选中用作底图。
  3. 如果您要选择将 OGC WMS 服务中的特定图层（而不是整个服务）添加至地图，请单击获取图层，然后选中要添加的图层旁的复选框。如果您要添加整个服务，则可跳过此步骤。
    提示：

    您可以在添加图层之前对任何图层进行重命名，方法是单击图层名称并键入一个新名称。

  4. 单击添加图层。

OGC WMTS web 服务

OGC WMTS 是遵循 OGC Web 地图切片服务规范的一组缓存图像切片。 添加 OGC WMTS 图层后，您可以更改透明度、设置可见性范围和更改刷新间隔等。

  1. 选择 WMTS OGC Web 服务，然后在 URL 字段中输入图层的 web 地址。

    以下为 WMTS 服务 URL 的示例：http://tileserver.maptiler.com/wmts

    注：

    Esri 并不拥有或维护此服务，因此无法保证此服务可随时访问。

  2. 选择要添加的图层。您只能添加一个图层。
  3. 如果要将图层作为底图使用，请选中用作底图。
  4. 单击添加图层。
  5. 如果要在 WMTS 中添加更多的图层，请重复上述步骤。

切片图层

切片图层是服务器上一组可通过 Web 访问的切片。当您从 Web 添加切片图层时，这些切片可通过来自 Web 浏览器的直接 URL 请求访问。为了在 Map Viewer 中精确地显示切片图层，请确保使用 Web 墨卡托投影（辅助球体）坐标系。

切片图层中相应切片的访问权限取决于切片图层的 URL。URL 包含的参数化值可用于 Map Viewer，以动态请求平移和缩放时与地图的范围和比例相对应的切片。例如，OpenCycleMap 切片图层的 URL 包含用大括号 ({}) 括起来的参数化值：http://www.opencyclemap.org/: http://{subDomain}.tile.opencyclemap.org/cycle/{level}/{col}/{row}.png。

  1. 选择切片图层，然后在 URL 字段中输入图层的 web 地址。

    您可从数据提供商处获取该 URL。

    如果需要通过 HTTPS 访问服务，请使用 https 添加图层。

    注：

    从 web 添加的切片图层必须使用 Web 墨卡托投影（辅助球体）坐标系以确保图层可在 Map Viewer 中正确显示。

  2. 如果切片图层包含一个或多个子域，请在子域字段中指定该切片图层的子域标注。使用逗号分隔标注。

    切片图层提供者使用子域在多个服务器间分发切片请求。并非所有切片图层都具有子域。如果切片图层不含子域，则从 Web 添加图层窗口将不显示“子域”字段。如果对子域标注有任何疑问，请联系切片图层提供者。

    注：

    如果切片跨多个服务器分布，则必须提供子域信息。

  3. 在标题字段中输入切片图层的标题。

    此地图的内容窗格会将您输入的标题显示为该切片图层的名称。

  4. 在制作者名单字段中为切片图层提供者输入任意版权信息或所需属性信息。

    此地图将在右下方显示制作者名单。

  5. 单击设置切片范围设置切片的覆盖范围。

    切片范围代表所有切片的范围。这是一项可选设置。设置范围后，将仅针对指定范围请求切片。但是，由于切片未根据范围大小进行裁剪，因此可能超出范围。

  6. 如果要将图层作为底图使用，请选中用作底图。
  7. 单击添加图层将图层添加到地图中。

KML 文件

KML 文件包含一组地理要素。

  1. 选择 KML 文件，然后在 URL 字段中输入图层的 web 地址。

    您可从数据提供商处获取该 URL。

  2. 单击添加图层将图层添加到地图中。

GeoRSS 文件

GeoRSS 文件是一种 web 订阅源，包括地理要素和位置。。

  1. 选择 GeoRSS 文件，然后在 URL 字段中输入图层的 web 地址。

    您可从数据提供商处获取该 URL。

  2. 单击添加图层将图层添加到地图中。

CSV 文件

以 URL 形式引用的 CSV 文件是基于 Web 的逗号分隔值文本文件，通常包括位置信息。您也可以将不包含位置信息的 CSV 文件添加为表格。这有助于使用连接要素分析工具将 web 上的非空间数据（例如销售信息）连接到空间数据（例如 ZIP Codes 图层）。

  1. 选择 CSV 文件，然后在 URL 字段中输入图层的 web 地址。

    您可从数据提供商处获取该 URL。

  2. 如果 CSV 文件不包含空间信息，可使用 Map Viewer 将其添加为表格。
  3. 单击添加图层。

Bing 底图

只有管理员使用 Bing Maps 密钥对地图进行配置后，才会显示 Bing 底图选项。

  1. 选择 Bing 底图，然后选择要使用的 Bing Maps 类型：道路、航空或混合。
  2. 单击添加图层将图层添加到地图中。

从文件添加图层

您可以通过导入存储在带分隔符的文本文件（.csv 或 .txt）、GPS 交换格式文件 (.gpx)、GeoJSON 文件（.geojson 或 .json）或 shapefile（压缩成 .zip 文件）中的数据向地图添加图层。来自文件的图层存储在地图中。

注：

向地图添加文件时，必须使用世界大地测量系统 (WGS 84) 作为其参考坐标系。

提示：

如果 CSV 文件、shapefile 或 GeoJSON 文件中包含大量数据，或者您希望创建可在多个地图中重复使用的图层，则可以发布托管要素图层而不是向地图中添加文件。

您也可以将不包含位置信息的 CSV 文件以表格形式添加到 Map Viewer。这有助于使用连接要素分析工具将非空间数据（与近期龙卷风相关的财产损失理赔）连接到空间数据（例如 ZIP Codes 图层）。

  1. 选择从文件添加图层。
  2. 单击选择文件并在计算机上定位文件。
  3. 如果添加的是 shapefile，请选择针对 Web 显示概化要素，或者保持原始要素。
  4. 单击导入图层。

    如果要添加 GeoJSON、GPX 文件或 shapefile，图层则会显示在地图上。如果要添加带分隔符的文本文件（CSV 或 TXT），请继续进行下一步。

  5. 如果要添加 CSV 文件或 TXT 文件，请执行以下操作：
    1. 根据需要执行以下操作之一：

      • 如果添加的是包含位置信息的带分隔符的文本文件（.csv 或 .txt），则请选择通过 Map Viewer 按需要使用坐标或地址或地点来查找要素。

        要按地址或地点定位要素，请单击要使用的定位器，或使用默认定位器。

        检查位置字段，然后单击单元格以更改所使用的位置字段或字段。如果要添加 CSV 文件，其中包含支持的坐标字段（例如 MGRS 或 USNG），请选择与您的坐标字段对应的位置字段（如果尚未选择）。

      • 如果添加的 CSV 文件不包含空间位置信息，可使用 Map Viewer 将其添加为表格。
    2. 单击添加图层。

    图层随即添加到地图中。

如果您在向地图中添加文件时遇到问题，请确保您的门户管理员在门户中配置了托管服务器。如果您在地图中添加或查看文件时仍遇到问题，请参阅疑难解答。

拖放文件

除了使用 Map Viewer 的添加按钮导入 .csv、.txt 或 .gpx 文件之外，还可以将文件从计算机拖放到地图上。Map Viewer 将处理这些文件，就像您使用添加一样。Google Chrome 和 Mozilla Firefox 浏览器支持拖放 .txt、.csv 和 .gpx 文件；但 Internet Explorer 和 Apple Safari 不支持该功能。

如果您在向地图中拖放或删除 .csv、.txt 或 .gpx 文件时遇到问题，请确保您的门户管理员在门户中配置了托管服务器。

无法将 shapefile 或 GeoJSON 文件从计算机拖放到地图上；您必须使用添加按钮。

添加地图注释

可通过添加地图注释图层直接将自己的数据添加至地图。向地图添加少量要素时这些图层会非常有用，例如，由城市公园和娱乐部门管理的游泳池。添加地图注释图层时，默认将其存储在地图中。

  1. 选择添加地图注释，然后输入图层的名称。
  2. 选择一个图层模板。
    每个模板包含一组相关的形状和符号。例如，“公园规划”模板包含表示野餐区和自行车路线的符号，而“石油天然气基础设施”模板具有表示炼油厂和处理设施的符号。
    注：

    灾害、常规基础设施以及损失的模板包括联合国人道事务协调办公室 (Office for the Coordination of Humanitarian Affairs) 提供的人道主义图标。源：OCHA。

  3. 单击创建。

    在地图的左侧将出现一个模板。

  4. 单击一个形状或符号，然后单击要向其添加要素的地图。按 Ctrl 键启用捕捉功能。

    捕捉功能可帮助您将正在添加的要素定位在地图上现有要素的旁边。要素的添加方式取决于它的几何类型；例如，添加点和圆时，在地图上单击即可，而在添加线和面时，需在地图上单击并绘制形状，然后双击完成。

  5. 添加下列关于弹出窗口中的要素信息：
    1. 标题 - 此标题显示为要素弹出窗口的标题。
    2. 描述 - 描述字段包括用于创建富文本（如超链接、颜色和样式等）的格式选项。
    3. 图像 URL -图像必须存储在可从您的组织网络访问的网站上，并且应采用 Web 格式，例如 PNG、GIF 或 JPEG。 要获得最佳效果，图像大小应为 200 像素（宽）x 150 像素（高）。其他图像大小将自动调整为该尺寸，因此您的图像可能被拉伸或收缩。
    4. 相关链接 - 在弹出窗口中单击图像时，此链接可打开一个网站。
  6. 或者，可通过单击弹出窗口底部的更改符号来创建您自己的符号。
    1. 选择来源，然后单击一个符号以使用预定义符号。
    2. 选择 URL，然后输入符号文件的 URL。符号必须存储在您的组织所在网络可以访问的网站上。 图像文件应为 PNG、GIF 或 JPEG 等 web 格式。要获得最佳效果，图像大小应为 200 像素（宽）x 150 像素（高）。其他图像大小将自动调整为该尺寸，因此您的图像可能被拉伸或收缩。
    3. 可使用符号大小滑块调整符号的大小。
    4. 执行以下任意操作以对地图注释进行更改：
      • 通过拖动显示在形状上的折点来更改现有符号的大小。
      • 从图层中移除要素的方法为：单击要素内部以重新打开弹出窗口，然后单击删除。
      • 通过在地图上单击符号内部来编辑现有要素。此操作将重新打开弹出窗口。
    5. 单击完成，然后单击关闭。

    将自动保存添加到弹出窗口的信息。

  7. 添加地图注释后，执行以下任意操作以对地图上图层的显示方式进行更改：
    • 设置可见性范围，以确定地图注释图层何时显示在地图中。
    • 隐藏所有与图层关联的要素的方法为：单击详细信息，然后单击内容，并取消选中图层名称左侧的框。
    • 可通过单击详细信息，然后单击内容来移除图层。单击图层，单击更多选项，然后单击移除。
  8. 要将地图注释图层保存为地图的一部分，请单击保存按钮 ，输入标题、标签和摘要，然后指定要保存地图的文件夹。单击保存地图。
    提示：

    您也可以将地图注释图层作为要素集合项目保存在您的内容中。通过执行此操作，可以编辑其项目详细信息，并与其他人共享。要保存地图或地图注释图层，您必须具有创建内容的权限。

添加路径图层

如果您已登录组织的帐户，可以使用 Map Viewer 获取方向并在地图中创建路径图层。如果您具有创建内容的权限，则可以在内容页面的我的内容选项卡上将路径图层保存为项目，将图层和地图一同保存，或者同时执行这两种操作。如您将路径图层保存为项目，则可以编辑其项目详细信息并与他人共享。可在我的内容或图层项目页面中，或通过使用 Map Viewer 添加菜单中的搜索图层选项将任意现有路径图层项目添加到新地图或现有地图。如果将路径图层另存为项目，则同样可以添加这些创建为分析工具（如查找最近点、计划路径和连接起点与终点）输出的路径图层。

在 Map Viewer 中，您可以配置图层、编辑路径以及对路径执行分析。具有一个或多个路径图层的地图与其他地图类似，可对其进行配置、通过 web 应用程序、故事模板、博客等进行共享，也可在 ArcGIS Enterprise 客户端（如 iOS 设备、Android 手机和 ArcGIS Pro）上使用。

创建新路径图层

要创建新路径图层，请执行以下操作：

  1. 登录门户。
  2. 打开 Map Viewer。
  3. 单击方向 ，随即在地图左侧显示方向窗格。通过添加起点和一个或多个目的地来获取方向。
  4. 单击保存按钮 ，为图层输入名称并选择要保存该路径的文件夹。单击保存。
    该路径在地图的内容窗格中另存为路径图层并在内容页面的我的内容选项卡上另存为路径图层项目。
    注：

    要保存路径图层，您必须具有创建内容的权限。

  5. 执行以下任一操作：
    • 单击共享路径以查看并编辑路径图层的项目详细信息或将其与他人共享。
    • 根据需要对路径进行更改，包括添加目的地，更改出行模式等。
    • 根据需要配置路径图层，包括更改透明度，重命名等。要执行此操作，请关闭方向窗格，转到内容窗格并单击更多选项按钮 以查看配置图层的选项。
  6. 通过执行以下任一操作来保存对路径所做的任何更改：
    • 在内容窗格中，单击更多选项按钮 ，然后单击保存图层。
      提示：

      保存路径后，您可以在内容窗格中对路径进行进一步更改，包括添加目的地，更改出行模式等，方法是单击内容窗格中的更多选项按钮 ，然后单击编辑路径。随即打开方向窗格，您可以在其中编辑路径。

    • 在方向窗格中，单击保存按钮 ，然后单击保存。
      提示：

      如要将更新的图层保存为不同的路径图层项目，请根据需要更改结果图层名称和文件夹，然后单击另存为新。

添加现有路径图层

如果您想要在 Map Viewer 中配置或编辑现有路径图层，请执行以下操作：

  1. 登录门户。
  2. 执行以下操作之一将现有路径图层添加到地图：
    • 打开 Map Viewer 并使用搜索图层选项将路径图层添加到地图。
    • 在内容页面的我的内容选项卡中，单击列表视图或格网视图中路径图层的缩略图。

    该路径即会显示在地图上，同时显示方向窗格。

  3. 执行以下任一操作：
    • 根据需要对路径进行更改，包括添加目的地，更改出行模式等。
    • 根据需要配置路径图层，包括更改透明度，重命名等。要执行此操作，请关闭方向窗格，转到内容窗格并单击更多选项按钮 以查看配置图层的选项。
  4. 要保存更改，请执行以下操作之一：
    注：

    要保存路径图层，您必须具有创建内容的权限。

    • 在内容窗格中，单击更多选项按钮 ，然后单击保存图层。
    • 在方向窗格中，单击保存按钮 ，然后单击保存。
    • 如要将更新的图层保存为不同的路径图层项目，请根据需要更改结果图层名称和文件夹，然后单击另存为新。
      提示：

      保存路径后，您可以在内容窗格中对路径进行进一步更改（包括添加目的地，更改出行模式等），方法是单击内容窗格中的更多选项按钮 ，然后单击编辑路径。随即打开方向窗格。

配置路径图层

如果打算同其他人共享路径（并具备共享权限），您可能需要为特定群体配置路径图层。例如，如果您就职于应急管理机构并计划与警方共享疏散路线地图，那么您可能需要将内容窗格中的路径图层和弹出窗口中的路径名重命名为可供警方使用的名称。

在路径图层上，可以更改透明度、重命名图层、将其隐藏在图例中、编辑路径并保存对图层的更改。如果地图中还有其他要素图层，还可以在内容窗格中上移或下移图层。您可以通过内容窗格中的更多选项按钮 访问这些选项。

路径图层包含下列子图层。在每个子图层上，您可以设置可见性、配置弹出窗口、更改样式、查看属性表、创建标注和将图层隐藏在图例中。但您无法更改子图层的名称。

  • DirectionPoints - 一组表示方向策略（如到达或离开停靠点、左转或右转以及路径沿途的其他事件）的点要素。该图层包含路径的逐步导航。默认情况下，该图层不会显示在地图上。可通过（在内容窗格中）选中图层名称旁边的复选框将其显示出来。
  • DirectionLines - 一组用于路径各段的线要素。此图层可用于在地图上显示逐步导航。默认情况下，不显示该图层。可通过（在内容窗格中）选中图层名称旁边的复选框将其显示出来。
  • RouteInfo - 路径的线要素图层。该图层包含总驾驶时间和距离。
  • Stops - 包含起始点和目的地，或仅含目的地的点要素图层。
  • Barriers - 包含添加的所有限制型或增加成本型点障碍的点要素图层。仅当将点障碍添加或保存到路径后才会创建该子图层。
  • PolygonBarriers - 包含添加的所有限制型或调整成本型面障碍的面要素图层。仅当将面障碍添加或保存到路径后才会创建该子图层。
  • PolylineBarriers - 包含添加的所有限制型线障碍的线要素图层。仅当将线障碍添加或保存到路径后才会创建该子图层。
注：

要了解有关可向路径添加的障碍类型的详细信息，请参阅障碍。目前，仅可在 Map Viewer 中创建限制型线障碍。但是，在 ArcGIS Pro 中，所有障碍类型都可以与路径一同创建和保存，并可在 Map Viewer 中查看。
