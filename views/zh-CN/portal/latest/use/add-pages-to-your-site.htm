页面可以提供更多空间，用于展开站点。例如，组织可以为每个部门添加页面。团队也可以使用页面来高亮显示工程的特定部分，其中包括目标、资源或状态更新。

与站点一样，可创建的页面数量没有限制。您还可以将同一页面添加到多个站点，以便在多个地点获取相关内容。

注：

页面可在文档集合中的站点搜索结果中找到。仅具有对页面访问权限的用户（核心团队、组织中的成员或公众）可以查看搜索结果中的页面。

哪些用户可以添加页面？

管理员和核心团队成员可以向站点添加页面。

添加页面

要开始执行操作，您必须已创建站点或者作为核心团队的成员以添加或编辑页面。页面将自动与核心团队共享，因此团队成员可以访问和编辑页面。

  1. 登录并单击单击概览页面上的站点。
  2. 单击以打开您要向其添加页面的站点。
  3. 在自定义面板上，单击右上角的页面图标。
  4. 要添加页面，请选择以下其中一个选项：新建页面或单击链接页面以添加现有页面。
    • 新建页面 - 输入页面名称，然后单击下一步。
      注：

      您提供的名称还用于自动生成页面 URL。空格将替换为连字符，并且所有特殊字符将被移除。每个页面 URL 的前面带有 /pages/。有关编辑页面 URL 的更多信息，请参阅本文档的编辑页面 URL 部分。

    • 链接页面 - 从可用页面列表中进行选择。或者，更新页面 URL。单击 链接页面确认您的选择，或单击选择其他页面返回。
  5. 单击保存。

接下来，确保刚刚添加的页面具有页面摘要。您为页面提供的摘要将显示在搜索结果中。

提供页面摘要

要为页面添加摘要，请执行以下步骤。

  1. 在站点的自定义面板上，单击右上角的页面图标。
  2. 在页面列表中，找到您想编辑的页面。
  3. 单击页面信息。
  4. 在页面摘要字段中输入摘要。
  5. 单击保存。

编辑页面 URL

完整的页面 URL 将为 https://webadaptorhost/webadaptorname/apps/sites/#/your-site-name/pages/your-new-page。

页面 URL 末尾的文本称为 slug。它可以进行修改，以帮助您实现符合组织现有标准的 Web 体验。

  1. 在站点的自定义面板上，单击右上角的页面图标。
  2. 在页面列表中，找到您想编辑的页面。
  3. 单击页面信息。
  4. 在页面 Slug 字段中输入新的 slug。
  5. 单击保存。

设计页面布局

您可以按照与设计站点相同的方式来设计页面。但是，您无法编辑页面的页眉或页脚，因为页眉或页脚继承自站点。

  1. 在站点的自定义面板上，单击右上角的页面图标。
  2. 在页面列表中，找到您想编辑的页面。
  3. 单击自定义面板中的布局。
  4. 将行卡片拖动到页面布局以开始设计您的页面。您可以按照与设计站点相同的方式来设计页面。
  5. 单击查看以预览页面。
  6. 单击保存。

更改页面共享设置

与您的团队、组织或公众共享您的页面。可以与站点以外的其他受众共享页面。例如，属于公共站点的私有共享页面，只有在登录时对其具有查看权限的用户才可访问。

  1. 页面的编辑导航栏上提供了共享选项。
    • 与组织共享 - 选择第二个图标可以与 ArcGIS Online 组织的每一位成员共享该页面。与 Enterprise 门户共享 - 选择第二个图标可以与 Enterprise 门户的所有成员共享该页面。
    • 与公众共享 - 选择第三个图标可以与公众共享该页面。
      注：

  1. 登录到 Portal for ArcGIS 并单击概览页面上的站点以查看站点列表。
  2. 在站点顶部的编辑导航栏中，选择以下共享选项之一：
    • 与团队分享 - 选择第一个图标可以与站点核心团队的其他成员共享该页面。团队成员必须登录才能访问该页面。
    • 与组织共享 - 选择第二个图标可以与 ArcGIS Online 组织的每一位成员共享该页面。与 Enterprise 门户共享 - 选择第二个图标可以与 Enterprise 门户的所有成员共享该页面。
    • 与公众共享 - 选择第三个图标可以与公众共享该页面。
      注：

      在创建页面时，建议您仅与核心团队或组织共享该页面，直到您准备好向公众开放站点。

在站点上提供页面的链接

在站点或其他页面中添加页面的链接。

使用菜单链接

您可以使用站点标题中的菜单链接添加页面的链接。有关详细信息，请参阅配置全球导航和菜单链接。

使用 HTML

如果要创建指向页面的超链接，请使用 HTML <a> 语法插入链接。如果在 <a> 标签中使用相对路径而非绝对路径，会防止在您单击链接时对整个页面进行刷新。

  1. 添加文本卡片或打开页面布局上的现有卡片，然后单击工具栏中的代码视图。
  2. 在希望插入链接的位置处插入 <a> 标签。
  3. 在 <a> 标签的 href="" 属性之间粘贴要添加的页面 slug，例如 <a href="#/mysite/pages/target-page">Visit the linked page</a>。
  4. 单击编辑导航栏中的保存以确认更改，然后单击视图以测试新的链接。
    5. 提示：

    要在新选项卡中打开链接，请直接在 <a> 标签上的 href 属性后追加 target="_blank"。

在多个站点上使用单个页面

您可以在多个站点上使用单个页面。如果特定站点目前正在使用您的页面，并且您希望在其他站点上使用该页面，请按照以下步骤进行操作：

  1. 在要添加现有页面的站点上，单击页面图标。
  2. 单击链接页面以查看组织中所有页面的列表。
  3. 选择您想要导入到站点的页面。
  4. 确认所选页面的详细信息并更改页面 URL。
  5. 单击链接页面以完成链接。然后将编辑页面和附加站点。
注：

无法在同一站点上包含使用相同 URL 的两个页面。

警告：

如果编辑已链接页面的布局，则将更改所有使用此页面的布局。

管理站点的页面

  1. 找到页面所属的站点并将其打开。
  2. 单击页面图标以查看属于该站点的页面。
  3. 单击删除按钮。
