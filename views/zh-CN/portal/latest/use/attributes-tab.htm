属性

在本主题中

  1. 标识

属性选项卡可用于执行应用程序级别的基本自定义设置，例如配置应用程序通栏、状态和其他数据源。

标识

可向通栏添加徽标、标题、子标题和链接。

  1. 如果可用，则应用程序徽标默认设置为组织定义的徽标。否则，请单击徽标图标并选择自定义以打开文件资源管理器，然后在单击上传后选择一个图像文件。在链接文本框中输入图像的超链接。如果您不希望应用程序中出现徽标，请单击徽标上的 X 图标将其移除。

    使用 Internet Explorer 8 浏览器时，上传图像的大小不得超过 21K。

    应用程序启动时，如有必要，可展开浏览器以查看所有上述项目，尤其是当文本字体较大时。

  2. 单击自定义字体样式按钮 以自定义应用程序标题的字体样式。
  3. 单击自定义字体样式按钮 以自定义应用程序子标题的字体样式。
  4. 单击添加新链接可将超链接添加到通栏并提供名称和有效的 URL。链接可以是浏览器支持的任何协议，例如 HTTP、HTTPS、MAILTO 和 FTP。
    5. 要编辑链接，请选择单击以编辑链接按钮 。要移除链接，请单击 X 图标。
    警告：

    最多可添加四个链接。

应用程序状态

选中应用程序状态复选框可以在退出应用程序时保持地图范围和图层可见性。该选项默认处于开启状态。构建 3D 应用程序时，此选项不可用。

注：

此选项仅适用于 10.5 或更高版本。

订阅者内容访问

  1. 单击订阅者内容访问旁边的箭头。这可以配置应用程序使用的 ArcGIS Online 订阅者内容，从而允许用户通过您的订阅进行访问。订阅者内容有两种类型：订阅者图层和高级服务。
    • 地图图层 - 如果应用程序具有高级服务，则将列出该高级服务。消耗配额图标 表示图层消耗配额。选中使用代理复选框可允许公共用户通过您的订阅访问图层。或者，可以设置请求限制和间隔来控制使用情况。
    • 自定义 URL - 如果微件中已配置高级服务 URL，可单击添加新 URL 以添加 URL 和标题。这将允许公共用户通过您的订阅访问服务。或者，可以设置请求限制和间隔来控制使用情况。
      注：

      应用程序启动时，自定义 URL 将覆盖微件中配置的 URL。

其他数据源

其他数据源选项可用于配置在应用程序级别共享的自定义数据，以便应用程序中的所有微件能够同时快速访问和响应数据。应用程序启动后，将从服务器查询所有已配置的数据源，并将这些数据源加载到浏览器客户端。

可以添加两种类型的数据源：图层和统计数据。图层是要素图层、表格或影像服务矢量图层的子集数据。统计数据可以存储分组数据（如果指定）的要素计数，并根据一个或多个数值字段来计算分组（如果指定）的总和、平均值、最大值和最小值。

注：

目前，仅信息图表微件具有可使用其他数据源的配置选项。

添加图层数据源

  1. 单击其他数据源，然后单击新增。在弹出菜单中选择图层。
  2. 从地图或门户选择一个图层，或输入一个服务 URL。支持源自要素服务、地图服务或影像服务的要素图层、表格和影像服务矢量图层。
  3. 在配置数据源窗格中，执行以下操作：
    1. 在名称下指定数据源的名称。
    2. 单击过滤器下的设置按钮以按表达式或集定义查询条件。还可以根据地图范围来启用和禁用过滤器。如果选中仅当前地图范围内的要素，则地图范围更改时将更新数据。
    3. 单击设置按钮可指定将要检索的字段。
    4. 在记录下方，指定将检索的记录数和顺序。数值下拉菜单中提供以下三个选项：
      • 服务限制 - 发布服务时指定的最大记录数。
      • 全部 - 服务中的所有记录。
      • 自定义 - 希望从服务中检索的记录数。
    5. 在刷新间隔下方，指定适用的数据源刷新间隔。
    6. 单击预览可预览通过上述步骤配置的数据。
    7. 单击确定。

添加统计数据源

  1. 单击新增，然后选择弹出菜单中的统计数据。
  2. 从地图或门户选择一个图层，或输入一个服务 URL。支持将要素图层、表格和影像服务矢量图层作为输入。这些内容可源自要素服务、地图服务或影像服务。
    注：

    数据源必须至少包含一个数值字段。

  3. 在配置数据源窗格中，执行以下操作：
    1. 在名称下指定数据源的名称。
    2. 单击过滤器下的设置按钮以按表达式或集定义查询条件。
    3. 在统计数据下方，选择统计数据所依据的组，然后设置针对其计算总和、平均值、最小值和最大值等统计数据的字段。
    4. 在刷新间隔下方，指定适用的数据源刷新间隔。
    5. 单击预览可预览通过上述步骤配置的数据。
    6. 单击确定。

编辑数据源

  1. 将鼠标悬停在表格中的现有数据源上，然后单击铅笔按钮 以打开配置数据源窗口。进行相应更改。
  2. 要移除现有数据源，请单击 x。
  3. 在设置刷新间隔下拉菜单中，选择要对数据源设置间隔的方式。
