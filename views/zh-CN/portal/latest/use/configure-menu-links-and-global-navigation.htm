配置全球导航和菜单链接

菜单链接即添加到页眉的链接。这些链接需要一个描述性标注，用于告知访问者在使用链接时将转至的位置。管理员和团队成员可以使用菜单链接来方便站点访问者导航至与该站点相关的其他内容，例如其他站点和页面或者外部 web 链接。

什么是全球导航？

每个新站点都包含全球导航。全球导航是一种站点功能，用于在站点页眉上方放置一个紧凑型栏。此栏包含以下选项：

  • 登录 - 登录选项允许管理员、团队成员和组织的其他成员访问您的站点。如果没有全球导航，则用户将需要登录到 Portal for ArcGIS Enterprise 并通过应用程序切换器来访问站点。
  • 用户个人资料 - 登录后，用户个人资料下拉菜单将提供对站点设置和概览页面的访问权限，具体取决于其权限。登录后，用户即可访问其用户个人资料和通知。
  • 搜索按钮 - 单击该按钮后，搜索按钮将展开为搜索栏，任何人都可以使用该搜索栏来浏览与站点共享的所有公共内容。访问者无需登录即可在站点上输入搜索内容。登录的成员还可以搜索私有内容（与其所属团队或组织共享的项目）。
  • 新建按钮 - 管理员和核心团队成员可以使用新建按钮来创建新内容。

创建菜单链接后，我可以链接到什么内容？

您可以使用 URL 链接到任何外部网站。您还可以从属于门户的项目中进行选择。支持以下项目：

  • Web 制图应用程序
  • 仪表盘
  • 数据集、shapefile 和 CSV 文件
  • 站点中创建和编辑内容的权限的用户
  • 页面
  • Web 地图
  • 文档
  • Survey123 表单

有关 Portal for ArcGIS Enterprise 支持的项目的完整列表，请参阅支持的内容。

向页眉添加菜单链接

常用指南建议页眉中最多应包含 7 个菜单链接，因为简洁的菜单最容易记住。但是，如果内容跨越的范围较大，则此规则并非始终适用。如果要添加大量菜单链接，则必须考虑菜单链接的顺序和有意义的标注。

  1. 登录到 Portal for ArcGIS 并单击概览页面上的站点以查看站点列表。
  2. 单击以打开站点。
  3. 在自定义面板上，单击页眉。
  4. 单击菜单。
  5. 单击添加链接，然后从以下两个选项中进行选择：
    • 至现有内容 - 从您、您的团队、您的组织或公众创建的内容中进行选择。
    • 至外部页面 - 在链接目标字段中复制粘贴链接。在链接名称字段中提供链接的名称。

    提示：
    描述性标注可以回答问题，而非简单地指示链接后面的内容类型。例如，与“常见问题”、“成功案例”和“了解更多”相比，“视频”标注比较一般。嵌入视频或照片作为故事或资源页面的一部分比没有上下文的视频链接更加有效。

  6. 单击保存。

移动设备上的菜单链接

要确保用户在移动设备上查看站点时获得最佳体验，需要将站点的页眉设计为移动响应型页眉。这意味着站点页眉中包含的所有内容（其中包括徽标、名称和菜单链接）都可以折叠到一个下拉菜单中，具体取决于设备的屏幕尺寸。

提示：

全球导航会影响移动设备上的页眉显示。如果您向页眉添加了徽标，则在移动设备上查看徽标时，其将显示在全球导航栏中。这意味着，如果您的徽标和全球导航背景均为浅色，则徽标的颜色对比度可能不足，无法进行访问。如果页眉中同时包含名称和徽标，请使用简称以确保站点的标题不会被截断。

更改菜单链接标注

请按照下列步骤更改菜单链接的名称。

  1. 打开要编辑的站点。
  2. 在自定义面板上，单击页眉。
  3. 单击菜单。
  4. 在要编辑的链接上单击设置按钮。
  5. 单击编辑。
  6. 在链接名称字段中输入新名称。
  7. 单击保存。

重新对菜单链接进行排序

菜单链接的显示顺序会有所不同。对于您认为对访问者最具价值的资源，通过将其链接放置在菜单开头可以为其授予较高优先级。最近的内容可以链接到菜单末尾。用户最有可能记住这些位置的链接。

  1. 打开要编辑的站点。
  2. 在自定义面板上，单击页眉。
  3. 单击菜单。
  4. 单击菜单链接，将其拖动到列表中的新位置。
  5. 单击保存。

配置社交媒体链接

将 Instagram、Facebook 和 Twitter 的图标添加到站点菜单。

  1. 打开要编辑的站点。
  2. 在自定义面板上，单击页眉。
  3. 单击社交媒体图标。
  4. 单击以打开要显示的图标。
  5. 对于每个要显示的图标，请粘贴要链接到的帐户的 URL。
  6. 单击保存。

移除菜单链接

请按照以下步骤从站点的页眉中移除菜单链接：

  1. 打开要编辑的站点。
  2. 在自定义面板上，单击页眉。
  3. 单击菜单。
  4. 在要编辑的链接上单击设置按钮。
  5. 单击编辑。
  6. 单击移除链接。
  7. 单击保存。
