配置站点设置

每个站点都有其自己的设置和功能。通过配置站点的设置和功能，管理员和核心团队成员可以自定义站点的工作方式。

每个站点都有其自己的设置。这些设置允许管理员和核心团队成员自定义站点的工作方式。设置包括以下配置选项：

  • 用于更改站点标题、描述和默认 URL 的基本选项。
  • 用于配置站点 URL 的自定义域、预览页面的底图和强制使用 HTTPS 的高级选项。

要开始执行操作，请登录到 Portal for ArcGIS 并单击概览页面上的站点以查看站点列表。

提示：

在浏览器中查看站点时，也可以对其进行编辑。确保您已登录，然后在站点左侧找到编辑按钮。

提供站点名称和描述。

请按照以下步骤更改站点的标题和描述。

  1. 在站点的编辑模式下，打开自定义面板。
  2. 单击设置。
  3. 在站点标题字段中输入您的站点的标题或将现有标题更改为新标题。
    注：

    站点标题将显示在站点编辑导航栏的第二个下拉菜单中。此标题也是在概览页面上访问的列表中用于识别您的站点的名称。通过执行使用页眉和主题选项来打造站点品牌中提供的步骤，可以配置在页眉中显示站点名称的选项。

  4. 在关于站点字段中提供一些与站点相关的背景信息。
  5. 单击保存。

配置 URL

站点的 URL 是用户用于访问您的站点的 web 地址。创建新站点时，将基于您在站点标题字段中提供的名称生成默认 URL。但是，您也可以按照以下步骤手动更改您的 URL。

注：
您所提供的名称将显示在组织标准站点 URL 的 结尾处 例如， https://webadaptorhost/webadaptorname/apps/sites/#/mysitename。最多不可超过 64 个字符。

  1. 在站点的编辑模式下，打开自定义面板。
  2. 单击设置。
  3. 在站点 URL字段中，输入您想使用的 URL。URL 预览将显示新 URL 的外观。
  4. 通过在自定义收藏夹图标字段粘贴图像 URL 来提供可选收藏夹图标。
  5. 单击保存。

使用自定义底图

您可以将站点配置为使用默认底图。站点的底图显示在站点的内容预览页面上。这意味着与您的站点共享的具有预览页面（数据集和要素图层）的任何项目都包括您选择的底图，以便用户可以查看和浏览空间数据。您的组织也可以在 Map Viewer 中配置地图。

注：

所有自定义底图必须在服务上启用 HTTPS 连接。

  1. 在编辑模式下，打开站点的自定义面板并单击设置。
  2. 单击配置地图。
  3. 使用下拉菜单选择要使用的底图。如果没有看到想要的底图，请确保要使用的底图属于您的组织的底图库。如果您不具有正确的权限，则可能需要管理员的帮助。

HTTPS 支持

仅允许通过 HTTPS 访问组织的 Portal for ArcGIS 设置用于确定您站点的 HTTP 或 HTTPS URL 行为。

配置功能

功能提供了站点功能的灵活性。有两种功能类型：标准选项和测试选项。标准选项已经过全面测试并发布。测试选项仍在开发中，允许您测试即将推出的功能。

  1. 在站点的编辑模式下，打开自定义面板。
  2. 单击自定义面板中的测试。
  3. 单击功能。
  4. 查看可用功能。将在选项下提供标准选项，并在测试下列出测试功能。
  5. 单击切换按钮可启用或禁用功能。
  6. 准备就绪后，请单击编辑导航栏中的保存以将更改应用于站点。

可用功能

以下功能可用于自定义站点的性能。

文档 iFrame

如果您的站点中包含 PDF 或其他文档链接，例如 Word 文档或外部 URL，Document iFrames 功能会直接在站点中显示链接的内容。如果禁用，则提供的链接将在新选项卡中打开，或者将在浏览器中下载文档。

API 资源管理器

启用 API Explorer 功能可为其他人员提供访问权限，以便其浏览 Geoservices API、生成查询以及查看预览页面中的响应。这对于用户了解如何通过编程方式访问数据，以及如何让其他人员使用 ArcGIS 数据构建系统集成来说十分有用。

应用程序页面

如果启用，应用程序页面功能会在预览页面上直接显示 Story Maps 和 Simple Map Viewer。如果禁用，则在搜索中选择这些项目时，会在新的浏览器窗口中将其打开。

私有项目视图

此测试选项允许登录的用户查看他们有访问权限的任何项目，例如与群组、组织或公众共享的群组。

注：

默认情况下，所有新站点均启用此功能。

局部投影

启用此功能将导致在其本地投影而非默认 WGS84 中下载 shapefile。

意见或建议

此测试选项允许用户在登录后查看数据集评论并在您的数据集上写入评论。

全球导航

全球导航将在站点的顶部添加一个导航栏。该栏包括登录、搜索、通知和用户个人资料。如果禁用此功能，则查看您的网站的用户将无法登录。

注：

默认情况下，所有新站点均启用此功能。
