配置时间设置

在本主题中

  1. 验证图层是否包含时间数据
  2. 对图层启用时间
  3. 配置时间设置
  4. 禁用时间动画

采集包含时间信息的空间现象可使地图用户了解在过去某一时刻发生了什么，或在将来可能发生什么。通过以动画形式演示基于时间的数据，可以形象地展示每一步的数据并查看随时间推移所呈现的模式或趋势。这些空间现象包括飓风、龙卷风及其他气象活动、人口迁移、土地利用和环境变化，以及野火或洪水进展等。

某些地图图层启用了时间；这些图层包含的空间数据集针对同一位置的不同时间包含不同信息。如果地图包含启用了时间的图层，则地图为时间感知型地图，可对显示进行配置，以显示特定时间段内的数据，或以动画形式呈现随时间变化的数据。

验证图层是否包含时间数据

可以通过查看 REST 端点来确定地图中的图层是否已启用时间。

  1. 在Map Viewer中打开地图。
  2. 单击详细信息，然后单击内容。
  3. 浏览至包含时间数据的图层，单击更多选项 ，然后单击显示项目详细信息。
  4. 在概述选项卡中，滚动至图层部分，然后单击超链接文本打开图层描述。如果图层已启用时间，则打开的网页中将显示名为时间信息的部分。

对图层启用时间

在将要素图层、影像图层或地图影像图层（地图服务）发布到 ArcGIS Server 站点前，可在 ArcMap 和 ArcGIS Pro 中的要素和影像图层上设置时间属性。一旦将这些启用时间的 ArcGIS Server Web 图层添加到 Map Viewer，即可对其进行配置以显示时间动画。

您也可以发布托管要素图层（其中包括时态数据）并在图层项目页面上为其启用时间。这样，您和您的组织（如果您共享要素）即可使用时态数据创建启用时间的地图。

要在托管要素图层上启用时间，则该图层的所有者或管理员可以执行以下步骤：

  1. 在内容页面的我的内容选项卡中，打开具有时态数据的托管要素图层的项目页面。
  2. 在概述选项卡的图层部分，单击时间设置。
  3. 选中启用时间复选框。
  4. 选择将时间数据记录为时间内的特定事件或者包含开始时间和结束时间的时间范围。
  5. 选择数据中的一个或多个时间字段。
  6. 单击确定。

将托管要素图层添加到 Map Viewer 时，会启用时间动画。

配置时间设置

  1. 在Map Viewer中打开地图。
  2. 单击显示高级选项。此时，可以进行如下配置。要保存配置更改，需要将它们保存到地图中。
  3. 完成配置时间设置后，单击确定。

回放速度

默认速度为时间间隔两秒。可通过将滑块向较慢或较快方向移动来调整回放速度。查看地图的任何用户都可以更改您设置的默认速度。

时间跨度

时间跨度部分显示了地图中每个启用时间的图层的时间轴或时间段。可以查看某一图层的时间轴与其他图层的时间轴的对比情况，还可以更改地图的开始时间和结束时间。默认情况下，地图的时间跨度设置为所有启用时间的图层的时间轴的并集。

仅当时间图层具有已完成的时间轴时，时间跨度才会处理对历史数据显示的配置。它不会处理仍在累积的时间数据。时间跨度的开始时间和结束时间都是显式设置的。

执行以下任意操作以更改时间跨度：

  • 拖动图层时间轴部分中滑块上的手柄手动设置地图的开始时间和结束时间。
  • 在开始时间和结束时间字段中选择明确的值。
  • 单击图层的时间轴，将地图的开始或结束时间设置为图层的开始或结束时间。

时间显示

使用时间显示部分来指定随着时间推移而显示的数据量。包括设置时间间隔，即刷新地图以显示此时有效的数据之前的持续时间。除时间间隔外，还必须指定是显示累积数据，还是只显示当前时间间隔内的数据。

  1. 输入作为时间间隔的数值。
  2. 从下拉列表中选择时间间隔的单位。下拉列表中显示的单位取决于图层数据中的时间轴。要更改这些单位，您必须编辑数据的图层属性。
  3. 对于随着时间推移，选择以下选项之一：
    • 选择仅显示当前时间间隔中的数据可仅显示位于时间间隔内的数据。例如，如果您具有表示 10 年内飓风轨迹的数据且要显示特定年份内发生的所有飓风的轨迹，则可选择此选项（并指定 1 年作为时间间隔）。随着地图以动画形式呈现，您可看到第一年、第二年、第三年的飓风路径，依此类推。每个年份的数据都是独立于其他年份显示的。
    • 选择逐步显示所有数据以显示一段时间内的所有数据。例如，如果您具有表示 10 年内飓风轨迹的数据且要逐年连续显示 10 年内发生的所有飓风的轨迹，则可选择此选项（并指定 1 年作为时间间隔）。随着地图以动画形式呈现，可以看到 10 年内累积的年度飓风路径。

禁用时间动画

如果不希望在地图上显示时间滑块，可在启用了时间的图层上禁用时间动画。

  1. 在 Map Viewer 中打开启用时间的地图。
  2. 单击详细信息，然后单击内容。
  3. 浏览至要更改的图层，单击更多选项 ，然后单击禁用时间动画。
  4. 单击启用时间动画以显示之前从地图中移除的时间滑块。
  5. 如果图层归您所有且您希望将时间动画设置保存为该图层的默认属性，请单击保存图层。
