内容指管理员、站点创建者和核心团队成员用作站点的一部分的数据集、应用程序、文档及其他项目。您可以在内容库中查看共享至站点的所有项目的集合，其中包括公共和私有内容（仅共享至核心团队或 Portal for ArcGIS Enterprise 的项目）。借助内容库，管理员和核心团队成员也可以编辑与站点共享的核心团队群组或其他更新群组共享的站点内容。

注：

项目不包括图像、视频和外部链接，因此您不必将其上传到门户或与群组共享。相反，如果您想要在站点上共享此类内容，可执行添加文本和图像和嵌入式视频、社交媒体和 iframe 中提供的步骤。

我可以共享什么内容？

ArcGIS Enterprise 站点支持可共享至门户的项目。

我如何共享内容？

要将内容添加到站点，您必须使用群组。群组可组织您要共享的内容（项目），如数据集、应用程序或其他站点和页面。使用群组管理器，管理员、核心团队成员和群组所有者可以将内容群组添加至站点。

内容显示在哪里？

内容库由添加到群组管理器的群组支持。管理员和站点的核心团队成员可以添加其自己的群组或使用由组织或公众共享的群组。

属于群组的共享到站点的项目将显示在以下位置：

  • 站点的搜索结果 - 当用户在站点搜索栏输入搜索时，与站点共享的属于群组的内容对与之共享项目的受众（群组成员、组织或公众）可见。
  • 图库、类别和搜索卡片 - 显示单个项目以供用户直接在站点或页面上浏览，或用户可以选择图标以按类别查看相关项目的群组（例如，水文、边界或林业）。有关详细信息，请参阅显示应用程序、数据和 web 地图。
  • 项目预览页面 - 数据集一类的项目包括预览页面，用户可以通过在地图上浏览空间数据并对其属性表进行排序来了解有关搜索结果的更多信息。这些页面还提供了以支持的格式下载项目并访问其 API 的选项。

查看内容库

按照以下步骤查看站点的内容库并访问您可以编辑的项目。

  1. 登录到 Portal for ArcGIS。
  2. 单击概览页面上站点卡片上的管理。
  3. 打开要使用的站点。
  4. 在编辑导航栏中单击站点的下拉菜单并选择内容库。
  5. 通过在搜索结果中单击项目名称来查看项目。
  6. 如果您具有编辑项目的权限，请单击页面左侧的编辑按钮以在门户中转至该项目的概览页面。在此，您可以编辑项目名称、缩略图、描述、标签和共享控制。

    注：
    默认情况下，如果该项目与核心团队群组共享，则站点核心团队的任何成员都可以编辑他们在内容库中看到的项目。要编辑属于其他群组的项目，该用户必须为管理员或群组所有者。配置了更新功能的群组成员也可以编辑项目。

接下来的步骤

按照此建议工作流将您自己的内容添加至站点。

  1. 查看受支持项目列表。
  2. 遵循准备数据的最佳实践。如果您计划使不同类型的数据可供下载，则建议执行此步骤。
  3. 使用群组管理器将内容添加至您的站点。
