复制和保存图层

在本主题中

  1. 复制图层
  2. 保存图层

图层所有者（或管理员）将定义图层的属性。当您创建的地图包含其他人所有的图层时，可以在您的地图中创建图层的副本，以便在复制的图层中配置不同的属性。如果您具有创建内容的权限，则可以在我的内容中将图层保存为新项目，从而能够在其他地图中使用您的图层配置。

  • 复制 - 要自定义图层配置而不影响源图层的配置，请在您的 web 地图中创建图层的副本。这将在您的 web 地图中创建一个引用源图层的图层。*

    此副本仅在您创建副本的 web 地图中可用。

  • 保存副本 - 如果您想要使涂层配置更改能够用于其他的 web 地图，需保存您的图层副本。此操作将在我的内容中创建一个图层项目。请注意，此新图层项目仍引用源图层。*

    保存图层副本的优点是您可以将配置更改保存到您的副本并将复制的图层及其更改后的配置用于其他 web 地图。

注：

*当您复制要素集合或地图注释图层时，将复制源图层中的数据。系统不会保持对源图层的引用。但是，对于以下所有图层源类型，您的副本都将引用源图层中包含的数据。

  • ArcGIS Server 图层
  • 托管要素图层
  • 切片图层
  • 路径图层
  • WFS、WMS 或 WMTS 图层

如果您更改图层副本的配置，将不会影响源图层的配置。如果删除副本，原始数据仍存在于源图层中，但将丢失图层配置。如果源图层的所有者将其删除或移除访问权限（换言之，他们停止通过您能够访问的方式共享图层），但您的副本引用源图层，则副本中的数据不再可用且不会显示在您的地图中。

复制图层

当您希望使用不同的方法来配置图层属性时，请在地图中复制该图层。例如，按医院服务区聚合的健康数据图层中可能包含许多种疾病的属性。您可以创建一个该图层的副本，并根据抑郁症发病率对数据进行符号化。您可以创建该图层的另一个副本，并根据阿尔茨海默症的发病率对此副本进行符号化。如果保存这些图层，可以在不同的地图中重复使用这些图层（数据已经按照需要进行符号化）。无需成为图层所有者也能创建图层副本。

您可以复制 ArcGIS Server 图层、导入到地图中的文件、托管要素图层、切片图层、地图注释、路径图层、WFS 图层、WMS　图层和　WMTS 图层。您无法复制底图、web 上的 CSV 文件、GeoRSS 或 KML。

如果复制的要素图层是可编辑的，则复制后的图层也可编辑。对副本所做的编辑会更改源数据。如果源图层的所有者删除了编辑功能，则您的副本将不可编辑。

  1. 验证您是否已登录，以及是否具备创建内容的权限。
    注：

    即使您没有创建内容的权限，您仍可以通过单击修改地图来复制图层。

  2. 在Map Viewer中打开地图，单击详细信息，然后单击内容。
  3. 浏览至要复制的图层，单击更多选项，然后单击复制。

    图层的副本随即被添加到地图内容中。现在，您即可以配置此图层的属性并将此图层保存为新项目。

保存图层

如果您拥有创建内容的权限，可将 ArcGIS Server 图层、导入到地图的文件、托管要素图层、切片图层、地图注释、路径图层、WFS 图层、WMS 图层和 WMTS 图层另存为我的内容中的新项目，并在其他地图中使用图层。您无法保存底图、GeoRSS、KML 或 web 上的 CSV 文件。在保存从文件中导入的图层或者保存地图注释时，您保存的是实际数据。

从我的内容中将保存的图层添加到 Map Viewer 后，您可以更改图层的显示属性。例如，您可以更改样式、配置弹出窗口、设置可见范围、设置刷新间隔、在图例中隐藏图层和应用过滤器。可以将这些更改内容保存为项目的属性，当图层在地图中出现时，随时都可以将这些属性应用于图层。

注：

只能在您所拥有的并已添加到网站的图层上保存项目属性。该配置仅适用于门户中的项目，不适用于 ArcGIS Server 站点上运行的实际服务。您无法将项目属性保存在从文件导入的图层上，因为这些图层是存储在地图中而不是在门户网站中作为单独项目而存储的。

请按照以下步骤保存图层的副本：

  1. 验证您是否已登录，以及是否具备创建内容的权限。
  2. 在Map Viewer中打开地图，单击详细信息，然后单击内容。
  3. 浏览至要保存的图层，单击更多选项，然后单击保存图层。
  4. 如果您是第一次保存图层或者正在保存图层副本，将使用图层配置在站点中创建一个新项目：
    1. 输入标题。
    2. 如果您的组织已设置内容类别，请单击分配类别并最多选择 20 个类别以方便其他用户查找您的地图。您也可以使用过滤器类别框来缩小类别列表范围。
    3. 输入用于描述该图层的标签和摘要。
    4. 在我的内容中选择要保存图层的文件夹。
    5. 单击创建项目。
    5. 带有您的属性配置的新项目已添加至我的内容。
