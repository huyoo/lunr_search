创建流域

创建流域工具可为输入图层中的每个点创建集水区。

注：

如果在 Map Viewer 中没有看到创建流域工具，请联系您的门户管理员。您的门户可能未进行水文配置。创建流域需要水文实用程序服务。

有关详细信息，请参阅配置实用程序服务。

工作流示意图

示例

  • 新斯科舍鲑鱼协会正尝试通过提高水电站坝区域的栖息地连通性来增加大西洋鲑种群数量。创建流域工具可用于确定在选定的大坝上修建鱼梯后可获得的栖息地数量。

  • 很多雨水道连接了河流、湖泊、海洋或其它水道，这使得任何流入水道的东西均成为下游区域的潜在汚染物。创建流域工具可用于确定汇入雨水道中的水从何而来。

用法说明

要求单个输入点要素。这些点可通过输入下拉菜单进行加载，或使用绘制工具进行创建。系统将针对每个点计算流域。

使用搜索到最近水系的距离修正没有直接位于水系线上的点。距离可采用英里、码、英尺、千米或米等单位进行测量。要使用输入点的准确位置，请将搜索距离指定为 0。

结果为两个图层，其中包含流域和调整后的输入点。流域图层含有与点和其他四个字段相同的属性：

  • PourPtID - 输入点要素中的唯一 ID 字段。
  • Area Square Kilometers 或 Area Square Miles - 流域面积。面积字段的单位将取决于个人资料中的单位设置。
  • Description - 介绍分析中所使用的源数据库。US30m 衍生自美国大陆适用的 30m NHDPlusV2.1 以及北纬 60 度与南纬 56 度之间地区，加上加拿大和阿斯拉加州在北纬 60 度的部分地区适用的 90m HydroSHEDS。
  • DataResolution - 分析中使用的数据源分辨率。

如果选中使用当前地图范围，则仅使用当前地图范围内的要素计算流域。如果未选中，则将分析输入图层中的所有要素，包括超出当前地图范围的要素。

限制

  • 输入必须是点要素。
  • 输入最多可以有 1,000 个要素。

创建流域的工作原理

创建流域工具使用托管数字高程模型 (DEM) 来创建集水区。所用 DEM 的分辨率取决于分析位置，且将记录在结果图层的 DataResolution 字段中。

对于流域相关的结果，输入点必须位于水系线上。搜索到最近水系的距离选项可将输入点移至搜索距离范围内最近的水系线上。如果不指定搜索距离，该工具将计算并使用保守搜索距离。

出于分析目的，已由 Esri 使用标准水文模型对水系线进行了预计算。如果搜索距离范围内不存在水系线，则使用搜索距离范围内累积流量最大的位置。

类似工具

使用创建流域根据水系线沿线的一个或多个点来确定上游汇流区域或流域。其他工具或许在解决略有差异的类似问题时十分有用。

Map Viewer 分析工具

如果要确定各点的下游方向水流路径，请使用追踪下游工具。

ArcGIS Pro 分析工具

创建流域使用与水文分析工具集相似的功能执行完整的工作流。
