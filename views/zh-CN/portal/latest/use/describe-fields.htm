描述属性字段

在本主题中

  1. 设置显示名称
  2. 提供字段描述
  3. 定义字段值类型

为帮助使用您的托管要素图层的用户理解每个字段所存储的内容，您应提供有用的易于阅读的字段显示名称、提供文本来描述每个字段的用途并定义每个字段所包含的数据类型。

作为托管要素图层的所有者或组织管理员，您可以在每个字段的详细视图中添加此信息。此信息可帮助用户确定图层中的数据是否满足其需要，如何以最佳方式将样式应用于图层，并为编辑者提供更新或添加属性值所需的信息。

设置显示名称

您可以更改其他用户查看托管要素图层的属性表时看到的字段名称。设置一个简明地传达字段内容的显示名称，以帮助其他用户了解数据。

更改字段的显示名称不会影响存储在属性表中的字段名称。因为显示名称是字段的属性，而非字段名称，所以可以包含空格和非字母数字字符。

要更改其他用户查看属性表时看到的字段名称，请执行以下步骤：

  1. 在图层的项目页面中，单击数据选项卡以显示表。
  2. 单击字段。
  3. 如果您的托管要素图层包含多个子图层，请从图层下拉菜单中选择要更改的子图层。

    字段列表显示每个字段的显示名称、字段名称和数据类型。

  4. 单击想要更改的字段的显示名称来打开字段的详细信息。
  5. 单击显示名称旁的编辑，并输入新名称。

    在传达字段内容的同时，尽可能保持名称简短。

  6. 完成后，单击保存。

提供字段描述

字段描述可以帮助您传达字段所包含的内容，而无需使用冗长复杂的字段名称。

在托管要素图层项目页面的数据选项卡上，可以在每个字段的详细视图中添加字段描述。

  1. 打开要添加字段描述的托管要素图层的项目页面。
  2. 单击数据选项卡，然后单击字段按钮。
  3. 单击第一个字段名称以打开字段的详细信息视图。
  4. 对于描述，可单击编辑，然后输入字段所存储信息的摘要。
  5. 单击保存以添加描述。

用户在使用 Map Viewer 中的图层时，可以在设置图层样式时以及在地图中打开表格时访问每个字段的描述。此信息有助于他们了解如何使用和符号化数据。

定义字段值类型

为帮助用户和客户端应用程序确定字段值的使用和表示方式，可以定义托管要素图层的每个属性字段中存储的值的类型。当您在 Map Viewer 中绘制图层或配置弹出窗口时，ArcGIS Enterprise 会使用字段值类型以显示最相关的选项。

字段值类型

下表列出了可分配给字段的每个值类型，以及每个值类型所表示的内容：

字段值类型每个要素的字段所包含的内容

名称或标题

表示每个要素的名称、标题、标注或关键字的文本。

描述

提供要素较长描述（而非只是名称或标题）的文本。

例如，假设您拥有一个图层，用于从公众采集有关社区问题的信息。可以使用“描述”字段值类型来定义该图层中的字段，以允许用户指定问题的性质。

类型或类别

字段值表示基于共同特征对要素进行分组的类型或类别；例如，土壤类型、分区代码、物种或资产类型。

计数或数量

表示特定属性数量的整数（无小数）。

百分比或比例

此字段中的数值反映了不同数量之间的关系；例如，人口中男性的百分比或者男性和女性数量之间的比例。

测量值

反映可精确测量的特征的数字；例如，高程、距离、温度或年龄。

唯一标识符

此字段中的值用于明确区分要素或实体；例如，评估人员的宗地编号、成员资格 ID、发票号或电子邮件地址。

每个实体的每个值都必须唯一，但这不意味着此字段中的所有值对于图层都必须唯一。

例如，如果您拥有两个图层：一个为员工图层，一个为作业现场图层，则两个图层都可以包含员工 ID 唯一标识符字段。在员工图层中，员工 ID 字段应包含唯一值。在作业现场图层中，员工 ID 字段可以包含重复的值，因为一位员工可能会在多个作业现场工作。即使作业现场图层中的员工 ID 字段不包含唯一值，该字段仍为唯一标识符，因为其可以将一个员工与另一个员工区分开来。

排序或分级

此字段中的值表示在排序或分级列表中要素的状态。例如，要素可以为以下内容之一：

  • 小、中或大
  • 第一、第二、第三或第四
  • 信息、警告、错误或故障

二进制

每个要素只可能具有两个值中的一个。其中的一些示例包括：

  • 开或关
  • 是或否
  • 真或假
  • 有人居住或空置

位置或地点名称

此字段中的值表示地理位置。此类字段值的示例包括街道地址名称、城市名称、区域、建筑物名称（例如 A.K. Smiley 公共图书馆）、景点名称（例如阿拉米达县展览会场或凯恩戈姆山国家公园）、邮政编码或国家/地区。

坐标

这些字段用于存储地理坐标值，例如 x、y、z、纬度或经度。

日期和时间

此字段中的值用于存储明确的日期和时间或者日期参考，例如一周中的几天、几个月或几年。

添加字段值类型

字段值类型也可以在托管要素图层项目页面的数据选项卡上每个字段的详细视图中进行定义。

请按照以下步骤定义字段值类型：

  1. 打开要添加字段描述的托管要素图层的项目页面。
  2. 单击数据选项卡，然后单击字段按钮。
  3. 单击第一个字段名称以打开字段的详细信息视图。
  4. 单击字段值类型的编辑。
  5. 使用上一部分中的定义来帮助您从编辑字段值类型下拉菜单中选择字段的值类型。
  6. 单击保存。

Map Viewer 将根据值类型信息来设置默认样式，以便更好地表示字段的内容。
