显示应用程序、数据和 web 地图

在本主题中

  1. 添加地图
  2. 添加单个应用程序或项目
  3. 创建内容图库
  4. 显示数据类别

使用以下卡片，核心团队成员可以显示一系列内容，包括仪表盘、故事地图甚至其他站点和页面：

  • 图库卡片 - 显示不同项目（包括其他站点和页面）的混合时十分有用。此卡片也是移动响应型卡片，意味着无论设备大小，内容都会正确显示。
  • 类别卡片 - 使用此卡片按类别为数据集和其他内容类型分组。您创建的每个类型都将由您选择的图标表示。
  • 地图卡片 - 在您的站点上显示web 地图或web 场景。使用行卡片设置，您可以调整其显示以跨越整个站点，或选择将其以固定宽度显示在其他内容旁边。
  • 应用程序卡片 - 显示通过 Web AppBuilder 构建的任意 web 制图应用程序。

站点和页面上显示的受欢迎的项目包括仪表盘、故事地图和 web 地图。这些项目在您的 Portal for ArcGIS 中创建并管理。当发布具有内容的卡片时，您可以从自己的内容或与您的团队、组织或公众共享的内容中选择。

要开始执行操作，请登录到 Portal for ArcGIS 并单击概览页面上的站点以查看站点列表。

提示：

在浏览器中查看站点时，也可以对其进行编辑。确保您已登录，然后在站点左侧找到编辑按钮。

添加地图

要向站点添加 web 地图或 3D 场景，请完成以下步骤。

    提示：
    通过将行卡片的布局选择为宽，可以使地图跨整个站点显示。
  1. 将地图卡片拖动到现有的行上。
  2. 将鼠标悬停在卡片上方，然后单击设置按钮切换到编辑模式。
  3. 单击查找 Web 地图。
  4. 通过搜索我的地图、我的组织地图或所有地图来查找 Web 地图。
  5. 要为地图显示标题，单击标题的切换按钮。
  6. 为您的地图设置高度值（以像素为单位）。

添加单个应用程序或项目

要显示单个应用程序，请完成以下步骤：

  1. 将图库卡片拖动到现有行中。
  2. 将鼠标悬停在卡片上方，然后单击设置按钮切换到编辑模式。
  3. 单击显示下拉菜单并选择内容显示方式（按类型或手动选择各个项目）以选择卡片上显示的内容。
    • 要基于类型显示内容，请单击动态，然后选择应用程序、仪表盘、数据集、站点、页面、Web 地图或文档。
      提示：

      应用程序中包括以下项目：应用程序、表单 (Survey123) 以及 Web 制图应用程序。支持的文档类型包括 PDF、Microsoft Excel、Microsoft PowerPoint、iWork Keynote、iWork Pages、iWork 文档 和 Visio 文档。

  4. 在显示的卡片下，选择 1。
  5. 在样式下拉菜单中，通过以下选项选择图库卡片的外观：
    • 图像 - 选择图标或缩略图。项目详细信息。
    • 拐角 - 选择直角或圆角。
    • 按钮文本 - 可在文本框中输入新名称，以对浏览按钮进行重命名。
  6. 单击保存。

创建内容图库

如果想要在站点或页面上共享 web 地图和其他应用程序，则图库卡片十分有用。您可以通过此卡片显示不同项目的混合，包括仪表盘、故事地图，甚至是其他的站点和页面。

  1. 将图库卡片拖动到现有行中。
  2. 将鼠标悬停在卡片上方，然后单击设置按钮切换到编辑模式。
  3. 单击显示下拉菜单并选择内容显示方式（按类型或手动选择各个项目）以选择卡片上显示的内容。
    • 要基于类型显示内容，请单击动态，然后选择应用程序、仪表盘、数据集、站点、页面、Web 地图或文档。要进行其他设置，您需要选择显示的卡片数量，以及在哪一群组或标签中对其进行查询。
      提示：

      应用程序中包括以下项目：应用程序、表单 (Survey123) 以及 Web 制图应用程序。支持的文档类型包括 PDF、Microsoft Excel、Microsoft PowerPoint、iWork Keynote、iWork Pages、iWork 文档 和 Visio 文档。

    • 要显示混合内容，请单击手动，然后单独选择内容，方法为单击添加内容，然后选择属于组织的应用程序、数据集、站点和页面的任意组合。
      提示：

      如果选择手动显示，则可以在自定义面板中调整卡片列表以重新排列卡片在布局中的显示顺序。单击卡片名称以将其拖动到列表中的新位置。

  4. 在样式下拉菜单中，通过以下选项选择图库卡片的外观：
    • 图像 - 选择图标或缩略图。项目详细信息。
    • 拐角 - 选择直角或圆角。
    • 按钮文本 - 可在文本框中输入新名称，以对浏览按钮进行重命名。

显示数据类别

使用类别对内容进行分组。您可以从各种图标中选择代表类别的图标，也可以提供自己的图标。

  1. 将 类别卡片拖动到现有行中。
  2. 将鼠标悬停在卡片上方，然后单击设置按钮切换到编辑模式。
  3. 编辑类别的链接文本。
  4. 选择使用标记查询或 使用群组。

    这将决定用户单击此类别时，搜索结果中将返回的数据集。

    • 标记查询 - 输入标记查询。。搜索结果中将返回在其相应企业级门户项目上具有此匹配标记的所有数据集。
    • 群组 - 选择一个群组。搜索结果中将返回该群组中的所有数据集。
  5. 从库中选择一个图标：
    1. 单击选择图标。
    2. 选择一个图标，然后单击选择。
    3. 单击图标右侧的灰色框以选择图标的颜色。
  6. 使用您自己的图标：
    1. 提供图标的 URL。
    2. 提供图标的替代文本，以帮助盲人用户并改善站点的可访问性。

    注：
    自定义图标应保存为 .svg 或透明 .png 文件。将对图标进行缩放以适应 120x120 像素。

  7. 单击保存。
