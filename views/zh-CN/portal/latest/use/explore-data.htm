浏览数据

  4. 在地图查看器中查看

数据集表格提供数据集电子表格部分的访问权限：要素（行）和其属性（列）。数据集表格中的多个功能可用于浏览和过滤数据。

要创建数据集表格，请执行以下步骤：

  1. 单击搜索结果中的数据集。
  2. 单击数据选项卡。

对数据集表格进行排序

单击列标题可以对每列或属性进行排序。文本按字母顺序排序，数字按值排序。单击一次，列按照 A–Z 或 1–10 的顺序排序。再次单击，列按照 Z–A 或 10–1 的顺序排序。

在预览地图中高亮显示要素

数据集表格的每一行都代表着地图中的一个要素。将鼠标悬停在任一行上，则会在页面顶部的预览地图中高亮显示该要素。

过滤数据

使用数据集表格，您可以通过过滤数据只显示所需的要素。随后，可以下载过滤后的数据集。您可以按地图位置或属性过滤。

按属性过滤

  1. 在数据集概况中，单击数据选项卡。
  2. 单击想要用于过滤数据集属性所在列的标题。
  3. 过滤数据。

    根据列字段类型和数据集大小，您可以选择使用滑动柱状图、滑块、输入范围、唯一值复选框或日期选择器进行过滤。

    过滤时，预览地图将反映更改。

  4. 单击另一列标题并键入下一查询可添加多个过滤器。
  5. 要移除过滤器，请单击文本框右侧的 x 图标。
注：

对于 useStandardizedQueries 设置为 false 的数据集，其中的日期字段不可过滤。

下载过滤后的数据集

数据集过滤完毕后，您可以下载此版本的数据，其仅带有过滤器中的属性。为此，请单击表上方蓝条中的下载数据选项卡并选择过滤后的数据集下方的一个选项。

在地图查看器中查看

要执行更复杂的分析和可视化，可以在地图查看器中打开数据集。

  1. 在数据集概况中，单击位于地图面板下方的创建 web 地图。
  2. 数据集将在地图查看器的新选项卡中打开。

收藏数据集

将单个数据集标记为收藏以备日后使用。您必须使用 ArcGIS 帐户登录，才能执行此操作。

  1. 在数据集页面上，单击收藏。
  2. 如果已过滤数据集，则可以选择原始或已过滤。
  3. 随即打开侧面板。按如下步骤编辑标题和描述。
  4. 单击保存或取消。
  5. 要查看您的收藏，请单击我的数据，然后单击星形图标。
