开始使用 ArcGIS Enterprise 门户

在本主题中

  1. 访问门户
  2. 熟悉门户网站
  3. 开始使用门户

ArcGIS Enterprise 门户允许您通过网站在组织内存储和共享数据、地图以及应用程序。在这个站点上，您可以找到您的组织所发布的地图和应用程序、专题地图和应用程序库、功能强大的搜索引擎、地图查看器、可加入和创建的组以及用于存储和组织内容的个人工作空间。

访问门户

要初步掌握使用 Enterprise 门户的方法，需从门户管理员处获得以下信息：

  • 门户网站的 URL

    URL 形式为 https://webadaptorhost.domain.com/webadaptorname/home/。例如，将 Web Adaptor 安装在名为 mywebserver.mynetwork.com 的服务器上的门户使用名为 arcgis 的 Web Adaptor，此门户的 URL 为 https://mywebserver.mynetwork.com/arcgis/home/。

    在受支持的浏览器中打开此 URL。Esri 建议使用最新版本的 Mozilla Firefox、Google Chrome 或 Internet Explorer 访问门户网站。

  • 门户帐户

    您的门户管理员可能已经为您创建了门户帐户，或者已将门户配置为使用您的网络登录帐户；如果没有，则您可能需要在首次连接到门户时创建一个门户帐户。请向门户管理员询问门户所用的身份验证方法。

    • 如果已经为您创建了帐户，请单击门户网站主页顶部的登录链接，使用您的帐户登录，并使用新的密码和安全问题更新您的个人资料。
    • 如果您的管理员已将门户配置为使用 Windows 登录帐户，您可能不需要登录。这取决于您的 web 浏览器及其设置；系统可能会提示您提供网络登录帐户和密码以登录到门户。
    • 如果需要在首次连接到门户时创建帐户，请单击门户网站主页顶部的登录链接，然后单击创建帐户。提供所需信息并单击创建我的帐户后，您便能够使用该帐户登录到门户。

    在您登录或进行身份验证之后，就可以开始使用此门户网站。

熟悉门户网站

管理员可能已经定制了门户网站的外观和功能，但是所有网站都具有以下功能：

  • 可供使用的地图和应用程序库
  • Map Viewer，用于创建地图并与其他人共享
  • Scene Viewer，用于在 web 场景中显示 3D 和 2D 图层并与其他成员共享
  • 为控制对门户中存储的地图、应用程序、场景和其他地理信息的访问权限而创建和管理组的方法
  • 用于和组织中其他成员共享项目的中心位置
  • 帮助定位地理信息、地图、应用程序和场景的搜索功能

可以使用门户网站中主导航栏上的功能来浏览以下位置：

图库

图库用于浏览各种专题地图、web 制图应用程序和移动应用程序。您所在组织的门户管理员负责选择要在图库中显示的项目，挑选对您的工程项目具有重要意义的内容，以及显示有用的示例，包括 ArcGIS Living Atlas of the World 中的内容。

地图

单击地图打开 Map Viewer，可使用此查看器构建交互式 Web 地图并与组织中的其他用户共享地图。您可以选择或添加底图、定义感兴趣区域以及将信息图层添加至地图。信息图层可以是在门户网站、web（如果您的站点经配置可访问此 web）或 ArcGIS Server 上找到的 ArcGIS Server 服务、KML 文档或 OGC 服务。您也可以创建要素以及从文本文件、GPX 文件和 shapefile 添加要素。您还可以更改符号并配置弹出窗口。

如果您的门户角色可以用来创建和共享内容，则可以将地图保存至个人工作空间（我的内容）并与他人共享地图。

场景

单击场景以打开 Scene Viewer。您可在此构建 web 场景。可使用 Web 场景显示 3D 和 2D 信息图层。

如果您的门户角色可以用来创建和共享内容，则可以将场景保存至个人工作空间（我的内容）并与他人共享场景。

群组

群组是项目的集合，通常与特定的感兴趣区相关。创建群组来组织和共享项目。群组所有者将决定可以查找群组的人员、其他人是否可以申请加入组以及可以贡献内容的人员。

群组页面包含以下三个选项卡：

  • 我的群组 - 此选项卡可显示您从属的群组以及您拥有的群组。
  • 专题群组 - 门户管理员希望高亮显示或者在组织内推广的群组将显示在此选项卡上。
  • 我所在组织的群组 - 您可以访问的组织中的所有群组将显示在此选项卡上。这意味着您将看到您拥有的所有群组、共享给组织的群组以及共享给公众的群组。

如果您的门户角色授予您创建群组的权限，您将在我的群组和我所在组织的群组选项卡上看到用于创建群组的选项。

内容

内容页面可用于查看门户中您具有访问权限的所有内容，还允许您打开单个项目的详细信息。可以通过以下其中一个选项卡来访问门户中的所有可用内容：

  • 我的内容 - 此选项卡可显示门户中您拥有的所有项目。您可以在文件夹中组织这些项目、将项目添加到门户，以及将您拥有的项目标记为收藏以方便查找。

    如果您的门户角色不允许您创建内容，则您无法看到此选项卡。

  • 我的收藏夹 - 已添加到收藏夹列表的所有项目将显示在此选项卡上。
  • 我的群组 - 适用于您所属群组的项目将显示在此选项卡上。
  • 我的组织 - 您可以访问的门户组织中的所有项目将显示在此选项卡上。这意味着您拥有的所有项目、与公众共享的所有项目以及与组织共享的所有项目均可显示。
  • Living Atlas - 默认情况下，ArcGIS Enterprise 门户可通过 ArcGIS Online 访问由 Esri 精选的 ArcGIS Living Atlas of the World 内容。您可通过 Living Atlas 选项卡来访问此内容。

    如果门户管理员不允许您访问此项内容，您将无法看到此选项卡。

可以通过选择列表、表格或格网视图来更改项目在内容的不同选项卡上的显示方式。还可以按项目类型、创建日期、最后修改日期或共享级别来过滤每个选项卡上的内容，以帮助找到所需内容。可以通过各选项卡来访问项目的详细信息页面。

组织

网站的组织页面主要供门户管理员管理门户网站和成员。如果您不是管理员，则无法使用此页面。如果可用，则组织允许您查看每个门户成员拥有的项目和群组，并查看每个成员的个人资料。

搜索

此站点具有强大的搜索功能，使您能够快速找出工作中所需的特定地图或应用程序。搜索结果可帮助您确定各项目的实用性并找到相关项目。它还使您能够在任何位置打开项目，例如，在 Map Viewer、ArcGIS Pro 和 ArcGIS Desktop 中打开地图。
