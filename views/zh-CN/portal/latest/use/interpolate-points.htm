插值点

“插值点”工具用于根据一组点的测量结果来预测新位置上的值。

注：

需要 ArcGIS Geostatistical Analyst 10.8 for ArcGIS Server 才能运行“插值点”工具。ArcGIS Geostatistical Analyst 10.8 for ArcGIS Server 包括在 ArcGIS Server Advanced 许可内。

工作流示意图

示例

在指定气象站测量降雨量。“插值点”可用于创建整个地区范围内已预测降雨量的连续图层。

用法说明

点图层将用作输入。输入图层必须具有数值字段才能作为插值的基础。“插值点”用于与温度和污染级别等随地形变化平缓的数据配合使用， 而不适用于人口或中位收入等在很短间隔范围内变化十分明显的数据。

可通过设置“插值点”工具来优化速度/精度或中间区域。预测越精确，计算结果的速度越慢，反之亦然。

可通过选中输出预测误差复选框来创建标准误差图层。插值图层中 95% 的置信区间可通过计算出插值、然后加上两个标准上限误差并减去两个标准下限误差进行计算。

如果选中使用当前地图范围，则仅插值当前地图范围中输入点图层中的要素。如果未选中，则将插值输入点图层中的所有要素，包括超出当前地图范围的要素。

限制

“插值点”工具仅可用于点要素。

类似工具

使用“插值点”根据点测量预测值。其他工具或许在解决略有差异的类似问题时十分有用。

Map Viewer 分析工具

如果要使用点测量或线测量来创建密度地图，请使用计算密度工具。

ArcGIS Pro 分析工具

“插值点”使用经验贝叶斯克里金法来执行插值（插值可在Geostatistical Analyst 扩展模块工具箱的插值工具集中获得）。

插值点同样适用于 ArcGIS Pro。要从 ArcGIS Pro 运行工具，则工程的活动门户必须运行 Portal for ArcGIS 10.5 或更高版本。您还必须使用相应帐户登录到门户，该帐户应具有在门户中执行标准要素分析的权限。
