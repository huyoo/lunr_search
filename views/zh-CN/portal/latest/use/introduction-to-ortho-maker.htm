Ortho Maker 简介

在本主题中

  1. 创建精确的正射影像地图
  2. 三步设置
  3. 汇总

Ortho Maker 是一款 ArcGIS Enterprise 中的 web 应用程序，用于通过无人机影像创建正射影像镶嵌和 terrain 表面产品。这些映射产品需使用指导工作流创建，可引导您完成工程的三个步骤：

  1. 工程设置
  2. 局域网平差
  3. 产品生成

Ortho Maker 由 Enterprise 和 ArcGIS Image Server 提供支持。它可使用分布式处理以及 Enterprise 和 ArcGIS Image Server 的存储功能快速处理由无人机采集的数百或数千个影像，并创建无缝的正射影像镶嵌、数字表面模型 (DSM) 和数字地形模型 (DTM)。其他增值产品包括可见光大气阻抗指数 (VARI) 地图，用于分析植被的覆盖和相对健康状况；还包括坡度和坡向表面地图，用于针对地形对发展和自然要素的影响进行适宜性分析或评估。您可以选择创建基于影像波段运算符的自定义产品，以创建指数（如城镇指数和含水指数）和其他类型的影像分析。

Ortho Maker Web 应用程序任务需以工程的形式进行管理，该工程将管理和追踪工程中定义的正射影像镶嵌组件。该工程中包括所有工程输入、辅助数据（如影像轮廓线和局域网平差报告）、中间产品（如影像集合）、快速局域网平差结果、最终产品以及每个处理阶段的状态。可以创建新工程，也可使用和编辑现有工程模板以满足特定作业的要求。

Ortho Maker 优势

您可通过 Ortho Maker 轻松上传和处理无人机影像。如果您是无人机处理的新手，则可通过默认的 Ortho Maker 设置轻松创建精确的产品，而无需对摄影测量参数有深层次的了解。如果您是高级无人机用户，则可通过 Ortho Maker 微调处理参数以满足您的需求。Ortho Maker 在完成无人机影像处理工程方面具有许多操作优势：

  • 可上传一个无人机影像集合并对其进行处理，以创建精确的正射影像和数字高程模型。
  • 轻松创建 VARI、坡度、坡向和山体阴影增值产品。
  • 使用 Enterprise ArcGIS Enterprise 部署实现强大的分布式处理和存储。
  • 指导工作流使工程设置和生产处理变得简单。
  • 查看您的无人机影像集合并删除不需要的影像。
  • 在几乎没有摄影测量经验的情况下处理您的无人机影像。
  • 上传和管理地面控制点以进一步提高处理结果的精度。
  • 如果您有处理无人机影像的经验，可微调各项处理参数。
  • 查看和共享详细的处理报告。
  • 共享您的结果，以便与其他支持 ArcGIS 平台的软件（如 Story Maps、ArcGIS Pro 和 ArcGIS Image Server 分布式分析搭配使用。

创建精确的正射影像地图

正射影像地图根据一系列俯视影像构建而成，这些影像在名为正射校正的过程中进行了校正，从而消除了传感器和地形变形。一旦每个影像的位置都被校正到已知精度，则这些影像就将镶嵌在一起。生成的正射影像地图镶嵌可被用作底图，以收集和测量要素，以及使用标准影像处理技术分析和获取信息。

正射校正采用摄影测量技术，通过名为立体影像的重叠影像来获取摄影测量模型和所需的表面模型，并将其用于校正影像中的系统和地形变形。使用重叠区域中的连接点和整个工程区域的地面控制点来调整一块或一组重叠影像。重叠立体影像之间的连接点通过自动计算得出，而地面控制点通常来自地面测量，有时也从现有的正射影像中获得。有关正射校正过程的更多详细信息，请参阅正射映射简介。

Ortho Maker 采用严格的摄影测量流程，以简化的指导工作流进行工程设置和管理、局域网平差以及正射映射产品的处理。该工程管理和复杂的处理是在幕后执行的，执行过程中使用了通过 ArcGIS Image Server 进行分布式处理和存储的 ArcGIS Enterprise 部署，其中包括根据您的具体配置采取的云端部署。Ortho Maker 工程管理器会组织所有输入，验证这些输入是否符合最低的纳入和使用标准，协助您根据该标准进行所需的局域网平差，并生成您在工程设置中指定的产品。随后生成校正解决方案和精度报告以进行验证。

产品会自动作为服务发布，以便与组织中的利益相关方共享。产品可在门户的 Map Viewer 中打开，其中 Map Viewer 中可用的工具和功能均可用于进一步分析。

Ortho Maker 产品可在 ArcGIS Pro、ArcGIS Image Server 分布式分析和 ArcGIS Online 中使用。例如，您可以根据 Ortho Maker 中生成的 DSM 和彩色红外正射镶嵌服务构建和测试计算密集型水文建模处理链以及 ArcGIS Pro 中的分割和分类链，然后使用您的 ArcGIS Image Server 分布式分析部署在云中的多个实例之间分配处理。

Ortho Maker 是 ArcGIS 生态系统中一款有价值的应用程序，可充分利用到 ArcGIS 平台不同部分之间的无缝集成。其优势包括可扩展的分布式处理和存储、直观的简化工作流，以及与 ArcGIS 平台的紧密集成，这些使得互操作性能够充分满足您的工程需求。

三步设置

Ortho Maker 指导工作流可通过三个步骤使工程设置和产品处理变得简单高效。

设置您的工程

Ortho Maker 工作流的第一步是设置您的工程。在这一步中，您将识别包含无人机影像和相关元数据文件的文件夹，然后对其进行验证。随后您将填写工程设置表单，在其中为工程分配名称和描述并选择处理工作流。

处理工作流会生成影像集合的覆盖范围地图，以验证您是否拥有适当的工程区域覆盖范围。您可以选择执行快速校正，以估计最终产品的精度是否能达到或超过您的工程精度要求。随后执行完整的局域网平差，验证最终精度，并生成任一或全部可能的产品。

您可通过工程设置页面定义设置，以执行快速或全面校正以及产品生成参数。通常，默认设置足以为大多数工程和应用程序生成高质量的产品。如果您是经验丰富的用户或您具有特殊工程要求，则可定义各种设置以生成您的正射镶嵌产品，如生成正射影像之间的接缝线、色彩平衡参数以及以米为单位定义输出像素大小 (GSD)。对于表面产品，您可以定义点云匹配方法以生成裸露地表、检测城区的地面要素、检测您表面中建筑物或树木的最大尺寸，并定义地面间距和像素尺寸。

您可以保存工程设置以创建工程模板。可以导入、编辑工程模板，并将其用于当前工程或其他工程。

计算局域网平差

校正步骤包括两个选项，运行快速校正或运行完整校正。快速校正会以较低的分辨率对影像执行局域网平差计算，以达到初始 QA 的目的。

计算初始局域网平差，以生成重叠影像之间的连接点，并且可以生成校正报告。一旦初始局域网平差计算完成，则可以通过添加地面控制点 (GCP) 来对其进行优化。GCP 可通过地面测量获取或从文件中导入。如果您的要素服务包含 GCP（以及可作为附件的 GCP 照片），则可以从要素服务中将其导入。还可从精度已知的地图或现有正射影像底图中添加 GCP。

如果您选择从底图添加 GCP，则控制点集合界面将允许您选择底图上的点，以及覆盖底图上点的任意数量影像上的相应点。可将采集到的任何控制点作为 GCP 进行分配，用于计算局域网平差，或可将其作为检查点，用于评估正射影像的精度。

一旦您在影像上加载并识别了 GCP，则系统将重新计算局域网平差。修订后的局域网平差运行速度很快，原因是使用了初始校正中得出的连接点，从而提高了效率。

生成产品

工作流中的第三步是使用 Ortho Maker 生成摄影测量产品。这些产品包括正射影像镶嵌以及 DSM 和 DTM 表面产品，并附有精度评估报告。可对这些产品的相关处理参数进行编辑或校正，使其符合您的工程需求，并可使用更新后的设置重新生成每个产品。

  • 可以生成其他衍生产品，包括来自正射影像、坡度、坡向和山体阴影图的 NDVI，以及来自地表产品的服务图层。
  • 作为产品生成过程的一部分，产品会自动发布到您的 ArcGIS Enterprise 栅格数据存储中，随后系统将根据其具体的授权和权限情况将其与组织成员进行共享。
  • 可在针对栅格分析、ArcGIS Pro 和 Map Viewer 客户端进行配置的 Enterprise 中进一步分析和处理产品。

汇总

Ortho Maker 是一款简化的应用程序，用于在生产环境中处理无人机影像。Ortho Maker 可引导您完成工程设置、局域网平差和产品生成，以生成正射影像镶嵌以及 DSM 和 DTM 产品。所生成的产品将自动发布并与您的组织成员共享，且适合在 ArcGIS Pro 和 ArcGIS Online 中进行进一步处理与分析。
