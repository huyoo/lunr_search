项目详细信息

在本主题中

  1. 查看和编辑概述信息
  2. 使用属性数据
  3. 可视化图层
  4. 编辑设置

门户中的每个项目都包括一个项目页面，该页面中具有各种信息、操作、选项和设置，并以选项卡的形式进行组织：概述、数据、可视化和设置。您所看到的选项卡以及选项卡上可用的选项和信息均取决于项目类型、您的权限以及您是否为项目所有者或管理员。

您可通过项目页面以多种方式深入浏览项目。例如，交互属性表集成了相关的记录和附件，您可以使用它来浏览属性数据并同时查看字段的各个方面，包括字段值、汇总统计数据和设置。内容创建者和管理员无需转到站点上的其他地点即可更改项目的设置。例如，您可以在概述选项卡中从场景配置应用程序，或者在设置选项卡中直接编辑应用程序设置。除了在 Map Viewer 中打开图层之外，还可以在可视化选项卡中配置要素图层样式、弹出窗口及其他图层属性。

要访问项目页面，请单击项目缩略图下的详细信息，或者在图库中将鼠标悬停在缩略图上并单击详细信息。如果您是项目所有者，可通过内容页面中我的内容选项卡访问项目页面。

查看和编辑概述信息

概述选项卡包括项目的概述信息，如描述、标签、数据源信息、创建日期、大小、共享状态以及项目是否已指定为权威或弃用。还包括状态栏和交互式一览表，来帮助项目所有者和管理员提供更完整的项目信息。 概述选项卡中的大部分信息可由项目所有者、管理员及其他具有相应权限的人员进行编辑。其他人可以使用概述信息来帮助其在使用项目前更好地了解项目。

概述选项卡还包括用于打开、添加到收藏夹、对项目进行分类、共享项目、编辑元数据和添加评级和评论的选项。某些类型的项目页面具有其他选项，例如，您可以从可配置应用程序的项目页面中发布应用程序、更新或下载诸如逗号分隔值 (CSV) 等文件、从 Web 地图或 Web 场景创建应用程序以及从启用导出的托管要素图层中导出数据。如您具备相应的权限，则可将地图和数据发布为托管 Web 图层。管理员以及具备相应权限的人员也可更改门户中的项目所有者。

打开项目

默认情况下，项目将在最适当的应用程序中打开。例如，地图在 Map Viewer 中打开，场景在 Scene Viewer 中打开，而图层包则在 ArcGIS Pro 中打开。有时，您还可以选择其他选项。例如，可以选择在 ArcGIS Pro 中而非 Map Viewer 中打开地图。有些项目只能下载，而有些项目未提供打开或下载选项。在项目页面概述选项卡右侧，您所能看见的操作按钮取决于项目类型。用于打开、下载等操作的最适用选项将作为第一个操作按钮提供。

有关受支持项目类型的完整列表，请参阅 可以将哪些内容添加到您的门户中？

标题

标题是用户访问项目时看到的名称，也是在内容页面上我的内容选项卡上显示的名称。该标题应代表项目所包含的内容。

汇总

用户在此门户网站中搜索内容时，结果列表中将返回此简短描述。

描述

用户打开项目页面时将看到此项目的详细描述。

缩略图

将项目添加到站点时，默认情况下会创建缩略图。缩略图将显示在图库、搜索结果、内容和项目页面上。如果默认缩略图不能传达要表示的信息，可以创建并加载其他缩略图。单击编辑缩略图以使用您的项目的自定义缩略图。

可拖动一个图像或浏览到文件。为获得最佳效果，请添加不小于 600 像素（宽）x 400 像素（高）或纵横比为 3:2 的 web 文件格式（如 PNG、JPEG 或 GIF）的图像。平移和缩放您希望在缩略图中显示的内容。根据图像文件的大小、分辨率以及自定义缩略图时的缩放程度，在保存该图像时可能需要重新采样和缩放。如果您添加 GIF 或 JPEG 格式的图像，则该图像在保存时将转换至 PNG 格式。

对于一些项目类型，可以通过选择底图并设置地图范围和缩放级别来创建自定义缩略图，而非上传图像。对于此选项，请单击根据地图创建缩略图。

使用条款

本部分提供有关如何使用此项目以及可使用此项目的用户的信息。例如，某个地图图层可能存在使用限制，要求其他人在包含该图层的屏幕截图上显示版权信息。

元数据

如果门户管理员已启用元数据，则元数据按钮随即会出现在项目页面的概述选项卡中。最初，只有项目所有者和管理员才能看到该按钮。如果所有者或管理员使用元数据按钮来访问内置编辑器并将其他基于标准的元数据包括在内，则任何有权访问该项目的用户也都可以看到元数据按钮，并可以使用它来查看为门户配置的元数据样式中的元数据。

此外，对于所有包含关联元数据的托管要素图层，还会在图层列表中显示元数据链接。通过该链接，任何有权访问托管要素图层项目页面的用户都可以在为门户配置的元数据样式中查看单个图层的元数据。同样，如果某表随元数据一同发布，您将看到表列表中的元数据链接。

如果门户管理员禁用元数据，元数据仍然是该项目和图层的一部分，但您将无法继续编辑门户中的项目级别元数据。

项目信息

通过提供完整的项目信息，您可以帮助他人查找、了解和使用您的项目。该项目信息部分帮助项目所有者和管理员标识缺失或不完整的项目信息，并提供改进的建议，例如添加更多标签或摘要。单击最佳改进建议直接转到要做出最大影响的部分，或单击了解更多通过完整的交互式建议改进清单来完成您的任务。当您进行改进时，例如添加使用条款或增加描述长度时，清单会跟踪您更新的部分以及状态栏更改以反映您的进度。

更改所有者

如果您是门户管理员，或者您是具有项目所有权更改权限角色的成员，则可以更改门户中项目的所有者。

文件夹位置

您可以将拥有的项目移动到我的内容中的其他文件夹。这使您可以组织项目，使其更容易被找到。它还可以帮助您识别相关项目。例如，您可以创建一个文件夹来存储相关项目。将项目（例如源文件和您从中创建的图层），一组包含这些图层的 Web 地图和场景以及所创建的包含这些 Web 地图的任何应用程序放在同一文件夹中。这样，您就知道该文件夹中的所有项目都是相关且相互依赖的。或者，您可以创建文件夹来存储为特定工程创建的项目，并将相关项目移动到每个工程文件夹。

如果您提前计划，则可以单击我的内容中的创建新文件夹按钮创建所需的文件夹。然后，在添加项目、发布图层或创建地图或应用程序时，可以选择相应的文件夹来存储项目。但是，有时无法在创建项目之前创建所需的所有文件夹。在上述情况下，可以按照以下步骤将您拥有的项目从一个文件夹移至另一个文件夹：

  1. 在我的内容中打开第一个文件夹，该文件夹包含要移到新文件夹的项目。
  2. 选中要移动的每个项目旁的方框并单击移动。
  3. 第一次执行此操作时，可能需要创建目标文件夹。如果是这样，键入可以表示新文件夹内容的名称，然后单击创建。在我的内容中创建一个新文件夹，您选择的项目将移至其中。
  4. 在我的内容中打开下一个文件夹，该文件夹包含您要移动的项目。选中每个项目旁边的框，然后单击移动。
  5. 如果目标文件夹已存在，则可以在列表中将其选中，然后单击移动。
  6. 重复步骤 4 和 5，直到将所需的所有项目组织到相同文件夹中为止。

类别

其他人可借助类别在门户中更轻松地查找内容。如果门户管理员已配置类别，则最多可以为项目分配 20 个类别。要分配或更改项目的类别，请单击类别旁边的编辑，然后选择要分配的类别。也可以使用过滤器类别框以缩小类别范围。完成后，单击保存。

标签

标签可帮助人们在搜索时找到您的项目。指定的标签应与您认为用户搜索项目时所用的关键字相对应。搜索或浏览项目时，组织成员和组织外的用户也可以使用这些标签来过滤内容。群组成员可以使用标签过滤器来查找与他们所属群组共享的项目。

制作者名单（属性）

您的项目可能包含来自不同机构和个人的数据。在此可以提供数据提供者的名单。

URL

您可以复制并粘贴应用程序或 ArcGIS Server 图层的 URL，并可在其他位置使用它们。例如，开发人员可能想要在其构建的应用程序中包含托管 Web 图层的 URL。

添加评级和评论

每个项目页面中都包括一个评级。评级是基于五颗星的加权平均值，一颗星代表项目较差，五颗星代表项目非常好。评级值提供了一个考虑项目获得的评级次数和平均评级得分的加权平均值。因此，具有两次评级且每次评级都是四星的项目在评级方面要高于一个只具有一次评级且评级为四星的项目。要对项目进行评级，您必须已登录且仅能对于非自己拥有的项目进行评级。每个项目仅能添加一个评级。尽管您可以更改评级，但计数将保持不变。要对项目进行评级，单击概述选项卡右侧详细信息区域的相应星形。

项目页面也可能包括用于评论的部分，具体取决于门户管理员是否选择启用评论。 评论将显示在概述选项卡的底部，可以由项目所有者或其他登录用户添加。发布评论后，您可以将其删除，但无法对其进行编辑。登录用户也可以回复评论。 项目所有者可删除对他们所拥有的项目发布的评论。 评论可以按从最新到最早进行排序（新），或者按最近回复（最新回复）。您可以订阅与所有人共享（公开）的项目评论的 RSS 源。您不能订阅仅与您门户共享（或私有）的项目评论。

添加到收藏夹

您可以根据已添加项目、搜索结果中的项目和门户中内容页面及库中的项目创建收藏夹列表。 当您通过搜索图层来添加地图时，可在内容页面我的收藏夹选项卡中查看您的列表。

要将项目标记为收藏，请单击概述选项卡中项目缩略图下方的添加到收藏夹。

要将收藏的项目从列表中移除，单击项目缩略图下方的从收藏夹中移除。

共享项目

根据您的权限，您可通过项目页面上的共享按钮直接共享项目。

通过 Facebook 或 Twitter 共享项目页面

每个项目页面都包括 Facebook 和 Twitter 按钮，可帮助您通过 Facebook 和 Twitter 帐户快速共享内容。（如果未显示这些链接，可能是您的门户管理员未启用这些链接，或者您不具有与公众共享的权限。） 可在门户网站中共享所有项目的项目页面，包括其他用户所拥有的项目页面。最好仅共享那些您已确定公开共享的项目页面；否则，某些用户可能无法访问这些链接。使用项目页面上的 Facebook 和 Twitter 链接时，您共享的是项目详细信息而非项目。要共享地图（而不是其项目页面），可使用地图上 的共享。

通过短 URL 共享项目页面

如果要通过提供页面链接与其他人共享项目页面，请使用详细信息部分中的短 URL 按钮。（如果未显示此链接，可能是您的门户管理员未启用此链接，或者您不具有与公众共享的权限。） 单击短 URL 按钮生成项目页面的短 URL，您可以将其复制粘贴到电子邮件、短信和其他通信渠道。您可共享网站中所有项目的项目页面短 URL，包括其他用户所拥有的短 URL。但是，在共享短 URL 之前，请确保该项目已经与链接的收件人共享；否则，有些人可能无法访问该 URL。例如，如果您向组织内外的个人提供短 URL，但该项目只与组织共享，组织之外的人将无法访问该项目。

更新

使用更新以计算机中的某一更新版本的文件替换某个文件。文件名必须匹配。更新项目时，将保留下载计数。通过覆盖更新托管要素图层。

使用属性数据

通过数据选项卡，您可以使用要素图层的属性表和表中的数据。通过以下任一选项，您可以在数据选项卡上查看属性表:

  • 表 - 通过此选项查看属性表时，您会看到表中的所有字段（列）和行、相关表中的字段和行以及所有属性数据。当您想要查看、排序、编辑或删除存储在表中的属性数据时，以及想要查看和添加附件（例如，相关照片或其他文件）时，此表视图最佳。图层所有者、门户管理员、数据管理者和所共享访问群组的成员可以编辑或删除属性数据。

    您也可以了解各个字段的详细信息 - 例如，查看字段是否为必填字段或查看其默认值 - 方法是单击您感兴趣字段的列标题，然后单击显示详细视图。表将缩小至您所选定字段的更具针对性的视图大小，该视图具有摘要部分、带有字段属性规则（如必填字段、默认值）的设置部分以及带有字段名称和类型信息的详细信息部分。在每个部分所看到的信息，取决于字段类型和您具备的权限。例如，在摘要部分，您将看到数值字段的直方图或文本字段的前 10 个唯一值的列表以及相关联的计数信息。

    如想选择要浏览的其他字段而无需返回到完整表，请单击左侧的选项按钮 并选择其他字段。完成对字段详细信息的查看后，您可通过单击关闭 (x) 按钮返回到完整表视图。

  • 字段 - 对于需要查看或操作属性表方案的图层所有者或门户管理员，此表视图更为有用。

    从字段页面查看表时，您会看到属性表中所有字段的列表，包括隐藏字段和任何相关表中的字段。通过单击排序依据按钮，您可以过滤特定的数据类型并在字段页面上更改字段的列表顺序。有关特定字段的详细信息，请单击字段的显示名称设置。

    图层所有者或门户管理员可以在字段页面上的托管要素图层中向图层添加或从中删除字段。所有者或管理员也可以为字段配置显示名称。默认情况下，显示名称与字段名称相同，但图层所有者或管理员可以更改该名称，以便查看属性表的用户更容易阅读和理解字段名。例如，对于名为 AVG_POP 的字段，显示名称可以设置为 Average Population。

注：

对于包含多个图层的要素图层，您可通过先在图层下拉菜单中选择图层来使用各个图层的属性和字段。

查看和编辑属性

可以在属性表中查看要素图层项目的属性数据。如果图层是托管要素图层，则图层所有者和门户管理员可以删除或编辑这些属性。如果托管要素图层可编辑，则数据管理者和共享的访问群组的成员可以编辑或删除属性。执行以下任意操作以查看或编辑属性：

  • 为了便于查看属性数据，可通过单击列标题并选择升序排列或降序排列来对属性进行排序。
    注：

    一些较早版本的 ArcGIS Server 可能不支持排序。

  • 要隐藏或显示属性字段，在表的右上角单击显示/隐藏列按钮。选中复选框可显示属性列。取消选中复选框以将其隐藏。
  • 要在表中编辑属性，选择单元格中的值，输入新值（或者如要将其删除，则移除该值）并按 Enter 键以保存更改。

查看和编辑相关记录、照片和文件。

如果图层具有相关记录，则具备相应权限的用户也可查看这些属性。如果图层是托管要素图层，则也可以编辑相关记录。可以看到，这些相关记录在表的右侧以列的形式显示，且具有斜体标题。

您可以查看属性的任意相关照片或其他文件，如具有编辑图层的权限并且在图层上启用附件，则也可添加其他照片和文件。

执行以下任意操作以查看和编辑相关记录、照片或文件：

  • 要查看相关记录，请在您要查看的记录对应的相关表列的单元格中单击显示。例如，如果宗地图层具有一个所有者的相关表，且您想要查看宗地的全部所有者，则在所有者列的相应单元格中单击显示以查看选定宗地的所有者列表。由此，您可以浏览表中的其他记录 - 例如，选择任意宗地 ID 以查看该宗地的所有者。如果相关表具有与其自身相关的记录，您也可以对其进行检查 - 例如，如果所有者表具有由宗地所有者拥有的其他属性的相关表，可单击显示以查看那些属性记录。
  • 要在相关表中编辑属性，选择单元格中的值，输入新值（或者如要将其删除，则移除该值）并按 Enter 键以保存更改。仅具备相应权限的用户可以在相关表中编辑属性。
  • 查看相关表时，可随时单击关闭 (X) 按钮来返回到之前的表格。
  • 如果图层具有相关照片或文件，可在表右侧的照片和文件的相应单元格中单击显示来对其进行查看。要添加附件，可单击添加并上传您想要附加的照片或文件。
  • 要隐藏或显示相关表或照片和文件列，在表的右上角单击显示/隐藏列按钮。选中复选框以显示列。取消选中复选框以将其隐藏。

查看字段信息

您可以查看表中任意字段的详细信息，包括其名称、别名、类型和字段设置，例如，字段是否为可编辑的必填字段以及字段长度等。

要查看有关字段的更多信息，单击您感兴趣字段的列标题，然后单击显示详细视图。表将缩小至您所选定字段的更具针对性的视图，该视图具有摘要部分、带有字段属性规则（如必填字段、默认值等）的设置部分以及带有字段名称和类型信息的详细信息部分。在每个部分所看到的信息，取决于字段类型和您具备的权限。例如，在摘要部分，您将看到数值字段的直方图或文本字段的前 10 个唯一值的列表以及相关联的计数信息。

如果您是图层的所有者或门户管理员，并且图层是托管要素图层，则可以从详细视图中删除字段或计算字段值。

如想选择要浏览的其他字段而无需返回到完整表，请单击字段列左侧的向下箭头并选择其他字段。完成对字段详细信息的查看后，您可通过单击关闭返回到完整表视图。

可视化图层

通过可视化选项卡，您可以对要素图层的默认属性进行更改，而无需在 Map Viewer 中打开图层。您可更改图层样式、应用过滤器、配置弹出窗口或标注图层中的要素。也可将图层隐藏于图例中或更改刷新间隔。与在 Map Viewer 中相同，可视化选项卡提供了用于浏览图层、访问底图库的导航工具以及查找地址或地点的搜索框。

如果您是项目所有者，则可保存对图层所做的所有更改。项目所有者和具有内容创建权限的用户也可选择将图层的副本保存为我的内容中的新项目。

要通过可视化选项卡使用要素图层，请执行以下任意操作：

注：

如果要素图层包含多个图层，首先从图层下拉菜单中选择要使用的图层。

  • 单击更改样式按钮 可对图层执行更改默认样式操作。
  • 单击过滤器按钮 可对图层应用过滤器。
  • 单击配置弹出窗口按钮 可为图层配置弹出窗口。
  • 单击创建标注按钮 可在图层中为要素创建标注。
  • 单击更多选项按钮 并单击缩放至来显示图层中的所有要素。
  • 对于启用了编辑追踪的要素图层，单击更多选项按钮 并单击过滤编辑内容以过滤地图上可见的编辑内容。
  • 单击更多选项按钮 并单击在图例中隐藏以在图例中隐藏图层。
  • 单击更多选项按钮 并单击设置图层刷新间隔以设置图层的刷新间隔。
  • 单击地图上的搜索按钮并输入地址或地名，以查找特定位置或要素。
  • 平移地图或使用导航工具来缩放或者转至默认范围。
  • 单击地图上的底图库按钮 以选择其他底图。
  • 单击保存图层来保存对图层的更改。此按钮对项目所有者、管理员或共享更新群组成员可用。
  • 如您具备创建内容的权限，则可单击另存为新图层以在我的内容中将图层的副本保存为新项目。在另存为新图层窗口中，添加标题、标签和摘要并根据需要选择存储该项目的文件夹，然后单击保存。如果要素图层具有多个图层，您可以选择保存所有图层或仅当前图层。

编辑设置

根据您的权限和项目类型，您可使用设置选项卡配置项目的设置，例如将代码附加到应用程序、更改范围，并允许他人保存项目的副本。 托管要素图层包括关于哪些用户能够对您的图层进行编辑和导出操作的其他设置。所有者也可在设置选项卡中将其项目删除。

常规设置

可配置以下常规设置。

内容状态

项目可以指定为权威或弃用。将项目指定为权威或已弃用可使其他人更容易找到权威项目，同时阻止使用过时的项目。

门户管理员及具有更新内容的管理权限的人员可以使用标记为权威按钮将项目指定为权威。指定为权威的项目使用概述选项卡上的权威徽标标识到门户成员。

如果您是项目所有者或管理员，或者您具有更新内容的管理权限，您可以使用标记为弃用按钮将项目指定为弃用。标记为弃用的项目使用弃用徽标标识到所有用户。

当门户成员在门户中浏览或搜索内容时，可以使用状态过滤器查找标记为权威或弃用的内容。在 Map Viewer 中向地图添加图层时，门户成员可以将搜索限制为标记为权威的图层。

删除保护

在不希望项目被意外删除时选中此选项，例如，当项目是用于图库中专题地图的图层时选中此选项。

删除项目

要从站点删除项目，请单击删除。要避免意外删除属于门户专题内容的项目（如专题地图中的图层），可启用删除保护作为项目设置。

如果您删除具有关联的托管 WFS 图层的托管要素图层，则也将删除该托管 WFS 图层。

范围

范围是您为项目指定的地理边界。您可以设置范围，使项目可以按位置查找。对于地图和图层项目，范围设置还将确定打开项目时默认显示的位置。

要设置或更改您所拥有或管理的项目的范围，请执行以下操作：

  1. 单击设置范围（第一次设置项目范围）或编辑范围（编辑项目范围）。
  2. 在设置范围窗口中，执行以下任意操作来定义项目的范围：
    注：

    您可以根据需要使用缩放按钮来帮助您导航到所需范围。

    • 要基于为组织指定的默认范围创建范围，请单击组织的范围。
    • 要根据数据源中定义的范围创建范围（仅适用于托管要素图层和 ArcGIS Server 要素图层项目），请单击图层中的当前要素。
    • 要导航至地点或地址，请在搜索框中输入关键字。关键字可包括地址、街道交叉口、地点、感兴趣点和经纬度坐标。范围以您指定的位置或地址为中心。
    • 要绘制表示范围的形状，请单击绘图工具，然后在地图上拖出一个框。
    • 要通过为范围边界指定坐标值（以十进制度为单位）来创建范围，请单击坐标并在顶部、底部、左侧和右侧框中输入坐标。
  3. 根据需要更改范围，方法是拖动范围框的任意折点，或单击框然后将其拖动到新位置。
  4. 单击保存。

Web 地图设置

下列设置可用于 Web 地图：

离线

如果您的地图支持离线使用，即如果地图中的所有图层均在保存地图时启用了离线使用，则离线部分将显示在设置选项卡上，且在默认情况下打开启用离线模式切换键。如果启用了离线模式，则您可以选择通过单击高级选项来更改默认高级离线选项。要禁用地图的离线使用功能，请单击启用离线模式切换键将其关闭。

对于将离线使用的地图，建议您提前创建地图区域来打包地图中特定区域的数据。地图区域创建加快并简化外业工作人员的地图下载过程。在地图区域下，单击管理区域来创建地图区域或管理现有地图区域。

有关离线使用地图以及创建和管理地图区域的详细信息，请参阅离线使用 web 地图。

另存为

选中此选项将允许其他人保存地图副本。如果您希望地图按照您配置的方式显示并且不希望他人轻易复制或保存地图，请取消选中此选项。

应用程序设置

设置地图选项，这些选项用于路径查找、测量、查找位置以及 ArcGIS 应用程序中的其他工具和功能。

图层设置

使用本节中的实用程序根据需要批量更新地图中图层的 URL。

  • 将图层更新至 HTTPS - 作为 web 地图的所有者或管理员，您可以更新 web 地图中的所有图层以使用 HTTPS。建议使用 HTTPS 访问所有图层，以在通过 Internet 传输时加密图层中的信息。要在访问图层时使用 HTTPS，请单击将图层更新至 HTTPS 并单击更新图层以确认更改。 地图中的各个图层都会受到检查以确定是否需要被更新至 HTTPS。如果发现有任何图层使用 HTTP，则门户将尝试使用 HTTPS 向同一图层发出请求，然后更新您拥有或管理的地图及所有关联图层项目。 如果图层不支持 HTTPS，系统会告知您这一事实，并且图层不会在地图中更新。如果报告图层不支持 HTTPS，请联系图层的所有者，以将图层配置为支持 HTTPS 或提供替代资源。
    注：

    当您尝试更新地图中的图层以使用 HTTPS 时，如果自定义符号源支持 HTTPS，则使用 HTTP 的任何自定义符号都将自动更新以使用 HTTPS。而其他图层自定义，例如嵌入到弹出窗口中的图像，则不会更新到 HTTPS。要更新这些参考以使用 HTTPS，请打开 Map Viewer 中的图层（或地图），并更新其中的参考。

    注：

    对于使用诸如 http://gisserver.domain.com:6080/arcgis/rest/services/exampleservice/MapServer 等内部端口 URL 添加到 Enterprise 门户和/或 web 地图的图层，不支持更新图层至 HTTPS 的功能。

  • 更新 ArcGIS Server 站点参考 - 如果要将 ArcGIS Server 图层从过渡站点移动到生产站点，或将其迁移至新的域，请单击更新参考更新图层 URL 以参考新站点。您的门户将检查地图中的图层并显示可供更新的 ArcGIS Server 站点。使用所选参考的图层将更新为使用指定为替换的新站点。参考站点的所有 ArcGIS Server 图层都将更新，同时，您拥有或管理的所有关联图层项目也将更新。托管图层和来自联合 ArcGIS Server 站点的图层不支持更新 ArcGIS Server 站点参考的功能。
    注：

    位于指定替换站点的图层必须与现有图层精准匹配。如果图层不匹配，可能无法按预期绘制或执行图层。门户会与替换站点建立连接，以确定图层在替换站点是否可用。如果这些图层不可用，您会收到通知，不可用的图层不会发生更新。

Web 场景设置

以下设置可用于 web 场景。

图层设置

作为 web 场景的所有者或管理员，您可以使用将图层更新至 HTTPS 实用程序批量更新 web 场景中的所有图层，以使其使用 HTTPS。建议使用 HTTPS 访问所有图层，以在通过 Internet 传输时加密图层中的信息。要在访问图层时使用 HTTPS，请单击将图层更新至 HTTPS 并单击更新图层以确认更改。ArcGIS Enterprise 会检查场景中的各个图层以确定是否需要将其更新至 HTTPS。如果发现有任何图层使用 HTTP，门户将使用 HTTPS 向同一图层发出请求，然后更新您拥有或管理的场景及所有关联图层项目。 如果图层不支持 HTTPS，系统会发出通知，并且图层不会在场景中更新。如果报告图层不支持 HTTPS，请联系图层的所有者，以将图层配置为支持 HTTPS 或提供替代资源。

注：

当您更新场景中的图层以使用 HTTPS 时，如果自定义符号源支持 HTTPS，则使用 HTTP 的任何自定义符号都将自动更新以使用 HTTPS。而其他图层自定义，例如嵌入到弹出窗口中的图像，则不会更新到 HTTPS。要更新这些参考以使用 HTTPS，请打开图层的项目页面并更新可视化选项卡上的参考。

托管 Web 图层设置

托管要素图层、托管切片图层和托管场景图层包含用于管理数据的其他设置。

应用程序设置

以下设置适用于应用程序。某些设置仅适用于特定类型的应用程序。

  • URL - 对于类型为 web 制图、移动或应用程序扩展的应用程序，可以编辑其 URL。例如，如果更改了应用程序的 URL，可以使用 URL 设置使应用程序项目指向新 URL。
  • 代码附件 - 您可以向应用程序附加代码（以 ZIP 文件形式）。如果您正在共享一个示例或可配置应用程序并且希望其他用户可以访问您的代码，则可使用此功能。

    要更新代码附件而不重置项目的下载计数器，可使用更新代码，然后上传一个与原始 ZIP 文件同名的 ZIP 文件。如果要附加一个新文件，应首先删除现有文件，然后使用附加代码。这将重置项目的下载计数。

  • 用途 - 对于类型为 web 制图、移动或桌面的应用程序，请指定所添加应用程序的用途。
  • 平台 - 对于桌面应用程序，请指定所添加应用程序的平台。
  • API - 对于 web 应用程序，请指定用于构建应用程序的 API。
  • SDK - 对于移动应用程序，请指定用于构建应用程序的 SDK。
  • 配置参数 - 当应用程序的用途设置为可配置时，您可为其添加可配置参数。
  • 应用程序注册 - 您可以注册应用程序以便他人能够登录 ArcGIS。也可以查看和更新注册信息。

具有嵌入式凭据的安全服务的 URL 协议

如果您拥有使用嵌入式凭据管理安全服务（例如 ArcGIS Server 要素图层或者打印或地理编码服务）的权限，则可以将安全服务数据源的 URL 协议从 http 替换为 https。建议仅使用 HTTPS 访问所有服务，以在通过 Internet 传输时加密服务中的信息。
