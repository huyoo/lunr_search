Ortho Maker 产品

在本主题中

  1. 产品窗格
  2. 栅格分析网关

Ortho Maker 可从重叠无人机影像生成正射影像镶嵌和数字高程表面产品。设置工程和计算局域网平差后，工作流的第三步是生成产品。

Ortho Maker 生成了四种标准产品：正射影像镶嵌、数字表面模型 (DSM)、数字地形模型 (DTM) 和影像集合。您可以选择或取消选择任何要处理的产品。

  • 影像集合 - 包括工程中所涉及的所有源数据和辅助数据的数据库。其中包括源无人机影像、集合中各个影像的地理定位信息、影像轮廓线、飞行数据、统计数据和关于集合中影像的其他特征。
  • 正射影像镶嵌 - 从影像集合进行镶嵌的地理配准影像产品，其中几何变形已经过校正和正射校正。
  • DSM - 数字表面高程数据产品，有时被称为第一表面。其中包括裸露地表的高程，以及建筑物和树木等地上要素。
  • DTM - 裸露地表的数字表面高程产品。它表示没有地上要素的地形。
注：

您可以从产品窗格导入和修改产品处理模板。模板包括产品定义和设置，只要工程拥有适当权限，就可将模板用于任意数量的工程。

可使用正射影像镶嵌、DSM 和 DTM 产品生成其他增值产品：

  • VARI - 可见光大气阻抗指数 (VARI) 用于突出处于光谱中可见光部分的植被，同时减轻光照差异和大气效应。非常适合 RGB 彩色影像，并利用全部三种彩色波段。
  • 坡度 - 坡度地图从 DTM 生成。它提供地形的梯度或陡度。必须在生成 DTM 产品后才能生成坡度产品。
  • 坡向 - 用于描绘坡度的罗盘方向或方向的地图。该地图从 DTM 生成。
  • 自定义 - 根据影像波段运算符、指数（如城镇指数和含水指数）、影像分割和分类，以及其他类型的影像分析创建自定义产品。

如果指定，可在工程设置期间生成产品，或者也可以从产品窗格生成或重新创建产品。如果在工程设置期间指定产品，可从产品窗格生成和访问产品，还可在该窗格中查看、验证和管理产品。

产品窗格

可从产品选项卡的产品窗格中创建和管理 Ortho Maker 产品。单击添加产品，然后从下拉列表中选择产品，以此选择要创建的产品。将向产品库中添加一个代表产品的卡片。各个产品卡片的下方均提供了可以管理产品和调整产品参数设置的选项。将鼠标悬停在产品卡片上方，然后单击创建以开始生成产品。将使用默认设置、来自导入工程模板的设置或针对工程的自定义参数设置生成产品。可重置或修改产品生成参数以重新创建产品。

注：

创建产品前需要先执行快速局域网平差或完整局域网平差。单击报告可生成局域网平差报告，可导出该报告并随工程一同保存。

生成产品时，产品卡片上的默认图标将被替换为产品的缩略图。产品卡片的缩略图下方提供产品管理图标和设置按钮，可用于查看和调整产品生成参数。

在生成的过程中，产品会显示在 Ortho Maker 查看器中。产品以创建的顺序显示，最新创建的产品显示在最上面。对于缩放、平移和优化产品图层而言，Ortho Maker 查看器的工作方式类似于 web Map Viewer。您可以打开和关闭产品图层、调整透明度、在不同图层之间滑动等等。有关详细信息，请参阅 Ortho Maker 查看器。

产品设置

通常来说，利用各个产品的默认设置足以生成各种应用程序产品。如果您拥有摄影测量和影像处理经验，可指定各个标准产品的局域网平差和产品处理参数，以适应独特的工程要求。除了始终使用默认参数的影像集合外，每个产品都拥有一套自己的参数。

产品设置窗格中的各个参数都搭配有参数设置选项的说明。在“参数设置”对话框中，单击与各个参数相关联的信息按钮可查看关于每个标准产品的各个参数的详细说明。

产品管理

单击产品管理按钮可查看管理选项：

  • 查看项目详细信息 - 打开产品 web 页面，其中显示包括摘要和说明、标签、使用条款、配额和文件位置等内容在内的产品信息和详情。您可以选择与个人或企业中的群组共享产品，也可以将产品设置为面向公众。在门户的 Map Viewer 和 Scene Viewer 中，或在 ArcGIS Pro 中查看和使用产品图层。
  • 共享 - 您可以选择与个人或组织中的群组共享产品，也可以将产品设置为面向公众。对此做出的选择将覆盖之前的所有共享设置。
  • 更新 - 使用当前参数设置重新生成产品。
  • 删除 - 删除产品。无法删除影像集合。

栅格分析网关

Ortho Maker 产品适用于 ArcGIS Enterprise 和 ArcGIS Pro、门户 Map Viewer 和门户 Scene Viewer 中的栅格分析。由于在 ArcGIS Enterprise 栅格分析部署中生成和共享 Ortho Maker 产品，因此它们有利于跨 ArcGIS 平台进行进一步处理、分析和共享。Ortho Maker 表面产品为栅格分析水文工具、terrain 分析、最小成本路径和其他邻近分析提供基础。对于分割和多光谱分析、指数、要素提取、更改制图和分析及其他内容来说，正射影像镶嵌是重要的底图。
