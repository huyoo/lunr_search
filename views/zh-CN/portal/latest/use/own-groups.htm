拥有群组

作为群组所有者，您可以编辑群组属性（缩略图、描述、标签等）、管理您的内容和与群组共享的内容、更改一些群组设置，包括可向群组贡献内容的用户、可以访问（即，搜索和查找）群组的用户。如果创建了一个允许他人申请加入的群组，那么，您可接受或拒绝群组成员资格申请。

创建群组后，您可以随时使用设置选项卡更改成员访问和加入群组的方式，以及在群组的内容和概述选项卡上默认显示的项目类型（例如，地图或图层）。作为群组所有者，您可能还需执行其他操作，具体取决于您选择的设置。例如，如果您将群组配置为仅允许群组成员搜索并查找群组（即，如果此群组是私有群组），则必须邀请成员加入该群组。对于可由组织成员或公众找到的群组，您可选择邀请成员加入，或者，如果您更希望让成员申请加入，则必须接受或拒绝他们的成员资格申请。

提示：

您可能希望允许组织中的任何人加入您的群组，而无需受到邀请或经过同意。可通过选择谁可以加入群组？下的所有人选项进行此操作。如果您选择此选项，成员可申请加入群组且会立即被授予成员资格，而无需您的参与。

如果找到了与群组内核心议题有关的内容，但其内容所有者不是群组的成员，则可通过将公共内容共享至群组来获取，这种做法非常有用。群组所有者可从群组中移除内容和成员，对群组启用删除保护以及根据需要删除其群组。所有者也可以选择改变群组成员的角色。例如，您可将一些群组成员提升为群组管理员，使他们能够帮助您处理群组的所有权任务。群组所有者和管理员还可以将群组中的项目分到等级类别，以便群组成员能够更容易找到所需内容。作为默认管理员的群组所有者也可以根据需要对一些群组成员隐藏群组成员资格信息。

管理新的成员资格申请

如果有人申请加入您所拥有的群组，您将看到一个成员资格申请链接。

  1. 确认您已登录。
  2. 单击站点顶部的群组，然后使用选项卡、过滤器、分类选项并根据需要搜索以查找您要管理成员资格申请的群组。
  3. 单击群组名称下的成员资格申请，在随即出现的弹出窗口内针对每个申请单击同意或拒绝。
    提示：

    您也可通过单击群组页面概述选项卡的成员资格申请链接，同意或拒绝群组成员资格申请。

如果同意用户申请，该用户将自动添加到您的群组中。如果拒绝申请，用户则不会添加到您的群组中。无论是哪种情况，用户都可以通过打开站点顶部的通知弹出窗口检查状态。

邀请成员

群组所有者和管理员可以将成员添加到其拥有或管理的群组中，可以选择成员是否需要接受邀请。 管理员和具备向群组中分配成员权限的成员还可直接向群组中添加成员。

  1. 确认您已登录。如果要直接添加成员，请确保您已登录到组织。
  2. 单击站点顶部的群组并使用选项卡、过滤器、排序选项，根据需要搜索以查找要邀请人员加入的群组。单击群组名称以打开群组页面。
  3. 单击邀请用户。
  4. 搜索要添加到群组的成员，并应用过滤器以缩小列表范围。

    可通过单个用户的姓名、用户名或用户个人资料说明中的内容来搜索用户，例如 Mike Jones Redlands GIS。

  5. 选择希望邀请其加入群组的成员。
  6. 要直接添加成员，请选中添加组织成员而无需确认。 如果您是群组所有者或管理员，或者您拥有查看成员拥有的群组和向群组分配成员的权限，则会看到此选项。
  7. 成员选择完成后，单击邀请用户以邀请用户加入您的群组，或单击添加至群组直接添加成员。
    注：

    在邀请用户窗口中，您只会看到一个按钮选项。当您选中相关选项以在无需确认的情况下立即添加组织的成员时，按钮会从邀请用户变为添加至群组。

用户会在通知弹出窗口中收到一条消息。他们可在群组页面上查看邀请以及接受或拒绝此邀请。如果他们接受邀请，他们将自动添加到您的群组，他们的名称将显示在群组页面的成员选项卡中。

更改成员群组角色

邀请成员加入群组后，可根据需要更新他们的群组角色。例如，作为群组所有者，您可将一些群组员提升为群组管理员，使他们能够帮助您处理群组的所有权任务 - 例如，邀请用户、管理成员资格申请、移除内容和成员，以及编辑群组属性。

注：

组织管理员和具有相关权限的用户也可更改群组所有者。

  1. 单击群组页面的成员选项卡。

    注：

    成员选项卡不适用于其成员资格由企业级群组控制的群组。

  2. 按全名搜索成员或应用过滤器以缩小群组成员列表的范围。根据需要对列表进行排序。
    提示：

    活动过滤器显示在结果页面顶部。单击过滤器旁边的 x 可将其清除。要一次性清除全部的活动过滤器，请单击清除过滤器。

  3. 选择要更改群组角色的成员并单击更新成员的群组角色。
  4. 在下拉菜单中，选择其他角色（群组管理员或群组成员），然后单击更新。

    注：

    您只能将您群组中具有创建和拥有群组权限的群组成员提升为群组管理员。建议您仅提升与您具有相似权限的成员，以使他们在群组中的所有权能力与您的能力相匹配。例如，如果您只具有以私有方式共享群组的权限（使这些群组仅供群组成员访问），但您希望提升一位具有可使公众和组织成员访问该群组的权限的成员，则该成员将具有比您作为所有者更多的群组管理能力。

移除成员

如果您不再需要某个成员继续留在群组中，可以从群组中将其移除。他们的项目也会从群组中移除。 用户的帐户和项目不会从门户网站中移除。要移除成员，请执行以下操作：

  1. 单击群组页面的成员选项卡。
    注：

    成员选项卡不适用于其成员资格由企业级群组控制的群组。

  2. 按全名搜索成员或应用过滤器以缩小群组成员列表的范围。根据需要对列表进行排序。
    提示：

    活动过滤器显示在结果页面顶部。单击过滤器旁边的 x 可将其清除。要一次性清除全部的活动过滤器，请单击清除过滤器。

  3. 选中复选框以选择要移除的成员。可根据需要选择任意数量的成员。
  4. 单击从群组中移除成员。
  5. 在移除访问窗口中单击移除。

隐藏群组成员资格信息

在某些情况下，群组所有者可能不希望群组中的每个人都可以看到群组成员列表。例如，为了保护社区和众包群组等公共群组成员的隐私，群组所有者可以选择隐藏群组成员资格信息。如果您是群组的默认管理员和所有者，则可以对除您自己、群组管理员以及具有查看所有成员和群组的管理权限的用户之外的所有人隐藏完整的成员列表（群组所有者和管理员除外）。

注：

此设置仅应用于群组页面的成员选项卡。项目所有者仍将显示在群组中各个项目的项目页面上。当邀请成员加入群组或从组织页面的成员选项卡管理成员时，组织成员能够将该群组视为过滤选项。

  1. 确认您是否以默认管理员的身份登录。
  2. 在站点顶部，单击群组。
  3. 在我的群组选项卡，选择由我所有过滤器，然后选择要隐藏成员资格信息的群组。
  4. 在群组页面中，单击设置选项卡。
  5. 在谁可以查看“成员”选项卡上的成员列表？下，单击隐藏成员。
  6. 在隐藏群组成员窗口中，单击隐藏。
    警告：

    此操作无法撤销。一旦隐藏了完整的成员列表，群组成员列表就无法再次显示。

对内容进行分类

如果您是群组的所有者或管理者，或如果您具有更新群组内容的管理权限，则可通过设置等级类别对群组中的项目进行组织。类别设置完成后，则可将群组中的项目分配给这些类别，以便群组成员在浏览或搜索群组内容时更易找到所需的项目。

注：

类别可帮助其他人快速发现地图、场景、应用程序和其他项目。您可以将类别配置为组织整个组织的内容或与特定群组共享的内容（如下所述）。

设置类别

群组的所有者和管理员可通过配置类别来组织群组内容。您可以创建自己的自定义类别或将 ArcGIS 的标准类别集（呈现在 ArcGIS Living Atlas of the World 中）、国际标准化组织 (ISO) 或 INSPIRE 用作起点。

  1. 单击群组页面的内容选项卡。
  2. 单击群组类别下的设置群组类别。

    注：
    如果您已为此群组创建了类别，请单击群组类别旁的配置群组类别按钮以对群组类别进行添加、重命名、删除或重新排序。

  3. 选择以下选项之一：
    • 自定义类别 - 创建您自己的类别以组织群组中的内容。
    • ArcGIS 类别 - 使用 ArcGIS 类别作为起点。您可以对其进行自定义或使其保持不变。这些类别显示在 ArcGIS Living Atlas of the World 中。
    • ISO 类别 - 使用国际标准化组织 (ISO) 中的主题类别作为起点。您可以对其进行自定义或使其保持不变。ISO 出版物 19115 可提供对地理信息进行描述和分类的方法。
    • INSPIRE 类别 - 使用 INSPIRE 指令中的主题类别作为起点。您可以对其进行自定义或使其保持不变。
  4. 如果您在上一步中选择了自定义类别，则将创建如下所示类别等级：
    1. 单击添加类别按钮，输入类别名称（最多使用 100 个字符），然后单击添加。
    2. 要创建子类别，请单击现有类别下的添加子类别按钮 ，输入子类别的名称（最多使用 100 个字符），然后单击添加。
    3. 根据需要创建其他顶级类别和子类别。
  5. 如果您选择了 ArcGIS 类别、ISO 类别或 INSPIRE 类别，请执行以下任一操作以查看或更改类别等级：
    • 使用类别旁边的展开 和折叠按钮 ，按需展开和折叠类别树中的类别。
    • 要添加类别或子类别，请单击现有类别下方的添加类别按钮 或添加子类别按钮 ，输入类别或子类别的名称（最多 100 个字符），然后单击添加。
  6. 执行以下任意操作以按需对类别进行额外更改：
    • 要重命名类别或子类别，请单击要重命名的类别或子类别的编辑类别按钮，输入新名称（最多使用 100 个字符），然后单击更新。
    • 要删除类别或子类别，请单击您要删除的类别或子类别的删除类别按钮。
    • 要更改等级中同一等级的类别顺序，请将鼠标悬停在类别旁边的展开 或折叠按钮 上，并将其拖到其他位置。
  7. 群组类别设置完成后，单击保存。

    您的类别将在群组类别下方列出，并可随时用于对群组中的项目进行分类。

    注：

    您在此等级中最多可以拥有三个级别的类别，总共最多有 200 个类别和子类别。为了提高效率，建议将项目分配到类别之前，首先创建全面而完整的类别等级和命名方案。

对项目进行分类

群组类别配置完成后，可对群组中的项目进行分类。对内容进行分类以便群组成员更容易查找到所需内容。

  1. 选择要分类的一个或多个群组项目然后单击分类。
  2. 执行以下任一操作：
    • 要一次对所有选定的项目进行分类，请选中复选框以选择相应的类别和子类别。每张项目卡片随即会进行更新以显示您所选的类别。
      注：

      选择子类别将自动选择等级中所有与其相关的类别和其上的子类别。但是，选择类别则不会自动选择其子类别。

    • 要单独分类项目，请单击项目卡片上的分配类别然后选择您想要的类别。您最多可为每个项目分配 20 个类别。您也可以使用过滤器类别框来缩小您的选择范围。
    • 要从特定类别或子类别中移除所有选定项目，请取消选中复选框以取消选择相应的类别或子类别。每张项目卡片随即会进行更新以反映您所移除的类别。
      注：

      取消选择类别和子类别时，将发生以下情况：

      • 取消选择类别或子类别会自动取消选择所有与其相关的子类别。
      • 如果在同一级别上未选择其他子类别，则取消选择子类别也会取消选择其上方的相关类别及其下方的子类别。

    • 要从类别中移除单个项目，请单击相关项目卡片上类别旁边的 x。
    • 要撤消或恢复类别分配，请根据需要单击撤消 或恢复按钮 。
  3. 当完成项目分类后，单击保存。

    将这些群组项目分配到指定的群组类别中。在群组中浏览或搜索项目时，群组成员可以使用这些类别来过滤内容。

    注：

    在对某个项目进行分类时，建议您尽可能选择最具体和最相关的类别，以确保项目可按预期显示在适当的分组中并被发现。例如，要分类 web 地图，请选择 Maps > Web Maps 而非 Maps。

共享公共和组织内容

将项目与您的组织或公众共享后，可将其与您的群组共享。如果内容所有者将项目设置为私有，项目将自动从群组中移除。

  1. 单击站点顶部的内容，然后找到要与群组共享的项目。

    已与所有人（公众）共享的项目都会以地球图标 进行标记。已与组织共享的项目会以建筑图标 进行标记。

  2. 单击项目名称以打开其项目页面。
  3. 单击共享。
  4. 选中要与之共享项目的群组旁边的复选框。如果要与您所拥有的多个群组共享项目，可选择多个群组。
  5. 单击确定。

移除内容

如果某一群组成员共享了一个您认为不适合群组内容的项目，您可将其移除。之后，该项目不再显示在群组中，但它并未从所有者帐户中删除。它仍对其项目所有者可用并保留其他共享属性 - 例如作为公共项目。要从群组中移除项目，请执行以下操作：

  1. 单击群组页面的内容选项卡。
  2. 要查找想从群组中移除的项目，请执行以下任一操作：
    • 在搜索字段中输入关键词。
    • 使用搜索过滤器缩减项目列表。例如，您可以按项目类型（地图、图层、应用程序等）、位置、项目最近修改日期、项目创建日期、标签和共享权限（所有人、组织或群组）进行过滤。
      提示：

      活动过滤器显示在结果页面顶部。单击过滤器旁边的 x 可将其清除。要一次性清除全部的活动过滤器，请单击清除过滤器。

    • 按标题、所有者、浏览次数或评级对项目列表进行排序。也可单击箭头以反转按字母顺序排序。
  3. 选择要从群组中移除的项目。可一次移除一个或多个项目。
    注：

    要在表视图中选择一个项目，需选中该项目旁边的框。要在列表视图或格网视图中选择一个项目，需选中缩略图左上角的框。

  4. 单击从群组中移除。
  5. 单击是以确认您要从群组中移除这一或这些项目。

共享群组

如果您有共享权限，则可使用群组页面概述选项卡内的共享按钮来与其他人共享群组：

启用删除保护

您可阻止您拥有或管理的群组被意外删除。此项设置对于保护特殊群组（例如，Open Data 群组）尤为重要。要在群组上启用删除保护，单击设置选项卡并选中删除保护下的复选框。

删除群组

要删除您拥有、管理或作为管理员的群组，请转到要删除的群组的设置选项卡，然后单击删除群组。在删除窗口中，单击删除。如果该群组已启用删除保护，则此选项不可用。
