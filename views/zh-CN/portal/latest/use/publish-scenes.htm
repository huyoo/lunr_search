发布托管场景图层

在本主题中

  1. 从 ArcGIS Pro 场景发布
  2. 从场景图层包发布

使用以下方法之一将托管场景图层发布到门户：

  • 将多面体或点数据添加到 ArcGIS Pro 中的场景并发布。通过 ArcGIS Pro 发布多面体或点数据时，将在门户上创建场景图层和相关要素图层。
  • 将 LAS 或构建数据添加到 ArcGIS Pro 的场景中并进行发布。当您从 ArcGIS Pro 发布 LAS 或构建数据时，在门户中仅创建场景图层。
  • 创建场景图层包 (.slpk)，将其添加到您的门户中，然后进行发布。当您发布场景图层包时，您的门户中仅创建场景图层。

托管场景图层包含缓存，以支持数据的快速可视化。要创建此缓存，门户的托管服务器需要 ArcGIS Data Store 切片缓存数据存储。

从 ArcGIS Pro 场景发布

将数据图层添加到 ArcGIS Pro 的场景中，配置场景并发布（共享）。

从 ArcGIS Pro 的场景中发布多面体或点数据将创建托管场景图层和相关要素图层。您可以选择为要素图层引用注册数据源中的数据（这将创建 ArcGIS Server 要素服务），或是将要素图层数据复制到门户中（这将创建托管要素图层）。发布 LAS 或构建数据将创建托管场景图层。

请注意，Web 浏览器无法显示您在制作场景时最初可能使用的某些复杂的制图符号。大多数符号类型均可用，但在某些情况下，符号会在发布时降级。有关受支持符号的详细信息，请参阅 ArcGIS Server 帮助中的创作地图以发布要素服务，并在发布多面体或点数据前对符号系统进行所需更改。

在发布之前，请确保您已登录到 ArcGIS Pro 的门户并且门户连接处于活动状态。门户帐户还必须属于具有以下权限的角色：

  • 创建内容。
  • 发布托管场景图层。
  • 如果发布要素图层并复制数据，则需要发布托管要素图层的权限。
  • 如果发布引用注册数据的要素图层，则需要发布基于服务器的图层的权限。

有关从 ArcGIS Pro 中发布场景图层的说明（包括可发布的场景图层类型），请参阅 ArcGIS Pro 帮助中的共享 Web 场景图层。

从场景图层包发布

场景图层包包含一个场景图层及场景图层数据。您可以在 ArcGIS Pro、Drone2Map for ArcGIS 或 Esri CityEngine 中创建场景图层包 (.slpk)。ArcGIS Pro 2.3 及更高版本允许您将 Revit 数据打包、将场景图层包上传至 ArcGIS Enterprise 以及发布建筑物场景图层。

提示：

Esri 将继续提高场景图层的可用性。ArcGIS Pro 2.4、Scene Viewer 和使用 4.12 ArcGIS API for JavaScript 构建的应用程序将使用版本 1.7 及更早版本的索引 3D 场景图层 (I3S) 模型（该模型用于集成式网格和 3D 对象场景图层）。I3S 版本 1.7 可提供比以前版本更好的性能和可扩展性。

为了利用这些改进，创建 3D 对象场景图层包地理处理工具现已支持创建 I3S 1.7 版本的场景图层包。您可以使用 I3S 转换器将现有的集成式网格和 3D 对象场景图层包 (.slpk) 更新到版本 1.7（该转换器可从 I3S GitHub 资料档案库中访问）。I3S 转换器还可识别出现有场景图层包的 I3S 版本，以便您确定是否需要对其进行转换。

有关在场景图层包中可以包含的数据类型的信息，请参阅 ArcGIS Pro 帮助中的场景图层包。

获取场景图层包后，请按照以下步骤将其添加到门户中并发布托管场景图层：

  1. 登录门户并打开内容 > 我的内容。
  2. 单击添加项目 > 从我的计算机。
  3. 单击浏览按钮，选择计算机上的场景图层包。
  4. 输入用于描述场景图层的标题。
  5. 如果门户管理员已配置内容类别，请单击分配类别，然后最多选择 20 个类别以方便其他用户查找您的项目。您也可以使用过滤器类别框来缩小类别列表范围。
  6. 输入标签术语，并以逗号分隔。

    标签是一些词语或短语，用于描述项目并帮助您在搜索时找到项目。Federal land将被视为一个标签，而 Federal,land则被视为两个标签。

  7. 选中将此文件发布为托管图层复选框。

    如果未选中该复选框，系统则会将场景图层包添加到我的内容，但不发布任何场景图层。您可以稍后通过打开场景图层包的项目页面并单击发布来进行发布。

  8. 单击添加项目。
