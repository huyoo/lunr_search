场景图层

在本主题中

  1. 3D 对象
  2. 建筑物
  3. 集成式网格
  4. 点
  5. 点云

Web 场景图层是缓存的 web 图层，它们经过优化，用于显示大量要素，可在 Scene Viewer 和 ArcGIS Pro 中查看。场景图层包括以下类型：3D 对象、建筑、集成网格、点和点云图层。共享场景图层时还会共享一个用于生成缓存的要素图层。

可以使用 ArcGIS Pro 生成的场景图层包将场景图层共享给 ArcGIS Enterprise 门户，或者您也可以通过 ArcGIS Pro 发布 3D 数据。从 ArcGIS Pro 直接发布时，将在门户中创建要素图层和场景图层。

发布 3D 数据时，对于要素图层，您有两个选项：

  • 复制要素数据，从而创建一个托管要素图层。托管要素图层和托管场景图层是相关联的；如果删除场景图层，则托管要素图层也会被删除。
  • 将要素数据保留在已注册的数据源中，从而创建一个ArcGIS Server要素图层。创建引用已注册数据的要素图层有助于提高发布性能，因为要素数据不会被复制。但是，这两个图层不会再紧密关联；删除场景图层时，要素图层不会被删除。

3D 对象

可使用 3D 对象场景图层表示和可视化 3D 对象，例如带纹理或不带纹理的 3D 建模建筑物。 通过 ArcGIS Pro，您可以从直接发布到 ArcGIS Enterprise 中的作为具有关联要素图层的 Web 场景图层的多面体数据创建一个 3D 对象场景图层，也可以从场景图层包 (SLPK) 创建。最后，Esri CityEngine 2016.0 及更高版本可以导出 SLPK，您可以从中发布 3D 对象场景图层。

建筑物

建筑物场景图层可用于可视化复杂的建筑物数字模型，并与建筑物的所有组件进行交互。通过建筑物场景图层，您可以在建筑物数字模型中探索建筑物的组成、属性和结构位置。通过建筑物场景图层，您可以查询和隔离领域图层（例如结构或建筑）或类别（例如墙壁或屋顶）。它们广泛应用于工程、建筑、测量和构造目的。您可以将 ArcGIS Pro 2.3 或更高版本的建筑物场景图层发布为 SLPK。

集成式网格

集成式网格数据通常通过自动化过程进行捕获，从而根据大型重叠影像集构造 3D 对象。结果通过三角插值结构将原始输入影像信息集成为带纹理的网格。集成式网格包括高程信息，可表示构建的 3D 要素和自然的 3D 要素，例如建筑墙体、树木、山谷和悬崖，它们带有逼真的纹理。通常为城市范围内的 3D 制图创建集成式网格场景图层，可使用 Drone2Map for ArcGIS 和 ArcGIS Pro 2.2 及更高版本创建此类图层，然后将其作为 SLPK 共享至 ArcGIS Enterprise 10.5 或更高版本门户。

点

为了确保所有客户端中的快速可视化，可使用缓存点场景图层来显示无法使用点要素图层显示的大量点数据。系统将自动细化点场景图层，从而提高较小比例下的性能和可见性。自动细化意味着在较小比例下并非所有要素均能显示出来；只有放大地图时，附加要素才会显示出来。例如，可利用点场景图层来显示城市中的树。 在 ArcGIS Pro 中，您可以从直接共享到 ArcGIS Enterprise 中的作为 Web 场景图层的要素图层来创建点场景图层。

点云

点云场景图层可在 Scene Viewer 和 ArcGIS Pro 1.4 或更高版本中快速使用和显示大量点云数据。也可以在 ArcGIS Pro 中配置这些图层，然后将其发布到 ArcGIS Enterprise 门户。 根据在 Scene Viewer 中配置的符号系统渲染器类型，ArcGIS Pro 中的点云场景图层显示将有所不同。Scene Viewer 支持以下来自 ArcGIS Pro 的渲染器类型：

  • 拉伸
  • 分类
  • 唯一值
  • RGB
