设置点图层样式

在本主题中

  1. 2D 标记
  2. 3D 对象
  3. 2D 计数和数量
  4. 3D 计数和数量
  5. 2D 类型
  6. 3D 类型
  7. 更改高程模式
  8. 插入注释

  10. 上传自定义标记

您可以对数据应用智能制图样式，来在场景中创建自定义可视化。您可以使用点样式更改要素或具有点数据的场景图层的外观。您可以选择使用属性将连续颜色或单个颜色应用于点的样式，或者选择将单个颜色应用于所有点的样式。

2D 标记

当要素位置的优先级更高时，请使用 2D 标记样式。此样式适用于整个国家/地区的零售店位置或某个城市的公园。例如，如果您拥有整个城市内的医院位置，在缩小至某一区域视图时，屏幕上的标记大小将保持不变，用户可轻松确定城市中的医院位置。

  1. 执行更改图层样式工作流中的步骤。
  2. 对于标记，从库中为您的标记选择 2D 形状和图标。
    • 从下拉菜单中选择基本形状或图标。例如，图标下的标记可显示场景中的不同点，例如机场或加油站。
    • 您也可以从桌面上传自定义标记。请参阅上传自定义标记。
  3. 对于颜色，单击颜色选取器可查看填充、轮廓和透明度选项。
    1. 单击填充可调整填充颜色和透明度。
    2. 单击轮廓可调整轮廓颜色、线宽和透明度。
      注：

      图标下的标记不包括轮廓选项。

  4. 对于大小，以像素为单位手动输入大小来更改标记的大小。
  5. 有关高程模式，请参阅更改高程模式。
  6. 有关注释，请参阅插入注释。
  7. 有关标注，请参阅应用标注。
  8. 对于清理，单击切换键可在导航场景时移除重叠要素。
  9. 对于增强透视，单击切换键可打开和关闭屏幕尺寸点标记的透视缩放。

    当增强透视开启时，图标、标注和注释等对象可在 3D 场景中实现更好的集成，这是因为距离照相机位置越远，尺寸越小。

3D 对象

如果您希望对场景中的实际尺寸对象进行符号化，请使用 3D 对象样式。您可以设置属性以提供这些对象的实际尺寸。在显示城市公园中的树时，使用高度属性设置每个树要素的大小就是一个很好的示例。

  1. 执行更改图层样式工作流中的步骤。
  2. 对于标记，请从 3D 对象的库中进行选择。
    从 3D 对象（已分为多个主题）中进行选择。例如，您可以使用植被标记显示各种植物和树木。 如果您的组织配置了 web 样式，则可以从通过 ArcGIS Pro 创建的样式中选择自定义标记。

    注：

    要在场景中获取完整的 3D 符号系统集（例如，植被），请安装 web 样式安装程序。有关其他信息，请参阅 Scene Viewer 要求。

  3. 对于颜色，可选择颜色并调整透明度。
  4. 对于大小，更改 3D 对象的大小。
    1. 从下拉菜单中选择属性以调整 3D 对象的尺寸。选择 <固定值>，手动输入统一尺寸。
    2. 从下拉菜单中选择测量单位。
    3. 在缩放时，请单击适应视图按钮 以在当前视图下将 3D 对象调整为适当的尺寸。
  5. 您还可以单击旋转下拉菜单基于属性旋转 3D 对象。

    可通过算术法（从 12 点钟方向顺时针）或地理法（从 3 点钟方向逆时针）旋转 3D 对象。如果不希望旋转对象，请从下拉菜单中选择 <无>。

  6. 有关高程模式，请参阅更改高程模式。
  7. 有关标注，请参阅应用标注。

2D 计数和数量

如果您拥有诸如人口统计数据等数据，并且要使用 2D 标记显示数据，请使用 2D 计数和数量样式。例如，您可以通过一系列颜色和对象来以 3D 效果显示社区的平均收入水平，例如从浅色小尺寸到深色大尺寸。可选择最适合您场景的色带和单色。

  1. 执行更改图层样式工作流中的步骤。
  2. 移动顶部和底部滑块控点可基于所选属性调整颜色和大小。

    有关详细信息，请参阅计数和数量滑块注意事项。

  3. 对于标记，请从 2D 形状和图标库中进行选择。

    您也可以从桌面上传自定义标记。请参阅上传自定义标记。

  4. 在颜色的填充选项下从库中选择色带。
    1. 单击反向色带 翻转颜色。
    2. 单击单色以更改为单色。如果使用单色，可调整透明度设置。
    3. 单击轮廓可调整轮廓颜色、线宽和透明度。
      注：

      图标下的标记不包括轮廓选项。

  5. 对于大小（最小 - 最大），选择希望如何调整标记的大小。

    Scene Viewer 会自动为像素大小提供默认范围。大小（最小 - 最大）字段可显示标记范围的最小尺寸和最大尺寸（以像素为单位）。

  6. 有关高程模式，请参阅更改高程模式。
  7. 有关注释，请参阅插入注释。
  8. 有关标注，请参阅应用标注。
  9. 对于增强透视，单击切换键可打开和关闭屏幕尺寸点标记的透视缩放。

    当增强透视开启时，图标、标注和注释等对象可在 3D 场景中实现更好的集成，这是因为距离照相机位置越远，尺寸越小。

3D 计数和数量

要使用实际尺寸对点数据进行符号化，请使用 3D 计数和数量样式。例如，如果您拥有城市资产的数据（如消火栓），则可以根据水量将消火栓显示为高型圆柱。因此，水量越大的消火栓颜色越深，尺寸越大。

  1. 执行更改图层样式工作流中的步骤。
  2. 移动顶部和底部滑块控点可基于所选属性调整颜色和大小。

    有关详细信息，请参阅计数和数量滑块注意事项。

  3. 对于标记，选择具有简单几何（例如圆锥或圆柱）的 3D 模型。
  4. 对于颜色，从色带库中进行选择。
    1. 单击反向色带 翻转颜色。
    2. 单击单色以更改为单色。如果使用单色，可调整透明度设置。
  5. 对于大小（最小 - 最大），选择希望如何调整 3D 对象的大小。
    在缩放时，单击适应视图按钮 来将 3D 对象调整为适合当前视图的尺寸。大小（最小 - 最大）字段可显示对象的最小和最大实际尺寸。
    • 例如，如果您拥有的点图层将十字路口的交通量显示为高型圆柱，您可能需要随着缩放更改高型圆柱的尺寸。为此，先单击适应视图按钮 以对当前视图下的圆柱进行比例化。然后更改大小（最小 - 最大）值来微调圆柱。
  6. 对于旋转，您可以基于属性旋转对象。

    可通过算术法（从 12 点钟方向顺时针）或地理法（从 3 点钟方向逆时针）旋转 3D 对象。如果不希望旋转对象，请从下拉菜单中选择 <无>。

  7. 有关高程模式，请参阅更改高程模式。
  8. 有关标注，请参阅应用标注。

2D 类型

如果您的点数据具有文本或数值属性，请使用 2D 类型样式。您可以使用不同的标记、颜色和大小来显示属性值。例如，您可以使用 2D 类型样式，以独特的符号系统来显示城市的感兴趣点。

  1. 执行更改图层样式工作流中的步骤。
  2. 修改属性值列表。

    2D 类型样式列出了已映射属性旁的之前已选属性的单个属性值。您可以针对每个值自定义符号系统。列表顶部显示了属性值、计数和操作标题。

    属性值

    针对已映射属性列出最常见属性值的名称。其余属性值将自动分组为 。理想情况下，图层显示的值将少于 10 个；如果超过 10 个，用户将难以进行区分。

    计数

    显示每个属性值的要素数。默认情况下，这些属性值将按要素数量的降序列出。您可以在属性值列表中向上或向下拖动值来自定义值在图例中的顺序。

    操作

    可用于将值从其他移入主属性值列表，方法为单击移出值 或移出所有值。

    其他

    允许您将任何值从主属性值列表拖动到其他。单击显示或隐藏可显示或隐藏场景和图例中的其他要素。

    通过选择属性值并单击其名称可对其进行重命名。

  3. 单击属性值顶部的 <全选> 可修改所有值。

    要返回修改单个值，请单击某值，或单击属性值列表顶部的 <单选>。

  4. 对于标记，请从标记库中进行选择。
    • 从下拉菜单中选择基本形状或图标。例如，图标下的标记可显示场景中的不同点，例如机场或加油站。
    • 您也可以从桌面上传自定义标记。请参阅上传自定义标记。
  5. 对于颜色，单击颜色选取器可查看填充、轮廓和透明度选项。
    1. 单击填充可调整填充颜色和透明度。
    2. 单击轮廓可调整轮廓颜色、线宽和透明度。
      注：

      图标样式标记不包括轮廓选项。

    3. 要为所有属性值应用色带，请单击 <全选>。单色随即变为色带。
      • 单击反向色带 翻转颜色。
      • 单击单色返回到选择单一颜色。
  6. 对于大小，以像素为单位手动输入大小来更改标记的大小。
  7. 有关高程模式，请参阅更改高程模式。
  8. 有关注释，请参阅插入注释。
  9. 有关标注，请参阅应用标注。
  10. 对于清理，在导航场景时单击切换键以移除重叠要素。
  11. 对于增强透视，单击切换键可打开和关闭屏幕尺寸点标记的透视缩放。

    当增强透视开启时，图标、标注和注释等对象可在 3D 场景中实现更好的集成，这是因为距离照相机位置越远，尺寸越小。

3D 类型

当您拥有具有文本或数值属性的点数据时，请使用 3D 类型。您可以使用不同的 3D 对象、颜色和大小来显示单个符号系统。例如，您可以使用 3D 类型样式以唯一方式显示不同类型街道设施的 3D 对象。

  1. 执行更改图层样式工作流中的步骤。
  2. 修改属性值列表。

    3D 类型样式列出了已映射属性旁的之前已选属性的单个属性值。您可以针对每个值自定义符号系统。列表顶部显示了属性值、计数和操作标题。

    属性值

    针对已映射属性列出最常见属性值的名称。其余属性值将自动分组为 。理想情况下，图层显示的值将少于 10 个；如果超过 10 个，用户将难以进行区分。

    计数

    显示每个属性值的要素数。默认情况下，这些属性值将按要素数量的降序列出。您可以在属性值列表中向上或向下拖动值来自定义值在图例中的顺序。

    操作

    可用于将值从其他移入主属性值列表，方法为单击移出值 或移出所有值。

    其他

    允许您将任何值从主属性值列表拖动到其他。单击显示或隐藏可显示或隐藏场景和图例中的其他要素。

    通过选择属性值并单击其名称可对其进行重命名。

  3. 单击属性值顶部的 <全选> 可修改所有值。

    要返回修改单个值，请单击某值，或单击属性值列表顶部的 <单选>。

  4. 对于标记，请从 3D 对象的库中进行选择。
    从 3D 对象（已分为多个主题）扩展库中进行选择。例如，您可以使用植被标记显示各种植物和树木。如果您的组织配置了 web 样式，则可以从通过 ArcGIS Pro 创建的样式中选择自定义标记。

    注：

    要在场景中获取完整的 3D 符号系统集（例如，植被），请安装 web 样式安装程序。有关其他信息，请参阅 Scene Viewer 要求。

  5. 对于颜色，将颜色更改为 3D 对象。
    1. 选择一个颜色，并调整透明度设置。
    2. 要为所有属性值应用色带，请单击 <全选>。单色随即变为色带。
      • 单击反向色带 翻转颜色。
      • 单击单色返回到选择单一颜色。
  6. 对于大小，更改 3D 对象的大小。
    • 您可以手动输入大小或选择用于调整 3D 对象大小的属性。从下拉菜单中选择尺寸的单位。在缩放时，可单击适应视图按钮 以在当前视图下将 3D 对象调整为适当的尺寸。
    • 如果在所有标记中选择属性，则标记将根据所选属性自动调整大小，且大小将无法使用。
  7. 对于所有标记，调整标记的大小和旋转。
    1. 选择属性和单位来定义所有属性值的大小。
    2. 按属性设置旋转选项。
      • 单击旋转下拉菜单，然后选择旋转所基于的属性。
      • 可通过算术法（从 12 点钟方向顺时针）或地理法（从 3 点钟方向逆时针）旋转 3D 对象。如果不希望旋转 3D 对象，请从下拉菜单中选择 <无>。
  8. 有关高程模式，请参阅更改高程模式。
  9. 有关标注，请参阅应用标注。

更改高程模式

您可以更改高程模式以自定义图层的位置。单击高程模式下拉箭头 。将出现可以确定图层与其数据垂直放置位置的选项。

地面上

在地面上叠加图层。

相对于地面

相对于地面高程放置数据。z 值会在可用时添加到地面高程。

相对于场景

将数据与 3D 对象场景图层和集成式网格场景图层的顶部对齐，具体取决于哪一对象具有较高的高程。如果图形并非直接位于 3D 对象或任何其他要素之上，则会与 terrain 表面高程对齐。

注：

Z 值将被忽略（如果存在）。

绝对高度

根据要素的几何 z 值位置垂直放置数据。如果几何不包括 z 值，则默认高程为海平面。

<自定义>

根据图层中定义的自定义高程（例如属性表达式）垂直放置数据。<自定义>选项仅在图层具有自定义高程时可见。

偏移

基于所选高程模式，应用正整数值或负整数值（以米为单位）在垂直方向上偏移高程。

  • 相对于地面 - 图层将与地面对齐，并基于输入的高度值在垂直方向上相对于地面进行偏移。
  • 相对于场景 - 图层将与场景图层顶部对齐，并基于输入的高度值在垂直方向上相对于场景图层进行偏移。
  • 绝对高度 - 图层将与分配的 z 值位置对齐，或默认为海平面位置，并基于输入的高度值在垂直方向上进行偏移。

注：

偏移不适用于在地面上高程模式。

插入注释

对于 2D 点样式，请使用注释选项改善 3D 场景中图标位置的感知。垂直注释线可将点标记与其描述的要素的位置相连。例如，可以在场景中将视觉上连接警察局图标标记的注释线与图标下方的警察局建筑物顶部相连。

要使用注释选项，请执行以下操作：

  1. 单击注释切换键以在图层中打开和关闭注释。
  2. 对于注释选项，请执行以下操作：
    • 对于大小，需要选择注释线的长度以及将从图标原始位置提升图标的高度。可以选择较大或较小的注释线。还需要设置数据的合适比例：城市、地区或世界比例。比例将影响缩小时能够看到注释线所需的时间。
    • 对于颜色，您可选择浅色或深色注释线。

应用标注

如果图层中的标注可用，您将会看到标注切换键。通过在场景中显示标注，您可以立即根据图层中的相关属性传达有关场景的信息。

  1. 单击标注切换键以在图层中打开和关闭标注。
  2. 可以通过以下任一操作在样式选项中配置标注：
    • 对于文本，请单击下拉菜单选择用于标注标记的属性。请在标记周围选择标注的定位。
    • 在大小下，选择标注大小（小、中或大）。
    • 对于颜色，您可选择浅色或深色文本。
    • 对于 3D 点样式，单击增强透视切换键可更加直观地显示标注，标注的位置越远，其大小越小。

上传自定义标记

对于 2D 点样式，您可以从桌面上传自定义标记。在上传图标前请先保存场景。

  1. 单击标记。
  2. 单击自定义。
  3. 单击 + 上传图像以添加自定义图标。
  4. 找到要上传的图像。
  5. 单击 + 上传图像以添加其他图标。
  6. 在自定义图库中单击您想要添加到场景的图标。
  7. 单击完成。

    图像要求：

    • 不超过 256 x 256 像素
    • 不超过 1 MB
    • 图像格式：PNG 或 JPEG

    注：

    上传的图标仅在您的场景中可用。您可以共享含有自定义图标的场景，但不会保存该场景的副本。
