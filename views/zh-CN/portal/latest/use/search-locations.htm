搜索位置和要素

在本主题中

  1. 使用搜索查找位置和要素
  2. 将搜索结果添加到地图注释图层

在地图中查找位置或数据的一个方法是搜索。可以通过平移和缩放或读取弹出窗口中的详细信息来进行查找，但是通常情况下，使用搜索工具更为快速和精确。例如，如果您需要查找最近报告遭到损坏的特定雨水道，而您只使用地图进行定位，那么您不得不对您所估计的雨水道区域进行放大并查看每个雨水道的详细信息，直到找到正确的雨水道。而如果使用搜索工具，则可以按照雨水道的 ID 对其进行定位。搜索位置时，可以输入地址、地点、感兴趣点、街道交叉口和经纬度坐标。

Map Viewer 和许多可配置应用程序都包含集成搜索，这种搜索可以合并资源并显示建议，使搜索成为获得正确结果的高效方法。缺省情况下，集成搜索将合并地图中的可用资源。资源可能包括组织配置的定位器和地图内部的可搜索图层。还可以在单一资源中进行搜索，例如，只在雨水道图层进行搜索。

地图会自动缩放到结果，并在该位置显示一个弹出窗口。该弹出窗口包括将结果添加到地图注释选项。

使用搜索查找位置和要素

  1. 打开包含集成搜索微件的 Map Viewer 或可配置应用程序。
  2. 将关键字输入到搜索框。关键字包括诸如地址、街道交叉口、地名、感兴趣点、经纬度坐标和要素 ID 等文本。
    注：

    输入的同时，将显示基于相似关键字的相似的最近搜索或建议。如果使用 Esri World Geocoder，将按名称或类型、地名和邮政编码来显示针对来自托管要素图层、街道地址、街道交叉口和感兴趣点中的要素的建议。 定位器可能启用了建议，也可能没有启用建议。

  3. 或者，如果您想要在单一定位器或可搜索图层中搜索，而不是在所有可用资源中搜索，请单击搜索框中的箭头并选择所需选项。默认情况下，可以搜索所有可用资源。

    地图会自动缩放到最接近的匹配位置，并在该位置显示一个弹出窗口。

将搜索结果添加到地图注释图层

可以使用 Map Viewer 搜索位置或要素，并将搜索结果添加到地图中一个新的或现有的地图注释图层。该图层与地图中的其他地图注释图层类似。可以配置样式和弹出窗口等属性，还可以采用不同方式配置图层副本等。如果您具有创建内容的权限，您可以保存图层以便在其地图中使用，并将图层和地图一同存储。

  1. 使用搜索在地图中查找位置或要素。
  2. 获取所需结果后，请单击添加到地图注释。结果将随即添加到新的地图注释图层。
    • 如果地图不包含地图注释图层，则搜索结果将自动添加到新的图层。
    • 如果地图已包含一个或多个地图注释图层，那么您可以选择将结果添加到现有地图注释图层或添加到新的地图注释图层。
  3. 要添加其他结果，请执行新的搜索并将结果作为新的地图注释图层或现有地图注释图层的一部分添加到地图。
  4. 要将图层和地图一同存储，请单击 Map Viewer 顶部的保存。必须具备创建内容的权限才能执行此操作。
