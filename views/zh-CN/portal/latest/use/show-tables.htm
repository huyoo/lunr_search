显示表格

在本主题中

  1. 在 Map Viewer 中显示表
  2. 显示项目页面中的表
  3. 查看属性和相关记录
  4. 查看相关照片和文件
  5. 选择并缩放至要素
  6. 查看表格时的注意事项

要查看图层中要素的相关信息，可以在Map Viewer中的地图底部或要素图层项目页面的数据选项卡上显示一个交互表。查看数据的表格视图是快速分析信息并开始做出决策的一种方式。您可以对数据进行排序，调整其大小，重新排序和隐藏数据，也可在地图上选择要查看的特定属性。例如，您可显示犯罪图层的表格，查看在过去两年中您所在地区已做出响应的不同类型的犯罪。您可以隐藏一些包含与您的任务不相关数据的列，或者将地图缩放至想要进一步调查的特定犯罪。

您可以查看和排序表中显示的属性数据，以帮助您更好地了解您的要素图层。还可以通过查看任意可用的相关数据表，以及任意相关的照片或其他文件附件来获得有关图层的详细信息。如果具有编辑图层的权限，您可以编辑或删除属性值，以及附加其他相关的照片和文件。有关详细信息，请参阅编辑表。如果您拥有图层，可执行诸如计算字段值等批量编辑操作。作为图层所有者，您还可以通过添加和删除字段更改存储在图层中的信息类型。

在 Map Viewer 中显示表

要在 Map Viewer 中显示要素图层的属性表，请执行以下操作：

  1. 打开想要在表中查看其要素图层属性的地图。
  2. 单击详细信息，然后单击内容。
  3. 请浏览至要素图层，然后单击显示表格 。该表将显示在地图的底部。
    提示：

    要垂直调整表大小，可向上或向下拖动表标题上方中央位置的分隔条。

  4. 单击隐藏表格 可关闭地图中的表格视图。

显示项目页面中的表

要显示要素图层项目页面中的表，请执行以下操作：

  1. 转至想要在表中查看其要素图层属性的项目页面。
  2. 单击数据选项卡。

查看属性和相关记录

表在地图中打开后，您就可以查看图层的属性。如果要素图层的拥有者已经创建了要素图层和另一个表之间的关系，您还可以从相关的表中看到这些属性。相关表的属性在表的右侧以列的形式显示，且具有斜体标题。

提示：

要创建包含相关记录的要素图层，必须从 ArcGIS Pro 或 ArcMap 发布图层。

您可以对表格按如下方式进行操作，以帮助您查找所需的数据：

  • 为了便于查找特定属性值，可通过单击列标题并选择升序排列或降序排列来对属性进行排序。

    例如，如果您要查找您所在地区犯罪表中所有机动车辆盗窃记录（行），请单击 crime_type 列标题，然后选择降序排列。系统会根据 crime_type 值以反向字母顺序排列记录，您可以滚动查看所有机动车辆盗窃记录。

    注：

    某些旧版本的 ArcGIS Server 中的 web 服务不支持分类。

  • 您可以隐藏列，留出空间显示所需的属性。要隐藏或显示属性字段，可单击选项，然后单击显示/隐藏列。选中复选框可显示属性列。取消选中复选框以将其隐藏。

    例如，您可以隐藏犯罪表中的 ID、纬度、经度列，这样您就可以仅看到 crime_type、date_time_reported、官员、arrest_made 和 case_number 数据。

  • 要重新对列进行排序，您需要编辑图层的弹出窗口。
    1. 单击内容窗格中图层下方的更多选项按钮。
    2. 单击配置弹出窗口，然后单击配置属性。
    3. 选择要重新排序的列的字段名，然后使用上移和下移箭头更改列的顺序。从上到下列出的列将在表格中从左到右进行显示。
    4. 单击配置属性对话框中的确认，然后单击配置弹出窗口窗格以应用更改。
    注：

    Map Viewer 中仅支持重新排序列。

  • 要查看相关记录，请在您要查看的记录对应列的单元格中单击显示。

    例如，犯罪图层与案例表相关。要在与特定犯罪相关的案例表中查看详细信息，请单击 case_number 列中相应单元格中的显示。这样，您就可以浏览案例表中的其他记录。如果相关表具有与其自身相关的记录，您也可以对其进行检查 - 例如，如果案例表具有相关的 evidence_collected 表，单击显示以查看相关的证据记录。

  • 查看相关表时，可随时单击关闭按钮 (X) 来返回到之前的表格。
  • 您还可以隐藏或显示相关表中的列。单击相关表中的选项按钮 ，然后单击显示/隐藏列。选中复选框以显示列。取消选中复选框以将其隐藏。

查看数值字段的统计数据

针对属性表中的各个字段（列），您可以查看其信息。 向您提供的信息包括字段数据类型和长度，以及字段是必填字段、可编辑字段还是应用有唯一约束。

如果字段存储数值字段，则还可以查看统计数据，例如行数、最小值和最大值、所有值的总和、平均值及所有值的标准差。

遵循以下步骤可查看数值字段的统计数据：

  1. 针对想要获取其统计数据的数值的字段，单击列标题。
  2. 执行以下其中一项操作：
    • 如果要查看项目页面数据标签中的表，请单击显示详细视图。表将缩小至您所选定字段的更具针对性的视图，并显示统计信息。
    • 如果要在 Map Viewer 中查看表，请单击统计数据。将显示已计算的统计数据。

查看相关照片和文件

要素图层可以包含相关照片或其他文件作为附件，可以从属性表查看这些附件。

执行以下任意操作以查看相关照片和文件：

  • 如果图层具有相关照片或文件，可在表右侧的照片和文件列的相应单元格中单击显示来对其进行查看。
  • 要隐藏或显示相关照片和文件列，请单击选项按钮 ，然后单击显示/隐藏列。选中复选框以显示列。取消选中复选框以将其隐藏。

选择并缩放至要素

可以使用要素图层的属性表，以在 Map Viewer 中选择并缩放至要素。为此，请执行以下任意操作：

  • 要在地图上缩放至某个要素，可单击表中的记录（行）并单击选项，然后单击居中所选要素。
  • 要在地图上选择多个要素，可按住 Shift 键并选择多个记录或一组记录。按住 Ctrl 键可选择两个或多个不相邻的记录。
  • 要清除所做的选择，可单击选项，然后单击清除所选内容。此选项用于清除所有已选记录。
  • 要创建一个用于过滤地图和表中显示的要素数据的查询表达式，请使用内容窗格中的现有过滤器浏览至要素图层，然后单击过滤器 。

查看表格时的注意事项

  • 您可以使用表格查看以下类型的图层：ArcGIS Server 10 SP1 及更高版本的要素和地图服务、托管要素图层、具有关联属性数据的托管切片图层、shapefile、GPX 文件和 CSV（文件和 Web）。
  • 项目页面的数据选项卡上所显示的表中不支持选择和缩放至要素。
  • 在 Map Viewer 中，表中显示的属性字段和格式将与弹出窗口中的配置相同。可通过在弹出窗口属性的配置属性窗口中编辑字段别名来更改字段名称。如果在表打开时对弹出窗口配置进行了更新，您需要先隐藏该表再重新显示才能看到更新。
  • 在 Map Viewer 中，如果选择缩放到单一记录，并且要素为线要素或面要素时，则地图会缩放到该要素的范围。如果要素为点要素，地图会平移到该要素，并在必要时缩放到该要素的可见比例。
  • 在 Map Viewer 中，如果选择缩放到多个记录，则地图会缩放到所有要素的范围。如果所有所选要素的范围超出了图层的可见比例范围，则无法看到所有要素。
  • 在 Map Viewer 中，如果已经在表中选择了一条记录，并在地图中打开了弹出窗口，那么表中所做的选择会被清除。
  • 在 Map Viewer 中，如果图层在地图中处于隐藏状态，则不能缩放到所选内容。
  • 启用了时间的地图中的图层的属性表将始终显示所有记录，不仅仅是给定时间内地图在屏幕上显示的记录。
