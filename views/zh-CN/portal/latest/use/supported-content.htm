关于受支持项目的注释

以下是与各项目相关的一些详细信息：

要素图层

站点支持启用查询功能的 ArcGIS for Server 要素和地图服务。 与群组共享这些要素图层后，用户可将该内容导出为 CSV、KML、shapefile 和 JSON。

CSV 文件

支持您上传到 Portal for ArcGIS 的 CSV 文件，该文件将显示在搜索结果中。可以对文件进行过滤、制表并导出为 .CSV 或 JSON 文件。

注：

当前仅支持小于 5MB 的逗号分隔值 (CSV) 文件。

影像服务

可以将栅格数据和航空影像共享为影像服务。影像服务必须包含字段才能用于添加。用户可将图像导出为 PNG、JPEG 和 TIFF 等格式，或导出为 GeoService。

注：

不支持地图服务内的栅格图层。

Web 地图

可以将 Portal for ArcGIS 中的 web 地图项目添加到站点，还可以在搜索结果中查找该项目以及在站点中查看该项目。

文档

Microsoft Office 文档和 PDF 文档将显示于站点的搜索结果中，并可以在站点中查看。

外部 URL

外部 URL 可作为文档链接添加到 Portal for ArcGIS 中。 这些链接将显示在搜索结果中，但不具有自己的预览页面。用户可通过单击相应的标题直接在新选项卡中打开外部 URL。
