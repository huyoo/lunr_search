可以将哪些内容添加到您的门户？

以下列表显示了可添加到您的门户中的项目类型。向已经与门户联合的 ArcGIS Server 站点发布的服务将以项目形式自动添加到您的门户。所有直接添加到门户的项目或将服务发布到联合服务器时添加的项目，都将显示在已添加或发布项目的帐户的个人存储空间（我的内容）中。

必须具有创建内容的权限才能将以下任意内容添加到门户。

您可以通过 web 浏览器将最大为 200 GB 的文件上传到我的内容。

  • 360 VR 体验 (.3vr)
  • AppBuilder 扩展模块 (URL)
  • AppBuilder 微件包 (.zip) - 仅门户管理员可添加此类型的项目。
  • 应用程序 (URL)
  • ArcGIS Desktop 加载项 (.esriaddin)
  • ArcGIS Explorer 加载项 (.eaz)
  • ArcGIS Explorer 应用程序配置 (.ncfg)
  • ArcGIS Explorer 文档 (.nmf)
  • ArcGIS Explorer 图层 (.nmc)
  • ArcGIS for Windows Mobile 包 (.wmpk)
  • ArcGIS Pro 加载项 (.esriaddinx)
  • ArcGIS Pro 配置 (.proconfigX)
  • ArcGlobe 文档 (.3dd)
  • ArcMap 文档 (.mxd)
  • ArcPad 包 (.zip)
  • ArcReader 文档 (.pmf)
  • ArcScene 文档 (.sxd)
  • CityEngine Web 场景 (.3ws)
  • 代码示例 (.zip)
  • 逗号分隔值 (CSV) 集 (.zip)
  • 逗号分隔的值 (CSV) 文件(.csv)
  • 计算机辅助设计 (CAD) 绘图 (.zip)
  • 深度学习包（.zip 或 .dlpk）
  • 桌面应用程序 (.zip)
  • 桌面应用程序模板 (.zip)
  • 桌面样式 (.stylx)
  • 文档链接（在线文档的 URL）
  • 要素服务 (URL)
  • 文件地理数据库 (.zip) - 如果您发布托管要素图层，则仅发布要素类（仅 x、y 要素）、表、附件和关系类。
  • 地理编码服务 (URL)
  • 地理数据服务 (URL)
  • GeoJSON 文件（.geojson 或 .json）
  • 几何服务 (URL)
  • 地理处理包 (.gpk)
  • 地理处理示例 (.zip)
  • 地理处理服务 (URL)
  • Globe 服务 (URL)
  • 影像集 (.zip)
  • 图像文件 （.jpg、.jpeg、.png、.tif 或 .tiff）
  • 影像服务 (URL)
  • Insights 模型 (JSON)
  • Insights 脚本 (JSON)
  • Insights 主题 (JSON)
  • iWork Keys (.zip)
  • iWork Numbers (.zip)
  • iWork Pages (.zip)
  • Kernel Gateway 连接 (JSON)
  • Keyhole 标记语言 (KML) 集合 (.zip)
  • Keyhole 标记语言 (KML) 文件（.kml 或 .kmz）
  • 图层 (.lyrx)
  • 图层文件 (.lyr)
  • 图层包 （.lpk 或 .lpkx）
  • 布局 (.pagx)
  • 定位器包 (.gcpk)
  • 地图包 （.mpk 或 .mpkx）
  • 地图服务 (URL)
  • 地图服务定义 (MSD) (.msd)
  • 地图模板 (.zip)
  • Microsoft Excel 文件 （.xls 或 .xlsx）
  • Microsoft PowerPoint 演示 （.ppt 或 .pptx）
  • Microsoft Visio 绘图 (.vsd)
  • Microsoft Word 文档 （.doc 或 .docx）
  • 移动应用程序 (URL)
  • 移动底图包 (.bpk)
  • 移动地图包 (.mmpk)
  • 移动场景包 (.mspk)
  • 网络分析服务 (URL)
  • 开放地理空间联盟 (OGC) GeoPackage (.gpkg)
  • 开放地理空间联盟 (OGC) Web 要素服务 (WFS) (URL)
  • 开放地理空间联盟 (OGC) Web 地图服务 (WMS) (URL)
  • 开放地理空间联盟 (OGC) Web 地图切片服务 (WMTS) (URL)
  • 定向影像目录 (.oic)
  • 正射映射工程 (.json)
  • 正射映射模板 (.json)
  • 可移植文档格式 (PDF) (.pdf)
  • Pro 地图 (.mapx)
  • 工程包 (.ppkx)
  • 工程模板 (.aptx)
  • 栅格函数模板（.rft.xml 或 .rft.json）
  • 关系数据库连接 (URL)
  • 规则包 (.rpk)
  • 场景图层包（.spk 或 .slpk）
  • 场景服务 (URL)

  • Web 制图应用程序 (URL)
  • Workflow Manager 服务 (URL)
  • Workflow Manager 包 (.wpk)
注：

在通过相关应用程序或工作流创建和保存项目时，一些项目类型会自动添加到我的内容中，但无法通过添加项目进行添加。例如，Web 地图项目通过 Map Viewer、ArcGIS Pro 或我的内容 > 创建* 来创建，而 Web 场景项目通过 Scene Viewer、ArcGIS Pro 或我的内容 > 创建来创建。
