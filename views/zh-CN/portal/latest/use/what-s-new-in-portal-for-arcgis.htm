ArcGIS Enterprise 10.8 包括更新、增强功能和漏洞修复。有关 10.8 中已解决问题的列表，请参阅 10.8 已解决问题的列表。

如果要从较早版本升级到 10.8，则必须重新授权软件。您可以从 My Esri 中获取新的授权文件。

不支持从 Portal for ArcGIS 10.2.2 或更早版本升级到 10.8。要将 Portal for ArcGIS 升级到 10.8，必须首先升级到 10.3 或更高版本；然后再升级到 10.8。

要确定您的硬件和软件组件是否仍与 10.8 版本兼容，并查看已弃用功能的列表，请参阅弃用通知。要检查哪些较早版本的 ArcGIS 产品与 ArcGIS Enterprise 10.8 兼容，请参阅门户与较早 ArcGIS 版本的兼容性。

组织管理

在此版本中，门户管理员拥有以下新功能和新选项：

  • 新成员默认值允许管理员为新成员设置默认用户类型和角色。指定的默认值会自动分配给自己创建内置帐户的新成员和由管理员添加或通过自动帐户创建添加的成员。
  • 现在有两个新选项可供默认管理员在组织中创建新的群组。管理员可以通过隐藏群组页面成员选项卡上的群组成员列表来保护群组成员的隐私。管理员也可以限制成员离开群组，这样可以确保成员留在群组中，除非他们被群组所有者或群组管理员移除。
  • 宗地结构和公共设施网络功能现在许可为用户类型扩展模块。用户类型扩展模块可以包含在用户类型中或作为附加许可分配。
  • 门户管理员现在可以使用 importlicense 命令行实用程序导入包含许可应用程序和用户类型的许可文件。
  • 管理员现在可以通过门户将 ArcGIS Enterprise 设置为只读模式。打开只读模式后，将无法创建、修改或删除门户中的大多数内容和设置、任何联合服务器以及 ArcGIS Data Store。只读模式不会妨碍门户的正常访问。这可以防止系统更新和 ArcGIS Enterprise 升级时造成停机和数据丢失。
  • 管理员可以配置访问组织的所有用户都能看到的自定义访问通知。启用后，消息将在访问门户时显示，用户必须接受通知的条款才能继续访问站点。
  • 管理员可以配置自定义信息通栏，将其显示在站点的顶部和底部以向用户显示重要信息。此功能可以用于通知访问您站点的用户有关站点的状态和提醒他们未来的维护情况。信息通栏可以在门户的组织设置中进行配置，因此您不再需要使用配置文件中的 classificationBanner 属性。
  • 门户管理员可以使用增强的安全权限模型，根据成员分配的权限来限制或授予 Portal Admin API 中一部分管理任务的访问权限。例如，管理员可以限制某些管理员与门户外观和视觉体验相关的门户设置交互的能力。他们也可以限制管理联合服务器或更新安全性设置的访问权限，例如密码策略更新。这样，组织管理员无需授予 Portal Admin API 的所有访问权限，即可指派特定管理任务。
  • 管理员可以通过新的 webhook 触发事件来监控对组织角色的更改，例如当创建新角色或更新现有角色时，将接到通知。同样，管理员现在也可以订阅更多具有包容性的触发事件。之前，webhook 触发事件允许管理员在特定项目、群组或用户受到更改时接到通知。例如，可以将触发事件设置为在特定用户登录或登出门户或者更新特定项目时触发。现在，组织管理员可以订阅更多时间，从而对用户、项目和群组的所有活动进行监控，例如查看组织中的任何项目是在何时添加或删除的。

数据管理

  • ArcGIS Server Manager 中的新选项允许门户管理员为直接注册到联合服务器的现有数据库和文件共享数据存储创建数据存储项目。这取代了 10.7.1 的工作流。
  • 更新托管矢量切片图层内容的工作流得到了改进。替换图层界面可以将当前托管矢量切片图层的内容替换为替换图层的内容。它还允许您选择当前内容的存档方式，以及预览应用于图层的样式。它可以在当前图层和替换图层不匹配时提醒您，以便您在替换内容之前纠正问题。
  • 在准备用于离线使用的 web 地图时，您现在可以绘制一个多边形以定义用户离线地图时地图中包括的数据区域。这样您可以更好地控制离线数据。您也可以为只读客户端计划更新，以允许其下载预打包更新。
  • 要帮助用户确定如何使用、编辑您共享给他们的托管要素图层中的数据以及设置数据的样式，请添加说明并定义应存储在每个字段中的值的类型。
  • 您可以使用 ArcGIS Pro 2.5 从托管要素图层和托管时空要素图层发布托管地图影像图层，从而使用更复杂的符号系统。这些托管地图影像图层将在 ArcGIS Enterprise 计算机上渲染，并可以以更快的速度绘制大量数据。托管地图影像图层也提供数据的只读访问权限，这意味着您可以与更广泛的受众分享托管地图影像图层。
  • 托管要素图层现在包括系统维护字段，用于存储线和面要素的长度和面积值。
  • REST 要素图层查询可以返回 RFC7946 GeoJSON 中的结果。
  • 如果您需要更改用于追踪托管要素图层编辑者登录的字段，可以禁用图层上的追踪要素的创建者和上次更新的用户设置，访问托管要素图层项目页面的数据选项卡中的要素图层表，编辑字段中的值，然后重新启用追踪要素的创建者和上次更新的用户。如果您使用此字段中的值来控制可以看到托管要素图层中要素的用户或控制可以编辑要素的用户，可以执行此操作。例如，如果 editor1 离开了您的组织，则其他编辑者将无法查看和编辑 editor1 创建的要素。在这种情况下，您可以将用于追踪要素创建者的字段中的 editor1 值替换为接管 editor1 工作的用户的登录名。
  • 从 ArcGIS Pro 或 ArcMap 发布托管要素图层时，可以定义基准面变换，确保数据在以 ArcGIS Online 底图为背景查看时可以正确绘制。

制图和可视化

  • 在 10.8 中，对于新组织，将默认在底图库中启用 Esri 矢量底图。升级到 10.8 的现有组织将会保留门户中配置的底图设置。
  • 您现在可直接在 Map Viewer 中自定义矢量切片图层的样式。
  • Map Viewer 有三个新的符号集用于支持公共设施社区：自来水、污水及雨水。公共安全符号集中包含了新的符号。

3D GIS

  • Scene Viewer 中的全新智能制图样式针对线和面提供完整的样式设置功能。您现在可以从建筑物覆盖区创建属性驱动 3D 拉伸，并按照使用类型等属性上色。此外，新的线轮廓允许您在场景中使用 3D 管、方形、墙面和条带创建可视化内容。
  • 在建筑物浏览器工具中使用新的楼层选择器以交互方式隔离建筑物各个楼层。探索并深入了解从 ArcGIS Pro（已具有支持自定义视图的预定义过滤器）发布的 BIM 建筑物。
  • 向场景添加动画水域可视化效果。无论您是从远处遥望海洋，还是在近处凝视湖泊，全新的水域面样式都可以帮助您创建逼真的视图（例如水下的阳光反射）。
  • 场景图层的性能得到不断改善。Scene Viewer 新的 I3S 规范版本 1.7 使得加载集成式网格、3D 对象和建筑物的速度提高了四倍。此外，新的客户端缓存有助于在探索场景时减少无数据区域。
  • Scene Viewer 中由 Esri 提供的图库中包括了新的 3D 模型（如自行车和消火栓），用于进一步自定义您的场景。此外，图库中新增了基础设施和娱乐 web 样式；为了保持场景中的强大性能，已将自动细节层次添加到逻辑示意图树的 3D 模型中；以及 Scene Viewer 现已支持 CGCS 2000 坐标系。
  • 可以使用图层属性中新的比例范围滑块，在场景中定义图层的可见性范围。例如，可以创建一个场景，该场景在放大时可以在显示国家/地区、城市和建筑物的图层之间进行切换。
  • 现在，可以在点云场景图层上显示弹出窗口。选择单个点以检查属性，例如分类或高程。

  • 性能改进：现在，3D 场景图层的加载速度可达以前的两倍。在场景中进行导航时，提高了矢量切片图层的渲染性能。

  • 视觉改进：当您在“设置”工具中选择质量时，新的光照和阴影模型增加了 3D 内容的对比度和清晰度。此外，Scene Viewer 改进了边缘及其相邻拉伸面的透明度渲染。

共享和协作

以下是共享和协作的新增内容：

  • 您现在可以与您拥有或直接通过群组页面进行管理的群组共享项目。
  • 门户改善了邀请大量成员加入群组时的体验。

ArcGIS Notebook Server

现在，如果您的 ArcGIS Notebook Server 站点与门户联合并使用门户进行配置，则可以使用门户中的新应用程序 ArcGIS Notebook Server Manager。Manager 是 ArcGIS Notebook Server 站点中常见管理任务和资源的直观界面。您可以查看和修改设置、查询日志和管理 notebook 容器。

分析

标准要素分析

ArcGIS Enterprise 门户提供了以下工具增强功能：

  • 创建行驶时间区域已更新以下新参数以改进工具的功能：
    • 行驶方向允许您选择创建的行驶时间区域是远离设施点还是朝向设施点
    • 将无法到达的区域显示为孔洞将返回一个详细的面，输出范围内无法通过道路网络到达的区域将不会包含在输出面中
    • 包括可到达的街道创建包括用于创建行驶时间区域的街道的输出图层
  • 连接起点与终点包括用于创建多个连接的以下新参数和增强功能：
    • 路径形状决定了输出路径的外观。直线创建连接起始点和目的地的直线，当您想要生成蛛网图或期望线时最好选择此选项（例如，要显示客户药房问哪个商店）。沿街道基于基础街道网络创建形状，当您想要在起始点和目的地之间生成路径时最好选择此选项。
    • 您现在可以将多个起始点连接到多个目的地。例如，如果您有 3 个商店，每个商店中有 50 个客户，您可以将所有店铺与其所有客户相连接。

应用程序

ArcGIS Mission

ArcGIS Mission 是 ArcGIS Enterprise 10.8 中的新应用程序，可以为组织和团队提供以任务为中心的实时态势感知和协作工具，帮助您全面了解操作环境。此应用程序允许用户创建、共享和实时监控任务，指定操作的地图和其他材料，简化对等通信并实时监控任务的执行。

ArcGIS Mission 由三个核心组件构成： ArcGIS Mission Manager、ArcGIS Mission Responder 和 ArcGIS Mission Server。Mission Manager 是一款 web 应用程序，允许用户创建任务并将任务配置到移动应用程序，在活动任务中共享工作以及监控任务。Mission Responder 是一款移动应用程序，允许用户执行 Manager 分配的任务，发送状态和位置更新以及通过对等通信共享信息。Mission Server 简化了 Manager 和 Responder 之间的对等网络。

有关详细信息，请参阅 ArcGIS Mission 帮助。

ArcGIS QuickCapture

ArcGIS QuickCapture 设计器现在随 ArcGIS Enterprise 一起安装，允许您完全在您的企业环境中创建和共享工程。或者，您也可以继续在登录 ArcGIS Enterprise 的状态下使用 ArcGIS QuickCapture 网站来创建和共享工程。当您使用 ArcGIS QuickCapture 网站时，您可以使用 QuickCapture 设计器的最新版本。ArcGIS Enterprise 10.8 中包含 ArcGIS QuickCapture 1.2 设计器。

有关详细信息，请参阅 ArcGIS QuickCapture 帮助。

Tracker for ArcGIS

Track Viewer web 应用程序现在随 ArcGIS Enterprise 一并安装，无需单独安装。您现在可以在查看轨迹时更改底图。

有关详细信息，请参阅 Tracker for ArcGIS 帮助。

Workforce for ArcGIS

Workforce for ArcGIS web 应用程序现在随 ArcGIS Enterprise 一并安装，无需单独安装。

有关详细信息，请参阅 Workforce for ArcGIS 帮助。

Web AppBuilder for ArcGIS

Web AppBuilder for ArcGIS 新增了一个微件，并改进了多个微件和主题。有关详细信息，请参阅 Web AppBuilder for ArcGIS 中的新特性。

  • 新 Business Analyst 微件允许您生成“Business Analyst 信息图表”和“经典报表”。
  • 您可以通过搜索要素图层在“方向”中添加停靠点。
  • 使用查询和过滤器，您可以对日期和时间组合执行查询。
  • “我的位置”微件提供新选项，可以在移动设备上显示罗盘方向和位置精度。
  • 通过“群组过滤器”，可以使用 URL 参数定义过滤器值。
  • “周边”微件具有可配置的缓冲区输入选项，可以使用精确距离进行搜索。
  • 您可以配置“公共通知”微件以在 CSV 下载中添加字段名称。
  • “智能编辑器”具有多项增强功能，包括支持相关日期以及使用 URL 参数定义预设值。
  • “筛查”微件允许您在报表中隐藏摘要表。
  • 还增强了多个微件和主题的可访问性。

Operations Dashboard for ArcGIS

早期的 Operations Dashboard 产品已于 2019 年 1 月 1 日停用。有关详细信息，请参阅“产品生命周期”页面。自 ArcGIS Enterprise 10.8 起，现有的操作视图、Operations Dashboard 扩展模块和 Operations Dashboard 加载项将无法为早期的 Operations Dashboard 产品所使用。这些内容现已停用，您将无法打开和查看它们。这些项目的项目详细信息页面将在门户中保持只读状态。请考虑使用 ArcGIS Enterprise 内置的改进过的新版本 Operations Dashboard 来重新创建这些项目。

可配置应用程序模板

提供以下新的 ArcGIS Configurable Apps 模板：

  • Attachment Viewer - 在地图中的要素图层上突出显示照片、视频或 PDF 附件。
  • Nearby - 帮助应用程序用户在距其提供的位置指定距离的范围内查找感兴趣的位置。
  • Zone Lookup - 帮助应用程序用户确定他们指定的位置是否位于预定义区域内，并返回关于该区域的信息。

Category Gallery 应用程序模板包括了一些新选项，允许您从图库中的缩略图打开应用程序，并控制登录选项、图库中针对每个应用程序显示的信息以及过滤窗格打开的时间。
