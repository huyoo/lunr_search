底图库微件

在本主题中

  1. 配置“底图库”微件
  2. 使用“底图库”微件

“底图库”微件可显示底图库，并允许您从库中选择一个底图作为应用程序的底图。底图可由用户定义，或者来自组织或门户。但是，添加到“底图库”微件中的所有底图必须使用相同的空间参考。此外，您还可以选择在运行时将应用程序与组织或门户中的“底图库”设置同步。

通常，微件中不包括切片方案不同于当前底图切片方案的底图。如果默认底图为动态图层，则可以切换到动态图层或 ArcGIS 切片地图服务图层。如果默认地图为切片图层，则可以切换到具有相同切片方案的切片图层或动态图层。为获得最佳性能，建议添加到库中的所有底图均应为缓存（切片）图层。

下表列出了“底图库”微件中受支持的底图和不受支持的底图：

  • 不受支持的底图
    • 空间参考不同
    • 切片底图的切片方案不同
    • 从动态底图切换为 Bing、OSM 或 WMTS
    • 从动态底图切换为切片影像服务底图
    • WMS 服务
  • 受支持的底图
    • 在切片底图之间进行切换
    • 在动态底图之间进行切换
    • 从动态底图切换为切片地图服务底图
    • 从切片底图切换为动态底图

配置“底图库”微件

可将此微件设置为应用程序启动时自动打开。要启用该功能，请在微件上单击在应用程序启动时自动打开此微件按钮 ，微件随即变成深绿色。

  1. 将鼠标悬停在微件上方，然后单击配置此微件按钮 以打开配置属性对话框。

    2. 组织或门户中的底图以及默认的始终与组织的底图库设置同步选项显示在配置对话框上。

    警告：

    虽然组织或门户“底图库”中的底图可使用不同的空间参考，但配置窗口仅显示与在 Web AppBuilder 中当前加载的 web 地图具有同一空间参考的底图。这是一个已知限制。

  2. 如果想要替换此微件的默认图标，还可单击更改微件图标。

    随即显示文件资源管理器窗口，供您浏览可用作微件图标的本地图像文件。

    注：

    使用 Internet Explorer 8 浏览器时，由于 Internet Explorer 8 自身的限制，上传图像的文件大小不得超过 21K。

  3. 如果希望应用程序在运行时与组织或门户中的底图库设置保持一致，请保留默认选项始终与组织的底图库设置同步。
  4. 或者，选择配置自定义底图，以从群组中导入底图或创建新的底图。
    • 单击导入。显示导入底图窗口时，从下拉菜单中选择一个群组，然后单击要添加到微件的地图。单击确定关闭此窗口。要添加来自不同群组的更多底图，请重复此步骤。
    • （该选项已弃用。） 单击新建。显示新建窗口时，提供标题和地图服务的 URL，然后单击缩略图将其替换为您自己的图像文件。如果需要，单击添加 URL 将地图服务的附加 URL 添加到底图中。单击确定关闭此窗口。重复此步骤以创建其他底图。

    您配置的底图会与所选缩略图一起列出。

    警告：

    虽然单个底图可以包含多个图层或服务，但这些图层必须使用相同的空间参考。

  5. 单击确定关闭此窗口。
    警告：

    添加到“底图库”微件中的底图必须使用相同的空间参考。

使用“底图库”微件

单击“底图库”微件将显示为此微件配置的所有底图。单击底图缩略图可将其设置为应用程序的活动底图。单击底图库窗口右上角的 x 可将其关闭。

注：

如果配置期间未包括在 web 地图中使用的底图，则在打开微件时会自动添加该底图。
