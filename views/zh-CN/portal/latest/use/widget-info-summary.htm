信息汇总微件

在本主题中

  1. 配置“信息汇总”微件
  2. 使用“信息汇总”微件

“信息汇总”微件可用于针对每个指定图层提供当前地图范围内的要素计数。可以展开微件面板中的每个图层，以显示当前范围内要素的列表，也可以按指定字段对图层进行分组。可以配置将微件中的点图层显示为聚类。

注：

“信息汇总”微件现在是一种面板内微件。已配置应用程序内的微件放置将不再位于右侧；将根据当前主题显示该微件。

只能将一个微件实例添加到 Web 应用程序中。

配置“信息汇总”微件

可将此微件设置为应用程序启动时自动打开。要启用该功能，请在微件上单击在应用程序启动时自动打开此微件按钮 ，微件随即变成深绿色。

  1. 将鼠标悬停在微件上方，然后单击配置此微件按钮 以打开配置属性对话框。
  2. 也可以选择单击更改微件图标按钮，然后将其替换为自己的图像。

    将显示文件资源管理器窗口，供您选择可用作微件图标的本地图像文件。

  3. 单击添加图层。
    每个添加的图层都会在地图中指定一个要素图层，微件会对指定的要素图层进行计数和显示。“信息汇总”微件支持以下图层类型：
    • 要素服务
    • 要素集合
    • 简单地图服务
    • 逗号分隔值 (CSV) 文件 (.csv)
    • Keyhole 标记语言 (KML) 文件（.kml 和 .kmz）
    • GeoRSS Web 订阅源 (URL)

    注：
    只有在组织中存储为要素集合项目的要素集合支持“要素集合”刷新。您可以在项目详细信息页面中找到此信息。

  4. 此外，还可为每个图层添加一个标签。
  5. 单击显示选项标题下方的编辑按钮 ，可更新符号系统选项和面板显示选项。
    • 使用符号选项卡更改地图符号系统、启用点图层的聚类，或者更新面板符号系统（微件面板中显示的符号）。
    • 使用面板选项卡管理微件面板中要素信息列表的显示方式。在图层选项下方选择一个字段和标签以利用所选属性显示要素列表。
    • 或者，选中组选项下的组要素复选框，按照指定字段或地图中使用的渲染器对列表中的要素进行分组。
  6. 也可以选择刷新要素集合图层类型，以根据指定间隔持续更新数据。如果启用刷新选项，则刷新间隔为必填项。
  7. 在操作下方，更改图层顺序或删除图层。
  8. 或者，选择显示要素计数，在微件面板上显示每个图层的要素计数。

    该计数将显示选定图层的要素数量。

  9. 可选择显示徽标图像向微件标题添加文本或图像。
    • 单击上传图像按钮以添加自己的图像。

      将显示文件资源管理器窗口，供您选择本地图像文件。

    • 在主面板文本文本框中指定要显示在徽标图像下方的文本，如上次更新。
  10. 可以选择启动时展开列表。这将在打开微件后自动展开面板中的首行。
  11. 可以选择显示所有要素以显示图层要素的完整列表和计数，无论当前地图范围为何。

    注：
    如果关闭 Web 地图中的一个或多个图层，即使选中了显示所有要素复选框，微件也不会显示已关闭图层的要素计数。微件只显示可见图层中的要素计数，无论这些要素在当前地图范围中是否可见。

  12. 也可以选择隐藏面板来启用应用程序，从而在不使用可见微件面板的情况下打开。

    注：
    如果选择上述选项，则需要将微件配置为自动打开。设置该选项可确保启用微件逻辑，而不显示面板。

  13. 可以选择禁用关闭微件后保留聚类符号，由此当关闭微件后，聚类符号将恢复为默认符号。

使用“信息汇总”微件

  1. 单击信息汇总微件按钮将其打开。

  2. 单击图层按钮可打开和关闭图层。
  3. 在微件面板中选择图层可展开要素列表。要素列表将基于当前地图范围，如果微件配置为显示所有要素，则将显示图层的所有要素。如果按指定字段对图层进行分组，可使用箭头将其展开，以显示选定组内的要素列表。
  4. 如果微件未显示所有要素，则平移和缩放地图可更新微件面板中的要素计数。
  5. 单击信息汇总微件或单击微件右上角的关闭按钮以将其关闭。关闭微件后，聚类图层将仍然可见（除非将微件配置为恢复为原始符号）。要关闭聚类，请在图层列表微件中取消选中此聚类。
