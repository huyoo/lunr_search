倾斜查看器微件

在本主题中

  1. 配置“倾斜查看器”微件
  2. 使用“倾斜查看器”微件

在 ArcGIS 中，“倾斜查看器”微件使用影像坐标系 (ICS) 在本地坐标系中显示影像，可提供相机位置的自然视图。倾斜影像显示在本地 ICS 中，可避免由于将其投影到地理坐标系或投影坐标系而引起变形。该微件允许用户在倾斜影像视图中进行查看、缩放和平移，或者切换角度以从其他照相机视角查看感兴趣的区域。

配置“倾斜查看器”微件

可将此微件设置为应用程序启动时自动打开。要启用该功能，请在微件上单击在应用程序启动时自动打开此微件按钮 ，微件随即变成深绿色。

  1. 将鼠标悬停在微件上方，然后单击配置此微件按钮 以打开配置属性对话框。
  2. 如果想要替换此微件的默认图标，还可单击更改微件图标。
    随即将打开文件资源管理器窗口，供您选择可用作微件图标的本地图像文件。
  3. 单击选择图层下拉列表，选择要与此微件结合使用的影像服务图层（来自输入 web 地图）。
    注：

    倾斜影像服务必须基于镶嵌数据集，可以使用具有每个输入影像的方向元数据的帧照相机栅格类型进行配置。

    “帧照相机”栅格类型包括以下两个必须存在的特定字段：高程和方位角，指的是照相机的视图方向，表示为以度为单位测量的角度。相机高程 = 0 表示瞄准地平线，并且地平线以上的角度为正，地平线以下的角度为负，因此角度 = -90 表示瞄准像底点。照相机方位角 = 0 表示在指定坐标系中瞄准正北的影像顶部，以及顺时针测量的角度（例如 90 = 东）。

  4. 单击高程字段下拉列表，从可用字段列表中选择用于定义传感器高程的字段。
  5. 单击方位角字段下拉列表，从可用字段列表中选择用于定义传感器方位角的字段。
  6. 定义一个搜索半径，以将绝对值设置为主地图中已放置的图钉周围的半径。

    由此可以增大与已放置图钉相关联的感兴趣区域，并查看包含该区域的倾斜影像。默认值设置为 0 米。

  7. 在选择列表信息字段中，选择要显示在栅格列表中的信息字段。
  8. 选中在栅格列表中显示缩略图选项可以启用微件中栅格列表内的缩略图。
  9. 选中同步主地图选项可以自动同步像底点地图与倾斜地图中相应范围所做的任何更改。

    像底点地图可能是底图，但通常是输入 Web 地图中的第二个影像服务，其中显示与倾斜影像同时捕获的像底点影像的动态正射映射。

  10. 选中启用距离和面积工具以显示倾斜地图窗口内的距离、角度、面积和周长测量工具。
  11. 单击确定，保存并关闭配置窗口。

使用“倾斜查看器”微件

初始化后，“倾斜查看器”微件将在像底点视图中显示影像。

  1. 单击微件图标打开包含倾斜地图的微件。
  2. 微件面板右下侧的转速计 允许用户选择要显示倾斜影像的方向。标度盘侧面的蓝色方块用于指示显示影像的方向。单击蓝色方块可将影像的视图切换到相应方向。
  3. 单击转速计中间的圆圈可将微件面板内显示的影像切换到像底点模式。
  4. 微件面板顶部的工具栏包括以下操作。选择其中一个操作以浏览影像。
    • 单击 可在主地图上放置图钉。由此可以定位某个区域并查看与之关联的倾斜图像。
    • 单击 按钮可清除主地图上显示的所有图形。
    • 单击 可缩放至倾斜地图窗口内所选栅格的范围。
    • 单击 可以手动同步像底点地图与倾斜地图中相应位置所做的任何更改。仅当同步主地图选项在配置窗口中处于禁用状态时，微件面板中才会显示此图标。

    • 单击 可列出适用于已显示地图范围的所有栅格影像。可以根据选择的角度在此微件的转速计内过滤此列表。

    • 测量图标 允许用户选择不同高度测量操作。下拉列表允许用户选择测量单位和测量操作。以下是可用的各种测量选项。
      • 高度：底部至顶层 - 通过从结构底部至结构顶部进行测量，计算结构的高度。假设测量是垂直于底部进行的。沿建筑物测量的线必须使其终点正好位于起点的上方。
      • 高度：顶部至顶部阴影 - 通过从结构顶部到地面上结构阴影的顶部进行测量，计算结构的高度。结构上的点及其阴影必须表示相同的点。
      • 高度：底部至顶部阴影 - 通过从结构底部到地面上结构阴影的顶部进行测量，计算结构的高度。阴影中的点必须表示垂直于底部的结构上的点。
      • 距离和角度 - 计算以交互方式绘制的线段的距离和方位角。
      • 面积和周长 - 计算以交互方式绘制的面的面积和周长。
      • 线性单位 - 允许您为线性测量配置测量预筛选单位。其中包括以下选项：
        • 英寸
        • 英尺
        • 码
        • 英里
        • 海里
        • 毫米
        • 厘米
        • 分米
        • 米
        • 千米
