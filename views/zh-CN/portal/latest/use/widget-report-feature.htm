报告要素微件

在本主题中

  1. 配置“报告要素”微件
  2. 使用“报告要素”微件

“报告要素”微件使您能够收集并管理用户给出的数据质量反馈。用户可以报告地图中出错或需要附加检查的现有要素。用户还可以通过在地图上草绘要素来标识缺失的要素。反馈将作为 Reviewer 结果存储在地理数据库中，可在此通过已定义生命周期工作流对其进行访问与管理。

许可：

“报告要素”微件需要 ArcGIS Data Reviewer for Server 扩展模块。

配置“报告要素”微件

可将此微件设置为应用程序启动时自动打开。要启用该功能，请在微件上单击在应用程序启动时自动打开此微件按钮 ，微件随即变成深绿色。

  1. 将鼠标悬停在微件上方，然后单击配置此微件按钮 以打开配置属性对话框。
  2. 或者，单击更改微件图标按钮，然后将其替换为自己的图像。
    将显示文件资源管理器窗口，供您选择可用作微件图标的本地图像文件。
  3. 提供 Data Reviewer for Server URL，然后单击设置。此新 URL 地图服务中的相关会话将填充默认会话下拉列表。
  4. 选择结果的 Reviewer 技术人员属性的填充方式。适用于填充此属性的选项如下：
    • 登录用户名 - 活动的门户用户名将填充此属性。
    • 输入默认值 - 预定义的值将填充此属性。
    • 允许用户输入 - 用户在提交过程中提供一个值。
  5. 从下拉列表中选择 Reviewer 会话用作默认会话，然后单击确定。

    Reviewer 结果将立即保存至您选择作为默认会话的会话。

  6. 默认情况下，当前地图中的所有图层均启用了在微件中进行错误报告。要排除一个或多个图层，取消选中包含列中的复选框。
  7. 要编辑图层的别名，双击当前别名然后输入一个新名称。

    图层别名值仅用于在微件 UI 中显示。图层值用于填充结果的 resourceName 属性。

  8. 单击确定保存并关闭“报告要素”微件的新配置。

使用“报告要素”微件

  1. 在选择选项卡上，从下拉菜单中选择一个地图图层。
  2. 放大并单击以选择要报告的要素。将弹出详细信息对话框。
  3. 从下列选项中选择适当的查看状态：
    • 添加要素
    • 删除要素
    • 移动要素
    • 修整要素
  4. 请提供有关此问题的注释（如果存在）。
  5. 选择该要素非一致性问题的严重性等级，1 代表最严重，5 代表最不严重。
  6. 在报告人字段中，输入报告此问题的人员姓名。

    在某些部署中，报告人字段将不显示。

  7. 单击报告。

    将显示报告已接收通知，表示反馈已接收。单击确定。

  8. 单击绘制选项卡，并在绘制工具条选项中选择一个几何类型。
  9. 转至地图上需要报告要素并进行绘制的准确位置。将弹出详细信息对话框。
  10. 从下列选项中选择适当的查看状态：
    • 添加要素
    • 删除要素
    • 移动要素
    • 修整要素
  11. 请提供有关此问题的注释（如果存在）。
  12. 选择该要素非一致性问题的严重性等级，1 代表最严重，5 代表最不严重。
  13. 在报告人字段中，输入报告此问题的人员姓名。

    在某些部署中，报告人字段将不显示。

  14. 单击报告。

    将显示报告已接收通知，表示反馈已接收。单击确定。
