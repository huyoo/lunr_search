Formatting a report

Formatting the report structure

Roadway Reporter allows you to format the output of your report by allowing you to change column headers and column order for the Adobe PDF and CSV file output formats. In addition, for PDF reports, there are a number of additional formatting options including page orientation, adding a header, footer, page number, date, font face and size for text, width and orientation of columns, and whether or not to include table borders and alternating row colors.

  1. On the Report Fields page, type a title for the report in the Report Title text box.
  2. Click on the appropriate PDF or CSV file output.

    注：

    If a report is output to a CSV file, you can open your output in Excel and do further formatting to your report.

  3. Edit the column headers of the summary fields by double-clicking the cell under the Title column.
    • You can edit the length of the summary columns by double-clicking the cell under the Length column.
    • You can change the order of the fields in the report using the arrows in conjunction with selecting the desired row to move the order up or down.
  4. Click Report Formatting to format the report.
    • This is an optional step. If you do not format the report, default format selections will be used.
    • Formatting is only valid when you choose PDF as an output.

Configuring report format

Report formats can be configured for mileage and segment reports when you choose PDF as an output. The graphic below is an example of a mileage report that has been produced using Roadway Reporter. In this example, -the title, subtitle, header, footer, page number, date and time stamp, font, notes, and alternating row colors are configured.

The following can be configured for reports in Roadway Reporter:

  • Title
  • Notes
  • Table
  • Header and Footer
  • Date and Time Stamp

Title

  1. Click the Title tab.
  2. Browse to the Title subsection. Use the text field to add or edit the title of your report.
  3. Use the drop-down arrow to select a font and font size.

    By default, the font is set to Helvetica and the font size is 12.

  4. Use the drop-down arrow to select a color from the color picker and click Choose.
  5. Select the font style.

    By default, the style is set to bold.

  6. Select the alignment of your content.

    By default, the alignment is set to center.

  7. Browse to the Subtitle subsection. Use the text field to edit the subsection.
  8. Follow steps 2 to 7 to select font, style, size, color, and alignment.

Notes

  1. Click the Notes tab.
  2. Specify the placement of your note.
    • Start of the report—Your note will be placed at the beginning of the report.
    • End of the report—Your note will be placed at the end of the report.
  3. Browse to the Title subsection. Use the text field to add or edit the title of your note.
  4. Use the drop-down arrow to select a font and font size.

    By default, the font is set to Helvetica and the font size is 11.

  5. Use the drop-down arrow to select a color from the color picker and click Choose.
  6. Select the font style.
  7. Select the alignment of your content.
  8. Browse to the Notes subsection and add a note in the text field.
  9. Use steps 4 to 7 to select font, style, size, color, and alignment.

Table

  1. Select the Table.
  2. Browse to the Summary Fields subsection.
  3. Use the drop-down arrow to select a font and font size.

    By default, the font is set to Helvetica and the font size is 10.

  4. Use the drop-down arrow to select a color from the color picker and click Choose to specify a font color.
  5. Select the font style.
  6. Select the alignment of your content.
  7. Use the drop-down arrow to select a color from the color picker and click Choose to specify the Background Color.
  8. Optionally select Yes if you'd like to place a thicker line to the right.
  9. Browse to the Column Headers subsection.
  10. Use steps 3 to 7 to select font, style, size, color, and alignment.
  11. Browse to the Table subsection.
  12. Use steps 3 to 7 to select font, style, size, color, and alignment.
  13. Specify a preference for borders:
    • All Borders—Adds borders to all columns and rows in your report.
    • No Border—Does not add borders to the columns and rows in your report.
  14. If you'd like to enable alternate shades in rows, use the slider to select Yes.

Header and Footer

  1. Select the Header and Footer tab.
  2. Browse to the Header subsection.
  3. Use the drop-down arrow to select a font and font size.

    By default, the font is set to Helvetica and the font size is 9.

  4. Use the drop-down arrow to select a color from the color picker and click Choose to specify a font color.
  5. Select the font style.
  6. Select the alignment of your content.
  7. Optionally add a date and time stamp by using the slider to select Yes.
  8. Optionally add page numbers by using the slider to select Yes. Specify a format for page numbers:
    • 1
    • Page 1
    • Page 1/10
  9. Browse to the Footer subsection.
  10. Use steps 3 to 8 to select font, style, size, color, alignment, date and time stamp, and page numbers.

Date and Time Stamp

  1. Select the Date and Time Stamp tab.
  2. Select a place in the body:
    • None—No date and time stamp will be added to the report.
    • Start of the report—The date and time stamp will be added to the start of the report.
    • End of the report—The date and time stamp will be added to the end of the report.
  3. Use the drop-down arrow to specify the font and the font size of the date and time stamp.
  4. Use the drop-down arrow to select a color from the color picker and click Choose to specify a font color.
  5. Select the font style.
  6. Select the alignment of your content.

Once you have finished, click OK to save the report formatting or click Reset to restore default settings.
