排除请求中的 MIME 类型

在本主题中

  1. ArcGIS Server 未使用的 MIME 类型

多用途 Internet 邮件扩展模块 (MIME) 是原始电子邮件协议的扩展模块，可用于在 Internet 上交换不同类型的数据文件，例如音频、视频、图像、程序等。ArcGIS Server 可发送多种 MIME 类型（作为标题），以供客户端（如 web 浏览器）决定数据文件的处理方式。例如，绘制地图服务的请求将产生指明 JPEG 图像类型的 MIME 标题。

在安全性限制较多的组织中，建议您对 web 服务器进行配置，以排除 ArcGIS Server 不使用的 MIME 类型。这样操作会为您的 GIS Web 服务器带来另外一层的安全性。此外，如果您怀疑自己的站点受到影响，可监测来自 ArcGIS Server 的响应以查看 GIS 服务器是否发送了已排除的 MIME 类型。
