使用服务器统计数据

  2. 默认报表
  3. 创建新报表
  4. 查看和修改现有报表
  5. 删除报表
  6. 导出报表
  7. 搜索报表
  8. 配置服务器统计数据

通过 ArcGIS Server Manager 中的统计数据报表库可显示、配置、更新、导出和删除服务器统计数据的报表。要访问报表库，请单击日志 > 统计数据。有关如何使用报表和统计数据的描述，请参阅以下各部分内容。

可用服务器统计数据

可在报表中显示以下统计数据。如果您的 ArcGIS Server 站点有多台计算机，统计信息将聚合站点中的所有计算机，最大运行实例数除外。

解决办法：

您可以使用 ArcGIS Server 管理员目录查看多机站点中各个计算机的统计信息。

每个报表仅能显示一个统计数据。要选择统计数据，请在报表中单击显示下拉对话框。

  • 请求总数 - 指定时间范围内客户端向服务发出请求的总数。
  • 平均响应时间 - ArcGIS Server 向客户端返回请求的结果所花费的平均时间。以秒为单位显示平均响应时间。
  • 最长响应时间 - ArcGIS Server 向客户端返回请求的结果所花费的最长时间。以秒为单位显示最长响应时间。
  • 超时 - 以下一种或多种情况的组合可能会报告服务超时：
    • 客户端保持某一服务的时间超过指定的最长使用时间，客户端将失去对此服务的引用。
    • 客户端的等待时间超过为服务指定的最长等待时间。
  • 最大运行实例数 - 指定时间范围内对特定服务运行的最大运行实例数。

默认情况下，每 30 分钟采集和存储一次统计数据。要更改统计数据的采集频率，请参阅下面的配置服务器统计数据部分。

支持的服务类型

统计数据仅适用于以下服务类型。有关这些服务类型的详细信息，请参阅可发布什么类型的服务。

  • 要素（也支持托管要素服务）
  • 地理编码
  • 地理数据
  • 地理处理（仅支持同步；不支持异步）
  • 影像（仅支持动态影像；不支持缓存影像）
  • 地图（仅支持动态地图；不支持缓存地图）
  • 网络分析
  • 搜索
  • Workflow Manager

默认报表

默认情况下，安装 ArcGIS Server 时，可获取统计数据报表库中显示的三个预配置报表。这些报表基于已在服务器上运行的服务。通过在模块中选择默认报表并更改显示的服务数量、统计数据类型和时间值可修改默认报表。此外，通过单击报表缩略图上的 X 按钮可删除报表。

创建新报表

要创建新报表，请执行以下步骤：

  1. 打开 Manager，然后使用具有管理员权限的帐户登录。如需此步骤的帮助，请参阅登录 Manager。属于用户或发布者角色的帐户无法创建报表。
  2. 单击日志 > 统计数据。
  3. 单击新建报表。
  4. 在图表的左侧，选择要显示统计数据的一个或多个服务，也可以选择一个文件夹。还可使用搜索字段来搜索服务。选择文件夹时，图表中将显示文件夹下所有服务的聚合值。还可通过单击复选框旁边的标记来更改资源（服务或文件夹）的颜色和标记符号。
  5. 单击显示下拉对话框，然后选择要在图表中显示的统计数据。统计数据始终在图表的 Y 轴（左）上显示。
  6. 单击期限下拉对话框，然后为统计数据选择要显示的时间段。期限始终显示在图表的 X 轴 （底部）上。选择自定义时间段可指定要显示统计数据的自定义时间范围。报告的时间与安装 ArcGIS Server 的计算机的时区相符。
  7. 通过单击图表顶部的输入报表标题文本可赋予报表一个标题。
  8. 单击保存。

随即保存报表且其他管理员和发布者可查看报表。还可通过单击导出按钮将图表导出为逗号分隔值 (CSV) 文件。默认情况下，报表每隔 30 分钟将自动更新。

查看和修改现有报表

如果使用具有发布者或管理员权限的帐户登录，通过在统计数据模块中单击现有报表的缩略图可查看现有报表。查看报表时，悬停在图表中折点的上方可获取详细信息。例如，如果您有地图服务的最长响应时间图，则悬停在特定数据点上方会显示特定响应时间 1.91 秒。

如果使用具有管理员权限的帐户登录，还可修改现有报表并通过单击保存来应用所有更改。如果单击关闭，系统会提示您保存更改。如果关闭浏览器，则不会保存更改。

要打开和修改报表，请执行以下步骤：

  1. 打开 Manager，然后使用具有管理员权限的帐户登录。如需此步骤的帮助，请参阅 登录 Manager。
  2. 单击日志 > 统计数据。
  3. 单击要修改的报表的缩略图。
  4. 应用对报表进行的更改。下面几个章节将对可更改的内容进行概述。
  5. 单击保存。

更改显示的服务或文件夹

可更改图表中显示的服务数量或文件夹中的服务数量。使用复选框可将服务和文件夹切换为打开或关闭。选择文件夹时，文件夹下所有服务的聚合值会在图表中显示为一行。

将文件夹切换为打开或关闭时，不会自动选中文件夹下所有资源的复选框。同样，取消选中父文件夹下的子资源不会自动取消选中父文件夹。通过设计此行为，可将整个文件夹与单个资源 （服务或文件夹）进行比较。例如，可选择站点（根）文件夹，然后将其与嵌套在子文件夹中的单个地图服务进行比较。

更改服务或文件夹的颜色和标记符号

可更改单个文件夹或服务的颜色和标记符号。为此，请单击资源（文件夹或服务）旁的相关标记，然后从列表中选择适用的标记和颜色。默认情况下，有八种颜色和四个标记可供选择。

更改统计数据类型和时间范围

可更改图表中显示的统计数据和时间范围（期限）。要更改显示的统计数据，请单击显示下拉对话框并选择所需统计数据。统计数据始终在图表的 Y 轴（左）上显示。有关各统计数据的说明，请参阅上面的可用服务器统计数据部分。

要更改显示统计数据的时间段，请单击期限下拉对话框并选择所需的时间窗口。选择自定义时间段以在自和至输入对话框中指定要显示统计数据的自定义时间范围。期限始终显示在图表的 X 轴 （底部）上。报告的时间与安装 ArcGIS Server 的计算机的时区相符。

统计数据报表有不同的 x 轴间距，具体取决于报表中所显示的时间窗口。数据点之间的 x 轴间距表示以下时间间隔：

  • 等于或小于 24 小时的时间窗口的间隔为 30 分钟
  • 1 天到 7 天的时间窗口的间隔为 4 小时
  • 7 天到 30 天的时间窗口的间隔为 1 天
  • 大于 30 天的时间窗口的间隔为 7 天

x 轴开始时间也将有所不同，具体取决于报表的时间窗口和对应的 x 轴间距。每个预配置报表时间段（最近 24 小时、最近 7 天和最近 30 天）的开始时间将为该报表时间段中所述的四舍五入到过去的最近半小时时间。例如，如果最近 24 小时报表的请求时间为 10:11 a.m.，则 x 轴开始时间应为前一天的 10:00 a.m.。自定义时间段会显示第一个时间间隔结束时的 x 轴开始时间。这意味着，自定义时间段为 12 月 1 日 12:00 a.m. 至 12 月 5 日 12:00 a.m. 的报表的 x 轴开始时间将显示为 12 月 1 日 4:00 a.m.，由于时间窗口介于 1 天和 7 天之间，因此时间间隔为 4 小时。将鼠标悬停在报表的第一个数据点上时，将显示汇总 12 月 1 日 12:00 a.m. 至 12 月 1 日 4:00 a.m. 数据的点。

更改标题

修改报表时可更新标题。为此，请单击标题文本并提供适当信息。确保明确指明图表中报告的内容。还可添加一些说明文本来支持标题。

删除报表

要删除报表，请使用具有管理员权限的帐户登录，然后单击统计数据模块中报表缩略图上的 X 按钮。

导出报表

保存报表时，还可以选择将报表导出为逗号分隔值 (CSV) 文件。这样您便可以将报表导入到第三方应用程序（如 Microsoft Excel）中。这种方法可以方便您共享和显示报表数据，而无需以物理方式登录到 ArcGIS Server Manager。要导出报表，请使用具有管理员权限的帐户登录并打开报表，然后单击导出按钮。

搜索报表

通过向统计数据模块中的搜索报表对话框中输入文本可搜索特定报表。例如，可提供以下信息来帮助您查找要查找的报表：

  • 服务名称（显示包含该特定服务的报表）
  • 文件夹名称（显示包含该特定文件夹的报表）
  • 关键字（显示与以下关键字相匹配的报表）
    • 根（显示包含站点 [根] 文件夹的报表）
    • 日、周、月，自定义（显示与指定的持续时间相匹配的报表）
    • 请求、平均、响应、超时（显示与指定的统计数据类型相匹配的报表）

配置服务器统计数据

默认情况下，每 30 分钟采集和存储一次统计数据。可以更改统计数据的采集频率和在磁盘上保存统计数据历史记录的时间。此外，还可禁用服务器统计数据。

要配置服务器统计数据，请执行下列步骤。

  1. 打开 ArcGIS Server 管理员目录并使用具有管理员权限的帐户登录。
  2. 单击使用情况报告 > 设置 > 编辑。
  3. 使用以下选项配置服务器统计数据：
    • 要禁用服务器生成统计数据，请单击启用使用情况报告下拉对话框，然后选择否。
    • 要更改聚合统计数据的持续时间，请在聚合统计数据间隔（分钟）字段中指定新值（以分钟为单位）。默认设置为 30 分钟。例如，要将持续时间更改为一天，请输入 1440。
    • 要更改在磁盘上保留服务器统计数据的最大时间段，请在保留统计数据历史记录的时间（天）字段中输入新值。默认设置为永久 (*)。例如，要将时间段更改为一年，请输入 365。统计数据信息将保存在 ArcGIS Server 系统目录中。
