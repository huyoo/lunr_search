在 Windows 8/8.1 中启用 IIS 及所需的 IIS 组件

在本主题中

  1. 所需 IIS 组件

在 Windows 8/8.1 中，ArcGIS Web Adaptor 要求启用 IIS 及特定 IIS 组件。如果未检测到 IIS 或未启用特定的 IIS 组件，则安装程序将无法继续。

如果您已启用 IIS 但缺少所需 IIS 组件，则安装程序将显示一条消息，提示缺少某些 IIS 组件。您可以选择允许安装程序自动启用所需 IIS 组件。但是根据组织安全策略，也可能需要按以下说明手动启用所需的 IIS 组件。

要了解在 Windows 8/8.1 中启用 IIS 及所需 IIS 组件的方法，请参阅以下说明。

  1. 打开控制面板并单击程序和功能 > 打开或关闭 Windows 功能。
  2. 启用 Internet 信息服务。
  3. 展开 Internet 信息服务功能，并确认以下列出的 Web 服务器组件是否已启用。
  4. 单击确定。

所需 IIS 组件

以下列出的 IIS 组件可以满足运行 Web Adaptor 的最低要求。如果您已启用了其他 IIS 组件，也无需删除它们。

  • Web 管理工具
    • IIS 6 管理兼容性
      • IIS 元数据库和 IIS 6 配置兼容性
    • IIS 管理控制台
    • IIS 管理脚本和工具
    • IIS 管理服务

  • 万维网服务
    • 应用程序开发功能

      • .NET 扩展性 4.5
      • ASP.NET 4.5
      • ISAPI 扩展
      • ISAPI 筛选器
      • WebSocket 协议

    • 常见 HTTP 功能
      • 默认文档
      • 静态内容
    • 安全性
      • 基本身份验证
      • 请求筛选
      • Windows 身份验证
