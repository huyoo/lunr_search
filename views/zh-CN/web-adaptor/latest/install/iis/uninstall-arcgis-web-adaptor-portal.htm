卸载 ArcGIS Web Adaptor

在本主题中

  1. 卸载 Web Adaptor
  2. 静默卸载 Web Adaptor

卸载 ArcGIS Web Adaptor 时，会将 Web Adaptor 从 Portal for ArcGIS 上注销，并且客户端应用程序将无法与您的门户通信。要以后重新建立此通信，请重新安装 ArcGIS Web Adaptor，并使用配置页面在门户上重新配置 Web Adaptor。有关详细说明，请参阅配置 ArcGIS Web Adaptor。

To uninstall ArcGIS Web Adaptor, follow the steps in the sections below.

卸载 Web Adaptor

要卸载 ArcGIS Web Adaptor，请完成以下步骤：

  1. 打开控制面板，然后单击程序和功能。
  2. 从程序列表中选择 ArcGIS Web Adaptor (IIS)。
  3. 选择卸载/更改，以从计算机中卸载 Web Adaptor。
